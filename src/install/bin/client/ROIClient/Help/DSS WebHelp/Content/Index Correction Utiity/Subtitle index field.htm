Subtitle index field

The Subtitle index field is defined as an alphanumeric, free-form entry field. This allows users to create subtitles ‘on the fly.’

Dates in the Subtitle field are not sorted in chronological order as a true date field would be sorted. For example, a date sort places “11/03/1999” before “01/07/2010”, but an alphanumeric sort places “01/07/2010” before “11/03/1999”. This may cause unexpected results when documents with date subtitles are listed in ProdName-here (such as in Composite view).

Subtitles can be used in Batch Compiler and COLD Agent, as well as at the DCS/QCI Workstation. Subtitle format should be defined at the enterprise level.
