Toolbar buttons

Button

Description

Allows you to scroll to the first, previous, next, or last document.

OR

The button label toggles, depending on which view option is active.

“View” displays for All Pending Documents or Expired Documents only. Opens a viewer window (if no viewer window is open) and loads the current document’s image into the viewer.

“Compare” displays for Inbound Exceptions only.  Opens two viewer windows (if no viewer window is open) and displays the inbound document image in one viewer window, and the image of an existing document that has the same index information in the other viewer window.

Pulls the latest data from database for the current view option.

Allows you to delete the current document entry from its queue.

Note: This button is disabled for All Pending Documents.


OR

The button label toggles, depending on which view option is active.

“Update” displays and is enabled only for Expired Documents only. Applies updates you have made to index information for the document you are currently viewing.

“Accept” displays only for Inbound Exceptions only. Enables you to overwrite an existing document when you have determined that the inbound document should be kept and the existing document should be discarded. Also applies any updates you have made to index field information.

Closes any open viewer window and exits the ICU application.
