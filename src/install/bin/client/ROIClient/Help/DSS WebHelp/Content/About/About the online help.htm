About the online help

Overview

Note: This information is effective with the OneContent™ family of products except where specifically noted. In addition, readers should assume that information for the OneContent™ product is generally applicable to the OneContent™ Business Folder product as well. For easier readability, such passages may cite only "OneContent."

Purpose

The online help provides user instructions for the Display Workstation (DSS) viewer.

The online help provides user instructions for the Index Correction Utility (ICU).

Scope

The DSS viewer is launched by the Index Correction Utility (and the Release of Information (ROI) Module).

The online help describes all DSS functionality available when the viewer is launched from any component.

The online help describes all ICU functionality available to an end user who is responsible for correcting document indexes. This document is effective for only a Medical Records domain.

Audience

The online help is intended for end-users who view document images.

The online help is intended for end-users who view document images to correct document indexes.

This information assumes that users have knowledge of OneContent software components, are familiar with Microsoft® Windows® user interface principles and terms, and understand the basics of operating a mouse.
