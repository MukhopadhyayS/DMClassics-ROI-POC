Index Correction Utility

Overview

The Index Correction Utility enables you to repair indexing information for many documents that fail to upload properly.

Note: The Index Correction Utility is an error-handling mechanism only. The Index Correction Utility does not replace the QC/Index application. Additionally, only one user at a time per facility should execute the utility; more than one may cause timeout errors.

Use the Index Correction Utility (ICU) to repair index information for documents identified by the Index Upload program as having index errors. This includes inbound documents that encounter exception conditions.

The utility permits you to:

  • Review and resolve inbound exceptions.
  • Repair index information for documents that have index errors.  The utility permits you to enter missing index information or to correct invalid index information and can be used within a single facility or across a multi-facility enterprise. The utility displays a subset of index fields displayed by the OneContent QCI workstation.
  • Manually review and process inbound COLD or HL7 documents that have triggered an "inbound exception."
  • Delete document information from the CAB_HOLD table.

The ICU application launches an image viewer window to display actual document images. The viewer window can display up to two images (that is, pages) at a time. The viewer includes its own online help.

ICU is intended for personnel whose responsibilities include correcting document index information. It allows users to repair documents by Index Upload as having an indexing error.  It also allows users to resolve inbound exception documents.

CAUTION: Do not use a database management tool (such as SQL Server Management Studio) to manually delete entries from the CAB_HOLD table. Manually deleting CAB_HOLD entries creates orphaned images that become inaccessible. Use the normal document processing functions to maintain the CAB_HOLD table.

Note: As of Release 17.6.2, some ICU functionality has been integrated with the OneContent application. Refer to the OneContent online help for more information and instructions.
