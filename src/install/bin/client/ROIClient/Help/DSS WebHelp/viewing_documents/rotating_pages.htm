Rotating pages

You can rotate only image documents or COLD documents viewed as images. You cannot rotate documents displayed as text, and although you can rotate a PDF document, you cannot save the rotation setting.

To rotate pages

Rules are:

• Rotation is applied to all pages in the same document. You can scroll back and forth within the document without losing the rotation aspect.

To rotate a page to the right, left, or 180 degrees, use any of the following methods:

To save the rotation aspect

Saving the rotation aspect is effective only for image documents. You cannot save the rotation aspect of a text-based COLD document that you are viewing as an image or a PDF document.

Rotation is retained only if you select the File > Save Image menu option. The system gives you the option of applying the rotation to all pages, applying it to only the current page, or discarding the rotation entirely.
