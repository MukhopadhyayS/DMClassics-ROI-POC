Viewing Documents > Adjusting the page view > Scrolling pages

Scrolling a single page

To scroll a page you are viewing, you can either use the standard window scroll bars or use the Drag Scroll feature.

  • Window scroll bars are effective for both text and image viewing.
  • The Drag Scroll function is effective only when you view a document as an image. It enables you to drag an image to reposition it within the display area.
Using window scroll bars
  1. View a document as text or as an image.
  2. Zoom in so that the page is larger than the display area.
  1. Left-click-hold a window scroll bar and drag it left or right, or up or down.
Using drag scroll

The Drag Scroll feature is available only when the viewer’s configuration file is properly configured to support it. The Drag Scroll menu item must be enabled for Rubberband Zoom to operate.

1. View a document as an image.
2. Zoom in so that the image is larger than the display area.
3. Enable Drag Scroll on the View menu.
4. Click and hold the left mouse on the page image, and then drag the image to reposition it as desired.
