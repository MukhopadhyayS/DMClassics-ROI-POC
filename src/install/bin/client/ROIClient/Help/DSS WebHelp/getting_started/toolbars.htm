Toolbars

Viewer toolbars contain buttons for quick and easy access to commonly used functions.

This toolbar allows you to manipulate an image and navigate quickly to a specific page in a document. It also provides buttons for adding deficiencies. The buttons are enabled only when a variety of specific conditions are true.

Note: Occasionally, using the Zoom + and Zoom - buttons distorts images. To restore the image, use the Best Fit function, and then zoom in with the Rubberband Zoom function and zoom out with the View menu or by pressing F4.

Showing or hiding the toolbars

Menu commands on the View menu enable you to hide or show the toolbars.

Select ToolBar from the View menu to hide or show the toolbar as preferred. A checkmark next to the menu command indicates that the toolbar is enabled.
