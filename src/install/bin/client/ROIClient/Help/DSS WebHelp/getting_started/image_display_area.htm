Image display area

The image display area contains the document displayed either as an image or as text.

  • Displaying a document as an image provides a broader set of functions, zooming in or out, and rotating and manipulating the image in various ways. Scanned documents are always displayed as images, never as text. Certain text documents can optionally be displayed as images.
  • Displaying a document as text allows faster display; it also increases legibility for some documents. Only text-based documents received through electronic feeds can be displayed both as images or plain text. The following example shows a document displayed as text.
  • Display attributes apply to all documents you open during the current session. Display attributes are not “remembered” from the current session to another unless you save them. See “User Preferences” later in this document for more information.
