View menu

The View menu allows you to zoom, rotate, and scroll the document, and to specify viewing a document as text. It also allows you to enable or disable toolbars, the extended status bar, and the popup menu feature.

Menu commands are enabled depending on whether you are viewing an image file or a text-based file.

  • If the Display as Text command is available, but not checked, you are viewing a text-based document as an image.

If the command is enabled (check mark ON), you are viewing a text-based document as text. When Display as Text is enabled, all other display commands on the View menu are disabled. (Toolbar, status bar, and popup menu commands are still accessible.)

  • If the Display as Text command is unavailable, you are viewing a scanned document.

Menu command

Toolbar icon

Description

Actual Size

Displays the image in its actual (100% zoom factor) size.

Zoom In

Enlarges the image by 50 percent of its current size.

Note: On occasion, using the Zoom In button distorts images. In such cases, click Best Fit to restore the image, and then use Rubberband Zoom to zoom in.

Zoom Out

Reduces the image by 50 percent of its current size.

Note: On occasion, using the Zoom Out button distorts images. In such cases, use the Best Fit function to restore the image, and then use the View menu or press F4 to zoom out.

Zoom

N/A

Allows you to choose the zoom factor ranging from 25% of the image’s actual size to 400% of its actual size.

Fit Width

Fits the image in the window so that the entire width of the page is visible.

Fit Height

 

Fits the image in the window so that the entire height of the page is visible.

Best Fit

Fits the entire image in the window.

Default Orientation

Displays the image in its default orientation (no rotation).

Rotate Left

N/A

Rotates the image 90 degrees counter-clockwise from its current orientation.

Rotate Right (or Rotate 90)

Rotates the image 90 degrees clockwise from its current orientation.

Flip

N/A

Rotates the image 180 degrees from its current orientation.

Drag Scroll

N/A

Toggles drag scroll mode.

Available only when you view a document as an image, you are in View mode, and the image is larger than the display area.

Note: The Drag Scroll menu item must be enabled for Rubberband Zoom to operate.

Rubberband Zoom

N/A

Toggles the ability to use the right mouse button to perform a rubberband zoom operation. When enabled, you can click and hold the right mouse button, and then drag the cursor to zoom an area.

Note: The Drag Scroll menu item must be enabled for Rubberband Zoom to operate.

Selecting Rubberband Zoom automatically deselects Popup Menu (if it is checked), and vice versa. These two menu commands cannot be enabled at the same time.

Display as Text

N/A

For non-scanned documents (that is, text-based documents) only. Toggles between displaying the documents as images or as text files.

Disabled if you are viewing an image document.

Note: Some documents may be hard to read if font sizes are small. Use this option to display the file as a text file for better readability. You can optionally select a font to improve readability. Allscripts recommends using a fixed-width font such as Lucida Console or Courier New.

Toolbar

N/A

Toggles to display or hide the main toolbar.

Status Bar

N/A

Toggles to display or hide the status bar.

Extended Status Bar

N/A

Toggles to display or hide the extended status bar.
