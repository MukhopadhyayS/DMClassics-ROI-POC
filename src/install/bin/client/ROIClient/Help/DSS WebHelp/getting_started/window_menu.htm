Window menu

The Window menu contains the following menu commands.

Menu command

Toolbar icon

Description

Save Settings

N/A

Saves the position setting of the window.

Save Position as Default

N/A

Saves the current window position and size as the default setting.

Restore Default Position

N/A

Restores the current viewer window to the last saved values for position and size.

Always On Top

N/A

Keeps the viewer window displayed in front of other open windows.
