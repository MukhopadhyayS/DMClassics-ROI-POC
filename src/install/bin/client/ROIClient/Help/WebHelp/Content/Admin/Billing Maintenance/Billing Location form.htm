Billing Location form

The Billing Location form allows you to configure the sales tax percentage for a specific billing location ("facility") instead of one percentage across the enterprise. When a billing location is selected during the release and invoice process, the sales tax is automatically applied and calculated.  The billing locations configured are used at the time of billing for associating invoices.

To open the form

Select the Administration tab > Billing Maintenance > Billing Location.

Item descriptions

Main Content Pane

Billing Location grid

Displays the billing location and associated information currently defined in the system. The grid title is preceded by the number of existing billing locations.

The Billing Location grid contains the following columns:

  • Default
    Indicates if this is the default location.
  • Billing Location
    Name of the billing location.
  • Tax Percentage
    Amount of the tax percentage.
• Description
Descriptions of the billing location (optional).

Create New

Allows you to enter the information needed to add a new billing location.

Delete

Deletes the selected billing location.

Detail Pane

Default

Indicates if this is the default billing location.

Billing Location

Identifies the name of the billing location.

Tax Percentage

Specifies the percentage of sales tax to apply to the billing location. This sales tax is applied and calculated during the release and invoice process.

Description

Provides the description for this billing location.  You can change the description at any time.  Descriptions are optional.

Save

Saves the new or modified billing location.

Cancel

Cancels the billing location you were adding or any unsaved changes you made to an existing billing location.
