Add Patient dialog box

The Add Patient dialog box enables you to add one or more patients to a request.  You can access this dialog box when you are creating a request or when you are viewing an existing request.

To open the form

  1. Select the Requests tab.
  2. Create a request or open an existing request.
  3. At the Patient Information form, click Add Patient (for a new request) or Add Another Patient (for an existing request).

The dialog box displays two tabs: Find Patient and Patient Information.

Item descriptions

Find Patient tab

The Find Patient tab displays by default.

This tab uses the same items as the Patients > Find Patient form with the exception of the Patient Requesting and Create Request buttons, which do not display on this tab.

See the Find Patient form for item descriptions.

Patient Information tab

The Patient Information tab can be accessed only if you click either New Patient or View/Edit Patient on the Find Patient tab.

This tab uses the same items as the Patients > Patient Information form, with the exception of the Patient Requesting and Create Request buttons, which do not display on this tab.

See the Patient Information form for item descriptions.

Additional items

This dialog box contains the following additional items.

Add Patient & Close

Adds the selected patient to the request, closes the dialog box, and returns you to the Requests > Patient Information form.

For a patient requestor on a non-EPN system, also launches the Potential Patient Mismatch dialog box if the selected patient added to the request does not seem to match the requestor.

Always displays.

Enabled only after you select a patient.

Add Patient & Continue

Allows you to continue adding patients.

Displays when either of the following is true:

  • Requestor is a patient and EPN is not configured
  • Requestor is not a patient

Enabled only after you select a patient.

When clicked at the Find Patient tab, adds the selected patient to the request and keeps the dialog box open so you can select another patient to add.

When clicked at the Patient Information tab, adds the selected patient to the request and returns you to the Find Patient tab, where you can select another user to add.

For a patient requestor on a non-EPN system, also launches the Potential Patient Mismatch dialog box if the selected patient added to the request does not seem to match the requestor.

Cancel

Discards unsaved selections, closes the dialog box, and returns you to the calling form.
