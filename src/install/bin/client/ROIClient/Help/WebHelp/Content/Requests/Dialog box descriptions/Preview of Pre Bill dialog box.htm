Preview of Pre-Bill dialog box

The Preview of Pre-Bill dialog box dialog box displays a preview of the invoice you are generating, and outputs the form using the selected method.

To open the form

  1. From the Release & Invoice form, click Create Pre-Bill.
  2. In the Pre-Bill dialog box, select a template, enter any notes, and click Continue.
  3. In the Preview of Pre-bill dialog box, select the output method.

Item descriptions

Print

Closes the preview dialog box and launches the Print output dialog box.

Displays only if you have print/fax permission.

Fax

Closes the preview dialog box and launches the Fax output dialog box.

Displays only if you have print/fax permission.

Save as File

Closes the preview dialog box and launches the Save As File output dialog box.

Displays only if you have the Export to PDF permission.

Email

Closes the preview dialog box and launches the Email output dialog box.

Displays only if you have the email permission.

Cancel

Cancels the creation of the cover letter and the release of the documents, closes the Release dialog box, and returns you to the Release & Invoice form.

Page controls

Functions (such as save the file, navigate pages, and zoom the view) that are available from the third-party product used to display previews.
