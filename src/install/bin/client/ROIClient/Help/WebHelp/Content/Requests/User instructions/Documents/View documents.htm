View documents

This function enables you to view one or more patient documents. Although patients can have non-OneContent documents, you can view only OneContent documents.

Note: The system does not store images of non-OneContent documents; it stores only information about them. If you access a non-OneContent document, the system displays a dialog box that allows you to edit document information. See Maintain non-MPF document information for details.

All users can view unlocked and non-VIP documents. Access to VIP and/or locked records is controlled by explicit permissions. If you do not have the permissions, the system omits the VIP and/or locked records and displays an alert. Also, while the document tree on the Patient Records form lists all encounters, it filters documents based on your record view permissions and on the document types associated with each record view.

You can access documents from these points in the system:

  • the Patients tab (patient documents only)
  • the Requests tab (all documents, including the Authorization Request)

To view documents from a patient record

  1. Find a patient.
  2. In the detail pane of the Find Patient form, select a patient and click View/Edit Patient.
  3. Navigate to the Patient Records form and find the document in the document tree. Use the Expand button to quickly drill down to the document or page level.

You can filter the document list to reduce the volume of documents in the list.

Use these options in any combination:

- Set/Update
- Record View
- Sort By options
- Only Disclosure Docs
- Show Versions
  1. Do either of the following:
- To open specific documents in a document viewer window, select one or more documents in the document tree and click View Selected.
- To open all eligible documents in a viewer window, click View All.

The system opens the document viewer in a separate window. You can use view and print functions provided by the document viewer application as needed. See online help available with the viewer if you need user instructions.

To view documents on a request

  1. Find and open a request.
  2. Go to the Requests > Patient Information form.
  3. Select a patient in the "Patient(s) on this Request" box.
  1. Do one of the following:
- To open specific patient documents in a viewer window, select one or more documents in the document tree and click View Selected.
- To open all eligible patient documents in a viewer window, click View All.
- To open the Authorization Request, click the View Authorization Request button.
- To open invoices, go to the Invoice History screen, select an invoice, and click View.

The system opens the document viewer in a separate window. You can use view and print functions provided by the document viewer application as needed. See online help available with the viewer if you need user instructions.
