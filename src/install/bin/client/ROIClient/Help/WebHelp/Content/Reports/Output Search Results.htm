Output Search Results

When working with reports, a user can export the search results for any request. This allows for easier viewing of all the requests that require additional work.

All the existing functionality of performing the search, and permissions, auditing, searching, etc. remains the same.

To open the form

  1. Select Reports, run the applicable report, and click Export.

    OR

    Select Requests > Find/Create a Request, search for the request, highlight the line item, and click Export Results. The system displays the Export Report dialog box.
  1. Enter a file name.
  2. Click the drop-down list and select the file format.
  3. Click the drop-down list and select the drive letter in which to save the report.  The system displays a list of all folders on the selected drive.  
  4. Click on a folder to drill down until the desired location displays.
  5. Click Save.

Item descriptions

File Name

The name for the search results file.

File Format

The format of the exported file.  Options include PDF, XLS, RTF (Microsoft Word editable), HTML, and CSV.

Save In

The location where the file is saved.

Save

Saves the current selections.

Cancel

Discards any selections and closes the dialog box.
