Invoice dialog box

The Invoice dialog box displays information for an invoice, and allows editing of individual items such as adjustments and payments.

To open the form

  1. Select Requestors > Account Management.
  2. On the Account Management form, double-click on the applicable invoice line item OR select an invoice line item and click View.  The system displays the Invoice - ### dialog box.

Item descriptions

Invoice details

Contains the following information:

  • Date Created
    The date this invoice was created.
  • Status
    The status of the invoice.
  • Total Charges
    The total charges from invoice creation.
  • Total Adjustments
    The total of all applied adjustments.
  • Total Payments
    The total of all applied payments.
  • Unbillable Amount
    The charges that are not billable.
  • Current Balance
    The total charges minus unbillables, adjustments, and payments.

Details grid

Contains the following information:

  • Total Charges
    All charges for the selected invoice.
  • Payment
    All payments applied to the invoice.
  • Amount
    The amount of the payment.
  • Date
    The date the transaction occurred.
  • Edit
    Opens the Payments dialog box.

OK

Closes the dialog box and returns to the Account Management form.

View Invoice

Opens the Preview of Invoice dialog box.
