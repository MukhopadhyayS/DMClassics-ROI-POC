Payment Methods form

The Payment Methods form allows you to view, add, modify, or delete payment methods that users can select when posting payments for ROI requests.  Examples include cash, check, and credit card.

To open the form

Select the Administration tab > Billing Maintenance > Payment Methods.

Item descriptions

Main Content Pane

Payment Methods grid

Displays the payment methods and associated information currently defined in the system. The grid title is preceded by the number of existing payment methods.

The Payment Methods grid contains the following columns:

  • Payment Name
    Names of the payment methods currently defined in the system.
  • Description
    Descriptions of the payment methods (optional).

Create New

Allows you to enter the information needed to add a new payment method.

Delete

Deletes the selected payment method.

Detail Pane

Payment Name

Specifies a name for the payment method. Payment method names must be unique. They are not case sensitive.

You can change the name for a payment method at any time.

Form Field

Indicates whether to display a Note in the Payment dialog box.

Description

Provides a description for the payment method.

Save

Saves the new or modified payment method.

Cancel

Cancels the payment method you were adding or any unsaved changes you made to an existing payment method.
