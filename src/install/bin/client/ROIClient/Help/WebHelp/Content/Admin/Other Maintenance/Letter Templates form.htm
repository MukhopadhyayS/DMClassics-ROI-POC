Letter Templates form

The Other Maintenance > Letter Templates form allows you to create, modify, upload, or delete letter templates, as well as change their attributes.

Multiple letter templates are delivered with the system.

Each letter template is associated with a letter type.

  • The "Cover Letter," "Invoice," and "Pre-Bill" templates are associated with letter types of the same name.
  • The Requestor Statement Letter template is associated with the "Statement" letter type.
  • The remaining templates are associated with the "Other" letter type.

You can associate multiple letter templates with the Cover Letter, Invoice, Pre-Bill and Statement letter types and select one of them as the default template for the letter type. You cannot select a default letter template for the Other Letter type because the subject matter varies.

Facility-specific templates

All letter templates reside at a central location and are available throughout the enterprise. If you need to specify certain templates for specific facilities, you can do so by using a naming convention designed for that purpose. For example, you can add a prefix to the file name that identifies the facility.

Non-ROI templates

The templates delivered with the system are related to ROI requests. You can add templates for other purposes as well. Be sure to indicate the purpose in the template name (and description). These templates should be associated with the "Other" letter type category.

Creating a new letter template

  • The easiest way to create a new letter template is to download, edit, and upload an existing template.
  • Be sure to change the file name if you do not want to overwrite the original template when you upload the file.
  • If you want to create a new letter template from scratch, you also need to know how to use Word's Merge function as well as which database fields can be included on a letter template.
  • Use Microsoft Word to create the file and save the file with the .docx extension.  

To open the form

Select Administration > Other Maintenance > Letter Templates.

Item descriptions

Main Content Pane

Letter Templates grid

Displays the letter templates and associated information currently defined in the system. The grid title is preceded by the number of existing letter templates.

The Letter Templates grid contains the following columns:

  • Default

    Check marks indicate the default letter templates for the letter types. (Does not apply to the "Other" letter type.)
  • Letter Type

    Functional categories of letter templates. Each letter template must be associated with a letter type.
  • Name

    Names associated with the letter templates currently defined in the system. These names will display to users for selecting a template.
  • Description

    Descriptions of the letter templates. Descriptions are optional.

Create New

Allows you to upload a letter template, making it available for users to generate standard letters for ROI requests.

Delete

Deletes the selected letter template. You can delete templates at any time, including the pre-defined templates.

Download

Downloads the selected letter template so you can change it, and then upload the revised template to make it available to users.

Details Pane

Default

If checked, specifies the default letter template for the letter type. (Does not apply to the Other letter type.)

Letter Type

Indicates the functional category for a letter type. All letter templates are associated with a letter type.

Name

Specifies the name for a letter template. The names must be unique. They are not case sensitive.

Upload File

Specifies the path and file name for a letter template you want to upload. The system automatically populates this field when you click Browse and then browse to and select a template.

Browse

Allows you to browse to and select the template you want to upload.

Description

Provides a description for the letter template. You can change the description at any time. Descriptions are optional.

Save

Saves the letter template you uploaded or any changes you made to its attributes.

Cancel

Cancels uploading, downloading, or deleting a letter template, or any unsaved changes you made to its attributes.
