Update Request Status dialog box

The Update Request Status dialog box enables you to view and/or modify status information for a request.

To open the form

  1. Select the Requests tab.
  2. Search for and select a request.
  3. Click View/Edit Request.
  4. Click Update Request Status.

Item descriptions

Status

Displays the current status and allows you to select a new status. Displays only those status options you are permitted to use and only those options that are valid selections based on the current status.

  • Always displays Completed and Logged
  • Canceled displays only if you have permissions to cancel a request
  • Denied displays only if you have permissions to deny a request
  • Pended displays only if you have permissions to pend a request
  • If documents have been released, does not display Denied or Canceled
  • Never displays Auth-Received, Auth-Received Denied, or Pre-Billed

Reasons for selected status

Displays a list of:

  • All standard reasons associated with the current status AND
  • All free-form reasons that already exist for this request in oldest to newest order

Displays a checkbox for each reason and allows you to select one or more reasons to associate with the current status or a status change. When checked, the reason displays on the Request Information form.

Requires you to enter a reason if you change the status to Canceled, Denied, or Pended.

For free-form reasons only:

  • Displays the delete icon ()  
  • Allows you to edit existing free-form reasons

Add Freeform Reason

Allows you to add a free-form reason for this request.

Always enabled.

When clicked, creates a text box in the Reasons box, where you can enter text.

Save

Saves any checked reasons and returns to the Request Information form.

Enabled only after you make a change.

Cancel

Cancels the status change, discards any entries you made, and returns to the Request Information form.

Enabled only after you make a change.
