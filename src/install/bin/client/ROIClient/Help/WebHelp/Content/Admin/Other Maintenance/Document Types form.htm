Document Types form

The Document Types form allows you to flag multiple document types in a code set as disclosure documents. You can also flag documents as Meaningful Use documents.

The designations can vary per code set. For example, a document type can be selected as a disclosure document or Meaningful Use document in one code set but not in another. By using different code sets, the document type designations can vary by facility.

Designating a document type as a disclosure document does not modify the document.

Note: Flagging disclosure documents affects only the ROI Module.

To open the form

Select the Administration tab > Other Maintenance > Document Types.

Item descriptions

Code Set

The code set to which the document types belong.

Note: Code sets are maintained in OneContent. For information, see the OneContent online help.

Document Types grid

Displays the document types that belong to the code set. The grid title is preceded by the number of existing document types in the code set.

The Document Types grid contains the following columns:

  • Document Type

    Names of the document types.
  • Disclosure

    If checked, designates the document type as a disclosure document for ROI purposes. Multiple selections are allowed.
  • Auth Request

    This column is not in use at this time. Checking items in this column has no effect.
  • MU Document Type

    Allows you to assign a Meaningful Use label to each document type. Options are: None, Clinical Summary, Discharge Summary, and Discharge Instructions.

Save

Saves any changes to document type designations.

Cancel

Cancels any unsaved changes to document type designations.
