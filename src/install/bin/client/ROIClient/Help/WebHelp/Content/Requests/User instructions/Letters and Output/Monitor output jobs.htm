Monitor output jobs

This function enables you to view a list of all active (that is, all uncompleted) output job requests. You can also resubmit, pause, resume, and cancel jobs. Items displayed on the screen depend on your user permissions, the output settings configured for your system, and the state of the existing output jobs.

You can access the Monitor Jobs dialog box in two ways: by selecting the Security > Monitor Jobs menu item, or by clicking the printer icon in the left pane.

Jobs are retained for a period of time (in hours) specified by the output services configuration, beginning when the job was completed. When the period expires, eligible jobs are automatically purged. The system automatically retries failed jobs until the retry period expires. If the job still fails, the system sets its status to Failed.

From the menu bar, select Security > Monitor Jobs. The system displays the Output Job Monitoring window.  See the OneContent™ online help under Admin > System Monitoring > Output Job Monitoring for instructions on working in this window.

Note: This feature is not supported in Internet Explorer 11. If you have Internet Explorer 11 installed, open the One Content application and check the status of the submitted jobs by selecting the Current User > Output Jobs function.
