Maintain billing tiers

To add, modify, or delete billing tiers to be used for calculating charges for ROI requests, do the following:  

  1. Access the Billing Tiers form.
  2. Follow the instructions below for the task you want to perform.

Note: When adding or changing a billing tier, you can set any or all values to $0.00.

To add a new billing tier

  1. In the main content pane, click Create New.
  2. In the detail pane, enter a Tier Name and select the Media Type to associate with the billing tier. The media type must already be defined in the system.
  3. Optionally, enter a description for the billing tier.
  4. In the Rates section, enter the Base Charge.
  5. In the page range fields, enter a page range and the cost per page.
  6. Optionally, click Add a new rate tier and enter a page range and cost per page for those pages.

The starting page of the next tier must be the ending page of the previous tier plus one. You can add as many page-level rate tiers as you need. The tiers you add are indicated by the delete icon .  You can delete these tiers as needed by clicking the icon.

  1. Click Save.

To modify an existing billing tier

Note: You cannot change the media type associated with a system-defined billing tier. You can change the description and per-page rates. System-defined billing tiers are indicated by theicon.

  1. In the main content pane, select the billing tier you want to modify.
  2. In the detail pane, modify the information as needed.
  3. Click Save.

To delete a billing tier

Note: You cannot delete system-defined billing tiers. They are indicated by theicon.

  1. In the main content pane, select the billing tier you want to delete.
  2. Click Delete.
  3. In the confirmation message, click Delete.
