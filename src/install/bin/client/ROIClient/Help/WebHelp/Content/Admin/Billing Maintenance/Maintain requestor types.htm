Maintain requestor types

To add, modify, or delete requestor types for processing ROI requests, do the following:

  1. Access the Requestor Types form.
  2. Follow the instructions below for the task you want to perform.

To add a new requestor type

  1. In the main content pane, click Create New.
  2. In the detail pane, enter a name for the requestor type and other required information.
  3. Enter optional information as needed.
  4. Click Save.

To modify an existing requestor type

Note: You cannot change system-defined requestor type names. They are indicated by theicon.

  1. In the main content pane, select the requestor type you want to modify.
  2. In the detail pane, change the information as needed.
  3. Click Save.

To delete a requestor type

Note: You cannot delete system-defined requestor types or a requestor type that has requestors associated with it.

  1. In the main content pane, select the requestor type you want to delete.
  2. Click Delete.  
  3. In the confirmation message, click Delete.
