List of Deficiencies dialog box

The List of Deficiencies dialog box enables you to view all deficiencies associated with an encounter. You can also filter the list to display only linked or only unlinked deficiencies.

To open the form

On the Patient Records form, double-click any encounter that displays the deficiency icon (), or click the deficiency icon directly.

Item descriptions

Display options

Determines the nature of the deficiency list. Options are:

  • All
    Lists all deficiencies associated with the encounter.
  • Linked Only
    Lists only deficiencies for the encounter that have been assigned to a physician.
  • Unlinked Only
    Lists only deficiencies for the encounter that have not been assigned to a physician.

Deficiencies grid

Lists deficiencies associated with the currently selected encounter, according to the selected Display option (All, Linked Only, or Unlinked Only), and provides information about each deficiency.

Initially sorted by Document Type in alphanumeric ascending order. Can be sorted by any column.

Information includes:

  • Physician
    Name of the physician responsible for the deficiency.
    Initial sort is alphanumeric ascending.
  • Type
    The type of the deficiency (signature, missing text, or missing document).
    Initial sort is alphanumeric ascending.
  • Document Type
    The name of the document that has the deficiency.
    Initial sort is alphanumeric ascending.
  • Reason
    The reason for the deficiency.
    Initial sort is alphanumeric ascending.
  • Status
    The status of the deficiency (Complete, Incomplete).
    Initial sort is incomplete followed by complete.
  • Locked By
    The name of the user who currently has the deficiency record locked.
    Initial sort is numeric ascending.

Close

Closes the dialog box and returns you to the Patient Records form.
