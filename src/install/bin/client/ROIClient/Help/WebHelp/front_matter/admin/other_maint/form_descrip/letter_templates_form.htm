Letter Templates form

The Letter Templates form allows you to create, modify, upload, or delete letter templates, or change their attributes.

Six letter templates are delivered with the system. They are Cover Letter, Invoice, Pre-Bill, Cancel, Deny, and Pend. These templates can be used for generating standard letters related to ROI requests.

Each letter template is associated with a letter type. The Cover Letter, Invoice, and Pre-Bill templates are associated with letter types of the same name. The remaining templates are associated with the "Other" letter type. You can associate multiple letter templates with the Cover Letter, Invoice, and Pre-Bill letter types and select one of them as the default template for the letter type. You cannot select a default letter template for the Other letter type because the subject matter varies.

Facility-specific templates

All letter templates reside at a central location and are available throughout the enterprise. If you need to specify certain templates for specific facilities, you can do so by using a naming convention designed for that purpose. For example, you can add a prefix to the file name that identifies the facility.

Non-ROI templates

The templates delivered with the system are related to ROI requests. You can add templates for other purposes as well. Be sure to indicate the purpose in the template name (and description). These templates should be associated with the Other letter type category.

Creating a new letter template

The easiest way to create a new letter template is to download, edit, and upload an existing template. Be sure to change the file name if you do not want to overwrite the original template when you upload the file.

If you want to create a new letter template from scratch, you need to use a rich-text-format text editor such as Microsoft Word and save the file with the .rtf extension.  

To open the form

Select the Administration tab > Other Maintenance > Letter Templates.
