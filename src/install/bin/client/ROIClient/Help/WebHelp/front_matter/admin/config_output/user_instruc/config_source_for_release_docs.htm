Configure the source for release documents and letters

To enable, disable, or edit information for the source or release documents and letters, do the following:

Note: The source depends on the type of documents or letters you are supporting, MPF or ROI (non-MPF).

  1. On the menu bar, select Administration > Configure Output Settings.

The system displays the Configure Output Settings dialog box.

  1. Select the Sources tab.

  2. Follow the instructions below for the task you want to perform.

 

To configure the source for retrieving documents and letters

  1. In the Sources grid on the Sources tab, select a source:

    • Select MPF to edit information for the MPFW Web server.

    • Select ROI to edit information for the ROI Web server.

  1. Click Edit.

The system displays the Edit Source dialog box.

  1. In the Edit Source dialog box, do one or both of the following:

    • Change the Description for the source.

    • Change the values in the Set the properties for the source and then click OK group box.

  1. Click OK.

 

To enable or disable a source

  1. On the Sources tab, select MPF or ROI.

  2. For the Enable option, select Yes or No.

  3. Click Close.
