Request History form

The Request History form allows authorized users to:

  • View a list of requests for a specific requestor.

  • Select a request to view or edit.

  • Create a request.

To open the form

  1. Select Requestors > Find Requestor.

  2. Search for and select a requestor. Then click View/Edit Requestor.

  3. Select Request History in the left pane.

Note: You can access this form at any time while viewing and/or editing requestor information.
