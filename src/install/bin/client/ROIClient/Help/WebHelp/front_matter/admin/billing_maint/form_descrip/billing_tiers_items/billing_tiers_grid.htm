Billing Tiers grid

Displays the billing tiers and associated information currently defined in the system. The grid title is preceded by the number of existing billing tiers.

The Billing Tiers grid contains the following columns:

  • Icons that indicate the type, or source, of the billing tier data:

  • The system icon indicates billing tiers that are system-defined. You cannot delete system-defined billing tiers or change the media type associated with them. You can edit the description and rates per page.

  • The user icon indicates that the data is sample data or is user-created. You can delete the billing tier or change the associated information.

  • Tier Name
    Names of the billing tiers currently defined in the system.

  • Media Type
    Names of the media types associated with the billing tiers.

  • Description
    Descriptions of the billing tiers (optional).
