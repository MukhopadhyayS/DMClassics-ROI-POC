Create a request with patient requesting

This task assumes that you want to create a request for a patient who is requesting his or her own records.

Tips: Selecting multiple rows...

This procedure instructs you to perform a patient search and to select at least one row from the search results grid. If the system displays multiple rows for the same patient, such as when the patient visits different facilities, you can select all applicable rows to include all of this patient's records on the request. The system warns you of a potential patient mismatch if the rows you select appear to be for different patients, and it prompts you to select one patient as the requestor. In this case, records for only the patient you select as the requestor will be included on the request.

The system may return related patients whose records you want to include on the request. This scenario arises when, for example, multiple family members are treated and one of them (a parent, for example) requests the records for all of them. To include records for related patients on the same request, you must declare the patient requestor, and then manually add the other patients to the request using the Requests > Patient Information tab. To avoid including records for a non-related patient, exercise extreme caution when adding patients to this type of request.

 

  1. Perform a patient search.

  2. On the Find Patient form, select the patient who is requesting records and click Patient Requesting. The system searches for a matching requestor record.

Note: If the patient has never requested records before, the displayed dialog box omits the following items: Address, Phone, Fax, and New Requestor button. Also, it labels the Select Requestor button as Select Patient as Requestor.

  1. At the Create New Request with Patient Requesting dialog box:

    • If the system finds one or more requestors that potentially match the patient, the system displays all found names in the grid. Select the matching entry from the list and click Select Requestor. The system proceeds to the Requests > Request Information form.

    • If no match is found, the system displays and highlights the patient's name in the grid. Click Select Patient as Requestor. The system then displays the Save New Patient Requestor dialog box.  

Select Pre-payment Required and/or Certification Letter Required (when appropriate), review the patient's information and correct it, if needed, and then click Save New Requestor. The system adds the requestor record and proceeds to the Requests > Request Information form.

  1. At the Request Information form, select a Request Reason and a Receipt Date, then click Save.

The request has been created. You can continue building the request now or later.

To continue building the request, see Add documents to a request.
