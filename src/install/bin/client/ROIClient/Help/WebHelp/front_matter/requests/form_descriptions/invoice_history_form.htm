Invoice History form

The Invoice History form enables you to view and/or output existing invoices and/or pre-bills for the currently selected request.

To open the form

  1. Select the Requests tab.

  2. Search for and select a request.

  3. Click View/Edit Request.

  4. Select the Invoice History function.
