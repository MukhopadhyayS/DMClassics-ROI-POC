Post payments or adjustments to a request

This task enables authorized users to post payments, debits, and credits to a request. You can post amounts both before and after you release documents, and can post all types of transactions at once before you save your entries. The system recalculates the balance on the request with each transaction you enter.

Rules for posting payments

  • You cannot update or delete a payment once you save it.  

  • If a payment is recorded incorrectly, you can post a debit adjustment to counter it.

  • During the life of a request, you can apply multiple payments.

  • You must post separate payments for separate payment methods. For example, if the requestor pays $50 by check and $50 by credit card to apply to the same request, you must post two separate payments.

  • You can split a consolidated payment across multiple requests. For example, a $100.00 payment received as a single check can be posted against two $50 requests.

Rules for posting adjustments

  • You cannot update or delete an adjustment once you save it.  

  • During the life of a request, you can apply multiple adjustments.

Instructions assume you are accessing an existing request that has documents selected for release and that billing information already exists for the request you open.

Note: If you make a mistake and want to begin again, you can click Revert to discard all unsaved entries and return the request to its last saved state.

 

  1. Find a request.

  2. Go to the Billing & Payment form and find the Adjustments & Payments for this Request box.

  1. Follow instructions below for the task you want to perform.

 

To post a debit adjustment

  1. In the Adjustment group box, select Debit and enter an amount.

  2. Optionally select a reason or enter a free-form reason for the debit.

  3. Click Add.

  4. Click Save to record your entries. Click Revert to discard unsaved entries.

 

To post a credit adjustment

  1. In the Adjustment group box, select Credit and enter an amount.

  2. Optionally select a reason or enter a free-form reason for the credit.

  3. Click Add.

  4. Click Save to record your entries. Click Revert to discard unsaved entries.

 

To post a payment

  1. In the Payment group box, select the method of payment and enter an amount.

  1. Optionally enter a note about the payment (such as a check number).

  2. Click Add.

  3. Click Save to record your entries.

Note: If a payment is posted to the wrong request, you can reverse it by posting a debit adjustment.
