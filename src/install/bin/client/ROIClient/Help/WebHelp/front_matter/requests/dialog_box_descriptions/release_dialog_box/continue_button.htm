Continue button

Continues with the release of documents.

Enabled only when any of the following are true:

  • Neither the cover letter or the invoice check box is checked

  • Either the cover letter or the invoice check box is checked and a template has been selected

  • Both the cover letter and the invoice check boxes are checked and templates have been selected for both

If a cover letter and/or invoice are checked, launches the Preview of the letter and/or invoice. If both are checked, the system appends the invoice to the letter in the preview.

If neither cover letter or invoice is checked, closes the Release dialog box and launches the dialog box for the output method (Print, Fax, or Save as File) specified on the Shipping tab of the Billing & Payment form.
