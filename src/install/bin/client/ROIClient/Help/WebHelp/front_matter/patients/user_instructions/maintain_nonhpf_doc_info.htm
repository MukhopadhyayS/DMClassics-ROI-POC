Maintain non-MPF document information

This task enables you to add, change, or delete information about non-MPF documents. You can maintain non-MPF documents for both MPF and non-MPF patients. You can add, change, or delete information for only one document at a time.

A non-MPF document resides outside of the McKesson Patient Folder and ROI system, but billing charges must be captured if it is released to a requestor. The information you enter is used to generate the billing charges.

Note: You can also create or edit non-MPF documents from the Requests > Patient Information form. See Add or edit non-MPF document information for instructions.

 

To create non-MPF documents

  1. Find a patient.

  2. Go to the Patient Records form and click New Document.

  3. On the Create New Non-MPF Document dialog box, enter document information.

  1. Click Save.

 

To change non-MPF documents

When you change information for a non-MPF document, the system automatically applies the updates to any request that includes the document. Once a non-MPF document has been released on any request, you can edit only the comments associated with the document. This ensures the integrity of the Accounting of Disclosure reports.

The document tree on the Patient Records form lists all encounters, but filters documents based on your record view permissions and on the document types associated with each record view.

  1. Find a patient.

  2. Go to the Patient Records form, and double-click the non-MPF document in the document tree.  

  1. On the Edit Non-MPF Document dialog box, change or enter information as needed.

  2. Click Save.

 

To delete non-MPF documents

Note: You cannot delete information for a non-MPF document that has been released.

  1. Find a patient.

  2. Go to the Patient Records form, and double-click the non-MPF document in the document tree.

  3. On the Edit Non-MPF Document dialog box, click Delete.

  4. At the confirmation prompt, click Delete.
