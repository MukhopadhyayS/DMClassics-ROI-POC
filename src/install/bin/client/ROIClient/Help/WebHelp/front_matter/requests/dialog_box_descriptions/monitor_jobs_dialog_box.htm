Monitor Jobs dialog box

The Monitor Jobs dialog box enables you to view output jobs currently in the output queues. Authorized users can hold and release queue activity, and can act on output jobs.

To open the form

On the menu bar, select Security > Monitor Jobs.

OR

Click the printer icon () in the left pane.

The dialog box displays two tabs: Job View and Queue View.

Note: Your system may not display all items described below. Some items depend on your user permissions and the output settings configured for your system.
