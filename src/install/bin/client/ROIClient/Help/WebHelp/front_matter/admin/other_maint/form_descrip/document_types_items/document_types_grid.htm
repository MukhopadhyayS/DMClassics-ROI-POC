Document Types grid

Displays the document types that belong to the code set. The grid title is preceded by the number of existing document types in the code set.

The Document Types grid contains the following columns:

  • Disclosure
    If checked, designates the document type as a disclosure document for ROI purposes. Multiple selections are allowed.

  • Auth Request
    This column is not in use at this time. Checking items in this column has no effect.

  • Document Type
    Names of the document types.
