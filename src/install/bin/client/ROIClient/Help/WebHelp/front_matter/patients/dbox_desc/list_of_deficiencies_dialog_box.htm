List of Deficiencies dialog box

The List of Deficiencies dialog box enables you to view all deficiencies associated with an encounter. You can also filter the list to display only linked or only unlinked deficiencies.

To open the form

On the Patient Records form, double-click any encounter that displays the deficiency icon (), or click the deficiency icon directly.
