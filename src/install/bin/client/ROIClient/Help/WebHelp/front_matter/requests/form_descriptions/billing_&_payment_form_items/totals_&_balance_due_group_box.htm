Totals & Balance Due group box

Displays a summary of costs, adjustments, and payments for this request.

Contains the following information.

  • Pending Release Cost
    Displays the sum of the pending release cost.
    (Also displays above the Billing and Shipping tabs.)

  • Previously Release Cost
    Displays the sum of all previous costs for this release (sum of each release cost).

  • Charge History button
    Displays only if the request has been previously released.
    Launches the Charge History dialog box, which displays the previous charge history for this request.
    Charge history includes, for each release event, the date of release and document, fee, and shipping charges.

  • Total Request cost
    Displays the sum of both the Pending Release and Previously Released Costs.

  • Adjustments & Payments Total
    Displays the sum total for adjustments and payments for this request.
    Displays a credit balance in green and a debit balance in black.

  • Balance Due
    Displays the sum total of all rows in the group box plus any shipping charges.
    Can be a balance due (in red), a credit balance (in green), or a zero balance (in black).
