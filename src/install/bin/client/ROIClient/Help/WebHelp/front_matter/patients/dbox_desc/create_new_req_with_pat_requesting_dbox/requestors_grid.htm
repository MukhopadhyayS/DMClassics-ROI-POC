Requestors grid

Displays the requestor search result set and allows you to a select requestor.

Initially sorted by Requestor Name in alphanumeric ascending order. Can be sorted by any column.

Information includes:

  • Requestor Name
    The name of the patient requestor.

  • Address
    The street address, city, state, and zip code of the patient requestor.
    Initial sort is alphanumeric ascending.

  • Phone
    The home phone for a patient requestor.
    Initial sort is numeric ascending.

  • Fax
    The fax number associated with the requestor's main address.
    Initial sort is numeric ascending.

  • DOB
    The patient requestor's date of birth in mm/dd/yyyy format.
    Initial sort is date ascending.

  • SSN
    The patient's Social Security Number.
    Initial sort is numeric ascending.

  • EPN (displays only for EPN systems and Requestor Type is Patient)
    Displays the patient's enterprise person number.
    Initial sort is alphanumeric ascending.
