About Other Maintenance

The Other Maintenance functions provide ROI administrators the capabilities described below.

This functionality is accessed by selecting Other Maintenance on the left pane of the Administration tab.

Designate document types as disclosure documents

Administrators can designate document types as disclosure documents for ROI purposes. This provides ROI users the ability to filter the document tree to display only disclosure documents.

Maintain letter templates

The ROI Module provides predefined letter templates that can be used for generating letters and invoices related to ROI requests. Administrators can download, modify, and upload the templates to meet the needs of the organization. They can also create new templates using a rich-text-format text editor such as Microsoft Word.

Define standard notes

Administrators can define standard notes that users can add to letters and invoices associated with ROI requests.

Mask Social Security Numbers

Administrators can set the option to mask Social Security Numbers so they do not display during online ROI sessions or in reports, letters, and invoices generated by ROI users.
