View release history

This function enables an authorized user to view the history of document releases for a request.

A request can have zero to many releases. The system lists all documents for all releases for the request. The system initially displays  a summary of the total number of all releases and the total number of all pages released for the request. If you filter the result set, the system recalculates the totals. The system notifies you if no releases exist for the request.

Releases are listed in chronological order (beginning with the most recent), then in patient name order. You can sort the list by any column. In addition, you can filter the list to display only releases for a specific patient, releases performed by a specific ROI user, and/or releases that occurred on a specific date or within a date range.

  1. Find a request.

  2. Go to the Requests > Release History form.

  1. Follow instructions below for the task you want to perform.

 

To view release events

View release events in the Releases grid. By default, the system displays all releases for all patients on the request. Releases initially display in chronological order (beginning with the most recent) by patient name.

 

To sort release events

To sort events in a different order, click in any column heading to sort the list by that value. Initial sort order for Date | Time is most recent to oldest. Initial sort order for the other columns is alphanumeric ascending.

 

To filter release events

To filter the list, do any of the following, in any combination:

  • Select the Filter Patients option, then select one or more patient names in the corresponding list. If you select only one patient name, the system displays that patient's name, gender, DOB, SSN, and EPN / Facility / MRN information.

  • Select a specific user name in the User Name box. The default selection is All.

  • Select a standard date range in the Release Date box: All, 30 days, 60 days, or 90 days. The default selection is All.

  • To set a specific date or a date range of your choice, select Custom Range in the Release Date box, then select dates for the From and To boxes. (Once you select a date, you can change the month, day, or year value using either the keyboard or the drop-down calendar.)
