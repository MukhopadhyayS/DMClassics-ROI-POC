Enter billing information

This function enables an authorized user to enter billing information for a request. You can enter billing information when you create the request or at any time during the request lifecycle, as long as the status is not Denied or Canceled.

The system automatically calculates the Balance Due total as you enter information.

 

  1. With the request open, go to the Requests > Billing & Payment form. (For an existing request, find and open it.)

Note: The Billing & Payment function is accessible only after you have clicked the Save & Bill button on the Patient Information form to generate billing for the request. See Release documents for details.

  1. Go to the Billing tab.    

  1. In the Document Charges group box, enter the number of copies you are releasing of each item.

Note: Documents are automatically assigned a billing tier, but you can override the assignment on the Patient Information form. See Override a billing tier for more information.

  1. In the Fee Charges group box, use the template displayed in Billing Type (if one is associated with the current requestor type) or select another template. The system applies the standard fees that are associated with the template, and you can change fee charges as needed. You can also create custom fees as needed.

If no template is selected, no standard fees are applied. You can still create custom fees as needed.

  1. In the Adjustments & Payments for this Request group box, enter adjustments and/or payments.     

The system keeps a running total (debit or credit) as you enter information.

  1. Click Save to save your entries. To discard unsaved entries and start again, click Revert.

You can also click Release, Pre-Bill, or Invoice, when available, depending on how you want to proceed.
