Request Information form

The Request Information form enables you to view and/or change information for the currently selected request. You can add or remove patient records to or from the request, view or edit the patient profile, select which records to release, add or edit non-MPF documents, and either save the request for later processing or save it and bill it at the same time.

Note: You can access this form at any time while creating a new request or viewing an existing request.

To open the form

  1. Select the Requests tab.

  2. Do either of the following:

    • Click New Request (to create a request).

    • Search for and select a request, then click Open / View Request (to edit an existing request).
