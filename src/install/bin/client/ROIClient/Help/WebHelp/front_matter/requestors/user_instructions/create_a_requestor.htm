Create a requestor

You can create requestors by clicking the New Requestor button from these screens:

  • The Requestors > Find Requestors form (this method is described below)

  • The Select Requestor (or Change Requestor) dialog box

All methods allow you to enter the same data. However, the system enables or disables data entry fields dynamically, depending on the Requestor Type setting. Basically, patient and non-patient fields are mutually exclusive.

 

  1. Select the Requestors tab.

  1. Click New Requestor.  

  1. Enable the prepayment and/or certification letter flags, when appropriate. You may need to consult your supervisor to determine whether these options should be enabled for the requestor you are creating.

  2. Select the type of requestor you are adding and enter a name.

Note: For a patient requestor type, the system splits the Name box into Last Name and First Name. It also displays the Patient EPN field if your system is EPN-enabled.

  1. If you are creating a patient requestor, enter the patient's EPN (if displayed), DOB, SSN, Facility, and/or MRN.

  2. Optional: Enter the requestor's main address, alternate address, and contact information as desired.

  1. Click Save.
