Queues grid

Displays the output queues and associated information currently defined in the system. The grid title is preceded by the number of existing output queues.

The Queues grid contains the following columns:

  • Type
    Types of queues currently defined in the system. The queue types are Print, Save as File, and Fax.

  • Queue Name
    Queue names associated with the queue types. The queue names you enter will display to users when they output documents and letters.

  • Description
    Descriptions of the queues.
