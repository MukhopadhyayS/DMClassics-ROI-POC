Enter shipping information

This task enables an authorized user to enter shipping information for a request. You can enter shipping information when you create the request or at any time during the request lifecycle, as long as the status is not Denied or Canceled.

Shipping charges are defined for each release within a request. This allows the system to accurately calculate shipping charges based on the total pages and output type for a specific release.

 

  1. With the request open, go to the Requests > Billing & Payment form. (For an existing request, find and open it.)

Note: The Billing & Payment function is accessible only after you have clicked the Save & Bill button on the Patient Information form to generate billing for the request. (See Release documents for more details.)

  1. Go to the Shipping tab.

  1. Select an output method.

  2. For Print and Save as File output methods, indicate whether the release will be shipped. (Shipping is not applicable if you selected the Fax output method).

If you select Yes for shipping, the system automatically populates address fields with the requestor's shipping address information. You can change information as needed. If you change information, the system prompts you to confirm that you want to apply the address changes to this shipment. (Changes are not saved in the requestor's profile.)

  1. At Shipping Method, specify the shipping agent.

  2. At Tracking Number, enter the shipment tracking number obtained from the shipping agent.

  3. At Shipping Charges, enter the shipping charges for the release. Obtain the shipping charges from the shipper's website, verbally from the shipper, or by some other means of communication with the shipping agent.

  4. Click Save. The system adds the shipping charges to the Balance Due total.

You can then click Release, Pre-Bill, or Invoice, when available, depending on how you want to proceed.
