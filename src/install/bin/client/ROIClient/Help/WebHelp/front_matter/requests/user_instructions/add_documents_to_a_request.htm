Add documents to a request

This function enables you to add one or more MPF and/or non-MPF documents to an ROI request. You can add documents when you create a request or later.

After a request has been initially released, if you add more documents to it and release them, the billing and payment calculation does not include base charges. The system calculates charges for only the new pages being released.

All documents for each patient on the request are available for selection, depending on your security permissions.

Note: To open an existing request that contains locked and/or VIP items, you must have the Access Locked Records and/or ROI VIP Status permission. The permissions enable the system to display locked and/or VIP items that you can include on the request.

  1. With the request open, go to the Patient Information form. (For an existing request, find and open it.)

  2. Conditional: If this request is for a single patient, skip to the next step. If this request includes multiple patients, select a patient name in the "x Patient(s) on this Request" box.

Note: Non-EPN systems can list the same patient multiple times, if the patient has encounters at different facilities. The Facility and MRN numbers combine to make a unique patient ID, even though the same patient is involved. An EPN system, on the other hand, will have only one entry for a patient who has encounters at different facilities.

  1. At the document tree, select the encounter(s), folder(s), document(s), or document page(s) for this patient that you want to release.

You can select the encounter or folder to include all documents it contains, and you can select specific documents or pages from an encounter or folder.  

Note: You can also change the Record View option, change the Sort By option, and/or select to show document versions or disclosure documents to affect the document list. A recommended practice is to always view disclosure documents for special instructions before fulfilling a request.

  1. Click the > button to add your selections to the Document(s) Selected for Release box.

  2. Repeat for the next patient, if one exists.

  3. Click Save. The request now contains documents to release.

You cannot release documents until you generate billing information. You can do that now or at a later time. If you are ready to generate billing information now, click Save & Bill and continue with instructions for Enter billing information.
