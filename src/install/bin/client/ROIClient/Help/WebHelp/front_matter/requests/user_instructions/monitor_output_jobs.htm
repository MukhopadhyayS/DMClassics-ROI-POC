Monitor output jobs

This function enables you to view a list of all active (that is, all uncompleted) output job requests. You can also resubmit, pause, resume, and cancel jobs. Items displayed on the screen depend on your user permissions, the output settings configured for your system, and the state of the existing output jobs.

You can access the Monitor Jobs dialog box in two ways: by selecting the Security > Monitor Jobs menu item, or by clicking the printer icon in the left pane.

Jobs are retained for a period of time (in hours) specified by the output services configuration, beginning when the job was completed. When the period expires, eligible jobs are automatically purged. The system automatically retries failed jobs until the retry period expires. If the job still fails, the system sets its status to Failed.

  1. From the menu bar, select Security > Monitor Jobs.

The system defaults to the Job View tab, selects all items in the boxes for Type, Status (except the Complete status), Queue, and User, and lists jobs that match the current selection criteria..

  1. View the list as needed or follow instructions below for the task you want to perform.

 

To filter the list of jobs

You can use any combination of filter criteria. The system lists all jobs based on the criteria you use.

On the Job View tab

On the Job View tab, the filter boxes are interdependent; the selections you make in one box drive the values displayed in the other boxes. To filter the job list:

  • At Type, select one or more queue types (Print, PDF, or Fax). By default, all types are selected.

  • At Status, select one or more job status options (Ready, Complete, Canceled, Canceling, Failed, Faxing, Paused, Pausing, Printing, Processing, or Resuming). By default, all options except Complete are selected.

  • At Queue, select one or more specific queue names, or All Queues.

  • At User, select one or more specific user IDs or All Users. To restrict the list to specific user IDs, click Select User.

    The system launches the "Select a user" dialog box. You can select one, some, or all users, but at least one user must be selected. When you are ready, click Select to populate the User box with your selections.

     

On the Queue View tab

To filter the job list on the Queue View tab, select the type of queue(s) to list (Print, PDF, and/or Fax). Then select a specific queue to list only jobs that reside in that queue.

 

To hold or release queue jobs, or refresh queue information

  1. On the Queue View tab, select a queue.   

  1. Click the button for the action you want (Hold, Release, or Refresh).

The system applies the Hold or Release action to all jobs in the queue if you are an administrator user. If you are a non-administrator, the system applies the action to only the jobs in the queue that you submitted. the system applies the Refresh action to the queue grid regardless of user permissions.

 

To resubmit, pause, resume, or cancel one or more jobs

  1. On either tab: Select one or more entries from the Jobs grid.

  2. Click the button for the action you want (Resubmit, Pause, Resume, or Cancel). Buttons are enabled depending on the current status of the job.

Notes: If you resubmit a PDF job, you cannot change the password that was originally used for the job. You cannot pause a job if it is currently printing, faxing, or saving to a file.

 

To view output settings

Click the View Output Settings button. The system displays the current settings in effect in read-only mode.

 

To refresh the list of jobs

Click the Refresh button in the lower part of the dialog box. The system displays current job information for the currently selected criteria.
