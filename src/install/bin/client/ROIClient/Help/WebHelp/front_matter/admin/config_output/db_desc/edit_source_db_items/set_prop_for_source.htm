Set the properties for the source and then click OK group box

  • Password
    Password used to connect to MPF WebStation. (Applies to the MPF source only.)

  • Port
    TCP port used to connect to MPF WebStation or the ROI Web server.

  • Protocol
    Protocol used to connect to MPF WebStation or the ROI Web server.

  • Host Name
    Server name or IP address for the MPF WebStation server or the ROI Web server.

  • User Name
    User name for connecting to MPF WebStation. (Applies to the MPF source only.)
