Document tree view

Allows you to navigate to and select records associated with the current patient.

For MPF patients, displays all folders (Global, MPF, and non-MPF). For non-MPF patients, displays only non-MPF folders.

  • "GLOBAL DOCUMENTS" folder lists all documents for the selected patient that are classified in MPF as global. Display is in chart order, most recent first.  Information includes document type name and admit date and time. Display is not affected by Filter, Record View, or Sort By selections.

  • If non-MPF documents exist for this patient, displays a "NON MPF DOCUMENTS" folder. The node lists documents, most recent first, with document type name, facility, department, and date of service. Display is not affected by Filter or Sort By selections.

  • Remaining folders represent MPF documents initially grouped by Encounter, Document Names, Version, and Page numbers. The system displays the encounter number, patient status, facility, and admit date. These folders can be sorted according to the Sort By option.

Also identifies encounters with one or more deficiencies () and, if you have adequate permissions, identifies VIP patients () and locked encounters ().
