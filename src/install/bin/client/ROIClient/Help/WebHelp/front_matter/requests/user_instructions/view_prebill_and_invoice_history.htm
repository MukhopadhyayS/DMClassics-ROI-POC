View pre-bill and invoice history

This task enables you to retrieve or recreate a previously created invoice or pre-bill letter for viewing or output. The system displays all pre-bills and invoices that have been generated for a request. All information is view-only, and you can view only one item at a time.

Required permissions are...

  • ROI Access Application AND

  • ROI Export to PDF (to generate output to a file) AND/OR ROI Print/Fax (to print or fax output)

 

  1. Find a request.

  2. Go to the Invoice History form.

  1. To view or output a pre-bill or invoice, select it from the grid and click View.

The system displays the preview dialog box (either Pre-Bill or Invoice) and allows you to select an output method. The system displays only the output options for which you have permission.

    • If you choose to output to a file, the system automatically assigns a file name based on request ID.

    • If you choose to fax the output, the system automatically populates the requestor's fax information and allows you to change it.

To help you monitor your output jobs, the system assigns an output job name based on request ID.
