Statements group box

Enables you to generate a statement and notes.

Items are:

  • Output Statement
    Triggers the system to create a statement to output.
    When selected, enables the Statement Template and Date Range fields.

  • Statement Template
    Lists all statement templates configured in the system and allows you to select one.
    Defaults to the template configured as the default selection.
    If no default is configured, then defaults to Please Select.

  • Date Range
    Specifies the date range to include in the statement.
    Preconfigured list of choices includes: 30, 50, 90, or 120 days, Year to Date, and All.

  • Select/Create Notes to add to the Invoice box and Create Freeform Note button

    • Displays only when the selected template is configured to support notes.

    • Displays all preconfigured notes associated with the template.

    • Allows you to select one or more notes to print with the cover letter.

    • Allows you to create additional freeform notes.
