Create New Device dialog box

Device queues are created to integrate third-party devices with ROI.  This includes devices such as those providing functionality to burn released documents and print labels containing patient information on CDs or DVDs.

To open the dialog box

  1. On the menu bar, select Administration > Configure Output Settings.

  2. Click the Devices tab.

  3. Click Create New.

Item descriptions

Device Type
Type of device. Open the drop-down list to display all available device types.

Device Vendor
Name of the vendor.  Open the drop-down list to display all available vendors.

Device Name
Name of the device.  

Device Description
Description of the device.  This is a free-form text field.

Host Name/IP
Name of the host or IP address of the device.

Port
The port associated with the host name or IP address.

Label Location
Location of the labels.  This must be a shared location.

Temporary PDF location
Location for the generated PDF files.  This must be a shared location.

OK
saves the devices with the options selected.

Cancel

Cancels creating a new device or any unsaved changed made for an existing device.

Test
Verifies the settings entered allow communication between ROI and the configured device.  A status message displays either a confirmation or error message.
