Disc Queue

Item descriptions

Queue Type
Specifies the type of queue.

Queue Name
Provides a name for the output queue. This name will be displayed to the user when the user selects to output documents or letters.

Queue Description
Provides a description for the output queue.

Device
Specifies the device to use.

Default Label Template
Specifies the name of the template to use for the disc label.

Note: The label template is created using Rimage CD Designer. Through this application you specify the parameters to appear on the label (for example, patient name, date of birth, date and time and so on).  More details about the parameters...

Specific parameters are created using the Rimage CD Designer to specify the disc label information.  

Both variable and static parameters (for example, the hospital logo) can be specified.

To ensure this information appears correctly on the  label, you must use the parameters below.  Set each parameter as a Merge field.  See the Rimage Label Designer documentation for details on how to create merge fields.

Note: If the request includes more than one patient or encounter, the patient or encounter values will be displayed separated by commas.  For example, if a request contains patients John Doe and Jane Smith, the patient name label will display John Doe, Jane Smith.

Default Disc Type
Specifies the disc type - either CD or DVD.

Password Required for PDF Files
Specifies any protection for the file.  If you select No, the system allows the user to password protect the file when it is output.  If you select Yes, the system protects the document. Documents include: MPF Documents, Supplemental Documents, Invoices, Pre-Bills, Letters, Attachments and so on.

Default Password
Indicates the password required for the PDF Files.

OK
Saves the queue with the options you entered.

Cancel
Cancels adding a new queue or any unsaved changes you made for an existing queue.
