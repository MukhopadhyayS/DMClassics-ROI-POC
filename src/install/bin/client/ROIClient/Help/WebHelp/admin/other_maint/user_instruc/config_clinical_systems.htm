Configure clinical systems

This task enables administrators to add and maintain a clinical system profile for each McKesson Patient Folder facility that provides documents to the ROI Module.

A profile represents the clinical system that is used by a facility. Each facility defined in the MPF Workstation application can be mapped to only one clinical system profile at a time. The association enables ROI Module users to "pull" Clinical Summaries on demand from the facility's clinical system when, for example, the information is needed for an ROI request.

You will need the following information for the clinical system profile you want to configure.

  • If the facility uses a Paragon system.  You need to know the name of the database, the name of the database server machine, and the login credentials for a valid database user.

  • If the facility uses a Horizon Clinicals system.  You need to know the name of the clinical system's database server machine, the login credentials for a valid database user, the type of database (that is, the name of the clinical system's database), and a Staff ID (that is, the ID that the facility's system uses to recognize the McKesson Patient Folder system).

  1. Select Administration > Other Maintenance > Clinical Systems.

  1. Follow instructions for the task you want to perform.

To add a clinical system profile

  1. In the main content pane, click Create New.

  2. In the detail pane at System Provider, select the clinical system type.

  3. Enter required data. See Configure Clinical Systems form for field descriptions.

  4. Click Test to validate your entries. If the test is unsuccessful, you must determine why and correct the cause.

  5. Click Save to save the configuration.

Note: To map the profile to its facility, log on to the MPF Workstation as an administrator user. Select the Administration > Facility function and go to the Clinical System tab. At the Clinical System box, select the facility's clinical system profile and click Apply.

To edit a clinical system profile

  1. In the main content pane, select the profile you want to modify.

  2. In the detail pane, edit information as needed. See Configure Clinical Systems form for field descriptions.

Caution! Changing information for any of the source clinical system attributes (like the database server name or database login credentials) may disrupt communication with that clinical system. Click Cancel to discard unsaved entries if you do not want to apply your edits.

  1. Click Test to ensure that your entries are valid and that the ROI Module can connect to the system. If the test is unsuccessful, you must determine why and correct the cause.

  2. Click Save to apply your entries.

To delete a clinical system profile

Note: While it isn't required, a Best Practice is to dissociate the clinical system profile from its facility before you delete it. Break the association on the Clinical System tab in the MPF Workstation > Administration > Facility function.

  1. In the main content pane, select the profile you want to delete.

  2. Click Delete.

  3. Confirm the action when prompted.
