Rates section

Specifies the base charge and page-level rate tiers (per-page charges) for the billing tier. You cannot delete the base charge or page-level rate tiers that are delivered with the system, but you can set the amounts to $0.00.

Rate information consists of the following items:

  • Base Charge: $ <amount>
    A flat rate to which any per-page charges are added.

  • Page <number> through page <number> = $ <amount> per page
    The page range and per-page charge for the page range you specify. For example, if you enter that Page 1 through page 10 = $ 0.50 per page, the charge for pages 1 through 10 would be $5.00. You can add as many page-level rate tiers as you need. The starting page number for each new page-level rate tier must be the ending page number of the previous tier plus one.

  • Amount for every page thereafter = $ <amount> per page
    The per-page charge for all pages after the page-level rate tier or tiers.

  • Add a new rate tier
    Clicking this button displays the fields needed to add a new page-level rate tier. Any page ranges you add must follow the previous page range sequentially.  You can delete any page-level rate tiers that you add by clicking the delete page tier icon next to the item.
