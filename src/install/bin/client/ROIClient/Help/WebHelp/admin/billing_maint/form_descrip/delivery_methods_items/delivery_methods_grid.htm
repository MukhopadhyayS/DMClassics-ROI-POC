Delivery Methods grid

Displays the delivery methods and associated information currently defined in the system. The grid title is preceded by the number of existing delivery methods.

The Delivery Methods grid contains the following columns:

  • Default
    If checked, indicates the default delivery method for shipping ROI requests.

  • Name
    Names of the delivery methods currently defined in the system.

  • Description
    Descriptions of the delivery methods (optional).

  • URL
    Website URLs for the delivery methods (optional).
