Configure location form

The Configure Location function allows you to specify a default country to associate with your MPF enterprise installation. This provides an additional reference point in the ROI Module application for several areas including billing, and patient and requestor maintenance. You must be an ROI Administrator to configure this option.

Note: The system audits the country code configuration event.

To open the form

Select Administration > Other Maintenance > Configure Location.
