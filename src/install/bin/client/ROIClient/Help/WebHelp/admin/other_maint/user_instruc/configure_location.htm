Configure location

To add, modify, or delete the default country location, do the following:

  1. Access the Configure Location form.

  2. Follow the instructions below for the task you want to perform.

To add a new default country location

  1. Click the Default Country drop-down list and select a country.

  2. Click Save.

To modify an existing default country location

  1. Click the Default Country drop-down list and select a new country.

  2. Click Save.

To delete a default country location

You cannot delete a default country location.  You can only modify it by making a new selection.
