Requestor Type

Specifies the name for a requestor type. Requestor type names must be unique. They are not case sensitive.

You can change the names of user-defined requestor types at any time. These requestor types are indicated by the user icon  in the Requestor Types grid.

You cannot change the names of system-defined requestor types. These requestor types are indicated by the system icon in the Requestor Types grid.
