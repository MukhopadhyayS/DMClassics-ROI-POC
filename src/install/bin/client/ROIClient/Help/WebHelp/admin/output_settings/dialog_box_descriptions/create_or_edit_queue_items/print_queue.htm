Print Queue

The attributes that are listed for the Print option vary depending on the printer. Typically, they include options such as the following, but more options, fewer options, or the same options with different names may exist. See the documentation for your printer for additional information.

Note: You may need to place the cursor on the right edge of the dialog box and pull it to the right to display the vertical scroll bar.

Item descriptions

Queue Type
Specifies the type of queue.

Queue Name
Provides a name for the output queue. This name will be displayed to the user when the user selects to output documents or letters.

Queue Description
Provides a description for the output queue.

Printers
Specifies the default printer to use when the print option is selected. All printers belonging to the selected queue will be listed.

Page Separator
Specifies if a page is inserted between each printed page.

Page Orientation
Specifies whether to print in portrait or landscape orientation (when available).

Collate
Specifies whether to collate the printed pages.

Number of copies
Specifies the default number of copies to print.

Color
Specifies whether to print the output in color (when available) or gray scale.

Duplex
Specifies whether to print on both sides of a page (when available).

Scaling
Specifies the zoom percentage of the page.

Pages Per Sheet
Specifies the number of pages to print on a single sheet of paper.

Location
Indicates the physical location of the printer.

OK
Saves the queue with the options you entered.

Cancel
Cancels adding a new queue or any unsaved changes you made for an existing queue.
