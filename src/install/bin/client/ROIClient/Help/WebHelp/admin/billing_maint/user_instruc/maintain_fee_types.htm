Maintain fee types

To add, modify, or delete fee types for processing ROI requests, do the following:

  1. Access the Fee Types form.

  2. Follow the instructions below for the task you want to perform.

To add a new fee type

  1. In the main content pane, click Create New.

  2. In the detail pane, enter a name and, optionally, an amount, a description for the new fee type, and if sales tax is applicable.

  3. Click Save.

To modify an existing fee type

  1. In the main content pane, select the fee type you want to modify.

  2. In the detail pane, change the information as needed.

  3. Click Save.

To delete a fee type

Note: You cannot delete fee types that are associated with billing templates.

  1. In the main content pane, select the fee type you want to delete.

  2. Click Delete.

  3. In the confirmation message, click Delete.
