About Reasons Maintenance

The Reasons Maintenance function provides administrators the ability to define reasons that users can select when performing ROI activities.

Note: To ensure that users do not get unexpected results, use only upper-case letters (A through Z), lower-case letters (a through z), and numbers (0 through 9) when entering reasons (Display Text) on the Reasons Maintenance forms.

This function is accessed by selecting Reasons Maintenance on the left pane of the Administration tab.

Adjustment reasons

Adjustment reasons provide the reasons for adjusting the balance due for an ROI request. They can include reasons such as Order Change or Billing Mistake.

Request reasons

Request reasons provide the reasons for ROI requests. They can include reasons such as Attorney Request or Patient Request.

Request status reasons

Request status reasons provide the reasons for changing the status of ROI requests such as a Denied or Cancelled request.
