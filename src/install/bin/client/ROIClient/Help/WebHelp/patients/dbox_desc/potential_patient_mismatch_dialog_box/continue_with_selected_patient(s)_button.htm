Continue with Selected Patient(s) button

Creates a new request that includes records for all the patients you select in the Patients grid and associates the request with the currently selected patient requestor.

Enabled only when at least one patient is selected.

Behavior depends on the trigger that launched the dialog box.

If

And

Then

Launched by the Patient Requesting button on any of the Patients tab

(not applicable)

The button launches appropriate dialogs to continue (depends on whether matching requestors are found)

Launched from the Add Patient dialog box

An EPN patient is selected

The button displays the request and adds the selected patient to the request

Launched from the Add Patient dialog box

A non-EPN patient is selected AND

The Add Patient and Close buttons were previously selected

The button displays the request and adds the selected patient(s) to the request

Launched from the Add Patient dialog box

A non-EPN patient is selected AND

The Add Patient and Continue buttons were previously selected

The button adds the selected patients to the request and returns to the Add Patient dialog box
