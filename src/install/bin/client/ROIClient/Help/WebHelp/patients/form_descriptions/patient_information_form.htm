Patient Information form

The Patient Information form enables you to:

  • Create a new non-MPF patient. In this case, all fields default to either blank or a default value.

  • View demographic information for existing MPF and non-MPF patients. In this case, the system displays existing information.

  • Add, change, or delete information for non-MPF patients if you have the required permissions.

  • Create a request.

To open the form

  1. Select Patients > Find Patient.

  2. Click New Patient.

OR

Search for and select a patient. Then click View/Edit Patient.

Note: You can access this form at any time while viewing and/or editing patient information.
