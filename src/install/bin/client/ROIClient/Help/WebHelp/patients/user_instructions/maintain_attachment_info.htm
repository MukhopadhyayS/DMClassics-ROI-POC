Maintain attachment information

This task enables you to change information for an existing attachment. You can change information for only one attachment at a time for both MPF and non-MPF patients.

When you change information for an attachment, the system automatically applies the updates to any request that includes the attachment. Once an attachment has been released on any request, you can edit only the comments associated with the attachment. This ensures the integrity of the Accounting of Disclosure reports.

Note: The patient's document tree lists all encounters, but filters documents based on your record view permissions and on the document types associated with each record view.

  1. Find a patient.

  2. Go to the Patient Records form and double-click the attachment in the document tree.  

  1. On the Edit Attachment dialog box, you can modify the following fields only if the attachment has not been released: Title, Date of Service, Page Count, Document Source, and Enc/Acct. You can always edit comments.

  2. Click Save.
