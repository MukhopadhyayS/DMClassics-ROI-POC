Encounters grid

Displays a list of encounters for the current patient and allows you to select/deselect encounters.

Initially sorted by Admit Date, most recent to oldest. Can be sorted on all columns.

Contains the following information:

  • Checkbox
    Checked state indicates that items relating to the encounter in this row will appear in the results (documents in the record tree or requests in the request history) when this filter is on.
    Initial sort is checked followed by unchecked.

  • Encounter
    Lists encounter numbers.
    Initial sort order is alphanumeric ascending.
    Self-pay indicates the primary payor is the patient.

  • Patient Status
    Patient status stored in MPF Workstation.
    Initial sort is alphanumeric ascending.

  • Facility
    Identifies the facility where the encounter occurred.
    Initial sort is alphanumeric ascending.

  • Patient Type
    Identifies the type of encounter (for example, Inpatient, Outpatient, or Emergency Room).
    Initial sort order is alphanumeric ascending.

  • Admit Date
    Displays the date and time the encounter occurred; format is mm/dd/yyyy.
    Initial sort order is most recent date to oldest.
