Preview of Letter - Other dialog box

The Preview of Letter - Other dialog box displays a preview of a letter that you are generating of type "Other," and allows you to select an output method.  

To open the form

  1. Search for, select, and open a request.

  2. Select Requests > Letters.

  3. On the Letters form, select Other as the letter type, select a template, enter any notes, and click Create Letter.  The system displays the applicable Preview dialog box.
