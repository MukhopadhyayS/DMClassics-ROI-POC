Document Charges group box

Displays charges for documents selected for release.

Copies

The number of copies of the released documents to bill for.

Allows you to enter the number of copies.

Defaults to 1. Valid range is 1 to 99.

Can be modified.

Pages

The number of pages for the corresponding billing tier.

System-generated and read-only.

Billing Tier

The name of the billing tier.

For an MPF document, always displays 'Electronic' in the first row.

For a non-MPF document, displays the name of the tier associated with the requestor type.

Amount

The amount of the charges for this line item.

Total

The total number of pages being released and the total dollar charge for the documents.

Automatically calculated based on copies, pages, and the base charge for each billing tier.

Billing Tier Total (pages and billing tier charges)

Displays the total number of pages to bill for and the total billing tier charges.
