Add attachments to a request

This task allows you to select one or more attachments and add them to an ROI request. The procedure is very similar to adding documents to a request.

Instructions assume that attachments already exist in the patient's record.

Adding attachments to a request is not the same as adding attachments to the patient's record. If the ATTACHMENTS node is missing from the document tree, or if the attachment you want isn't listed in the ATTACHMENTS node, for instructions see:  

  • Download attachments (to download files from a remote system)

  • Add attachments to a patient record (to acquire files from a local file system)

  1. With a request open, go to the Requests > Patient Information form.

  2. Conditional: If this request is for a single patient, skip to the next step. If this request includes multiple patients, select a patient name in the "x Patient(s) on this Request" box.

  1. At the document tree, expand the ATTACHMENTS node.

  2. Select the attachment(s) and/or individual attachment file(s) that you want to release.

  1. Click the > button to add your selections to the Document(s) Selected for Release box.

  2. Repeat as necessary for other patients on the request, if any exist.

  3. Click Save. The request now contains attachments to release.

If you are ready to release and documents and invoice, click Save & Bill and continue with instructions for releasing and invoicing.
