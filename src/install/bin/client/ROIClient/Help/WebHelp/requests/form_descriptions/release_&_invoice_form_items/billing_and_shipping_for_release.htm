Billing and shipping for release

This section displays information and controls for document charges (that is, billing tiers for documents selected for release) and Fee Charges.

Note: If the request has a status of Cancelled or Denied, all information is read-only.

Unbillable Request

If the charges for the request billable.  Non-billable charges display as a discount under the Totals & Balances Due group box.

Billing Location

The location associated with this request.

Document Charges group box

Contains the following items.

  • Copies
    The number of copies of the released documents to bill for.
    Allows you to enter the number of copies.
    Defaults to 1. Valid range is 1 to 99.
    Can be modified.

  • Pages
    The number of pages for the corresponding billing tier.
    System-generated and read-only.

  • Billing Tier
    The name of the billing tier.
    For an MPF document, always displays 'Electronic' in the first row.
    For a non-MPF document, displays the name of the tier associated with the requestor type.

  • Amount
    The amount of the charges for this line item.

  • Total
    The total number of pages being released and the total dollar charge for the documents.
    Automatically calculated based on copies, pages, and the base charge for each billing tier.

  • Billing Tier Total (pages and billing tier charges)
    Displays the total number of pages to bill for and the total billing tier charges.

Fee Charges group box

  • Billing Template
    Displays the name of the billing template (also called the billing type) on which to base fee charges.
    If a default billing template has been configured for the current requestor type, defaults to that template.
    Otherwise, defaults to Please Select, lists all billing types assigned to the current requestor type, and allows you to select one.

  • Fee Types
    Displays all the fee types currently configured for the template specified in the Billing Type box, and displays a total of the charges generated by the fees.
    Defaults to the amount configured on the template.
    Allows you to modify the amount to charge.

Custom Fees grid and Create Custom Fee button

Displays the following columns and controls.

  • Custom Fees
    Displays the name of any custom fee created for this request.
    Allows you to modify the name.

  • Amount
    Displays any current charge for the associated custom fee.
    Allows you to modify the amount.
    Format is dollars and cents.

  • Delete icon
    When clicked, displays a confirmation prompt to delete the Custom Fee line item.

  • Custom Fees Total
    Displays the total charges for all custom fees.
    Read-only.

  • Create Custom Fee button
    Allows you to create a new Custom Fee line item.
