Save New Patient Requestor dialog box

The Save New Patient Requestor dialog box enables you to create a new patient requestor when you are creating a request.

To open the form

  1. Search for and select a patient on the Patients tab.

  2. Click Patient Requesting.

If no requestor was found matching the selected patient(s), the system displays the Create a New Request with the Selected Patient Requesting dialog box and allows you to select a patient to use to create the new requestor.

  1. Select a patient and click Select Patient as Requestor.

Note: If a matching requestor was found, the system does not display this dialog box.

Item descriptions

This dialog box uses the same items as the Requestor Information form, except for the Delete Requestor and Create Request buttons, which do not display on this dialog box.

See Requestor Information form for item descriptions.
