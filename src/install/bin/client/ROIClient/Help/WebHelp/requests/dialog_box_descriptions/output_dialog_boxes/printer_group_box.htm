Printer group box

Allows you to select a printer and displays printer attributes. Also allows you set printer properties.

  • Name
    Lists all printers that have been configured in alphanumeric order.
    Defaults to the first printer in the list for first-time print or when the last-used printer has been deleted; otherwise defaults to the last printer you selected.

  • Properties
    Allows you to view and update properties for the selected printer to output the selected job.
    Changes apply to the current job only.
    Options depend on the printer you select and cannot be predicted.

  • Status, Type, Where, and Comment
    Read-only.
    Displays information for the queue you selected, including the queue status, the queue type, the destination where the output will go, and any comment associated with the queue.

  • Page Separator
    Displays only when the Page Separator option is enabled in the Configure Output Settings function; otherwise, does not display.
    Defaults to unchecked.
    When enabled, inserts a separator sheet between released documents.
    When used, the separator sheet contains the user name (lastname, firstname), user department, printer name, and date and time according to system settings.
