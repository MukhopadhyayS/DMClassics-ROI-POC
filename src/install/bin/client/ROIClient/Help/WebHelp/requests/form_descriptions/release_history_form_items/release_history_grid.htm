Release History grid

Lists all release events that match the filter criteria. Also displays the total number of release events and the total number of pages released.

Read-only.

Default sort order is by Date | Time in most recent to oldest order. Can be sorted on all columns. Includes the following information:

  • Date | Time
    The date and time when the document was released.

  • Patient Name
    Patient name associated with released document.
    Used as secondary sort with default. Initial sort order is alphanumeric ascending.

  • Encounter
    Encounter associated with released document.
    Initial sort order is alphanumeric ascending.
    Self-pay indicates the primary payor is the patient.

  • Document - Version - Subtitle
    Name of the document released, including the version of the document and its subtitle.
    Initial sort order is alphanumeric ascending.

  • Pages
    Description of the actual pages released. Example: Pages 1-4, 6, 10-12.
    Initial sort order is numeric descending.

  • Request Password
    Generated by the system and is specific to the request.
    Applies to all documents and attachments that are output as PDF files for this request and displays only when queue passwords are applied to the output.
    Cannot be changed and must be manually communicated to the requestor.

  • Queue Password
    Displays the password specified for the queue by default or by the user. A default value is specified by the ROI Administrator, but can be overridden on the Save As File dialog box.
    Applies to all documents in the queue. Must exist to encrypt the PDF file.
    When specified, always displays.

  • Shipping Address
    Shipping address specified at the time of the release.
    Initial sort order is alphanumeric ascending.

  • Shipping Method
    Method used to ship the released documents to the requestor.
    Initial sort order is alphanumeric ascending.

  • Shipping Tracking Number
    Tracking number issued by the shipping vendor associated with the release.
    Initial sort order is alphanumeric ascending.

  • User Name
    The full name of the user who released the document.
    Initial sort order is alphanumeric ascending.
