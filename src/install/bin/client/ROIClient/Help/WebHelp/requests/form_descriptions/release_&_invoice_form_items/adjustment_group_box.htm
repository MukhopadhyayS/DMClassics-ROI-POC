Adjustment group box

Allows you to create an adjustment to the billing charges for this request.

  • Debit
    Indicates a debit amount.

  • Credit
    Indicates a credit amount.

  • Amount
    Allows you to enter the dollar amount to debit or credit.
    Cannot be zero.

  • Reason
    Indicates the reason for the adjustment. Required.
    Defaults to Please Select.
    Allows you to select a standard reason or to enter a free-form reason.

  • Add button
    Adds the adjustment to the request.
    Enabled only if you have entered an amount and a reason.
