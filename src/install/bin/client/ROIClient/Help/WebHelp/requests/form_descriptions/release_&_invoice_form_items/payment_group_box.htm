Payment group box

Allows you to record a payment for this request.

  • Method
    Allows you to select a method of payment (for example, check, cash, or credit card) for a specific payment. Payment methods are defined by each site. Defaults to "Please Select."

  • Amount
    Allows you to enter the dollar amount of the payment.
    Cannot be zero.

  • Payment Note
    Allows you to enter more information about the payment.
    Displays only when the payment method is configured to support this field.
    Entry is optional.

  • Add button
    Applies the payment to the request.
    Enabled only if you have selected a payment method and entered an amount.
