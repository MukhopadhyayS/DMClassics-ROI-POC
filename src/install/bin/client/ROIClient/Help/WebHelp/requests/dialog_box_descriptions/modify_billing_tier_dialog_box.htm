Modify Billing Tier dialog box

The Modify Billing Tier dialog box allows you to select a new (that is, a different) billing tier for a non-MPF document that is associated with a request. This function is effective with only non-MPF documents.

To open the form

  1. Find a request.

  2. Go to the Patient Information form.

  3. Select a non-MPF document that is listed in the Document(s) Selected for Release box.

  4. Click Modify Billing Tier.
