Cancel button

Behavior depends on the type of preview on display.

If this is a preview is of

Then the button

A cover letter or a letter of type "Other"

Cancels the creation of the letter, closes the preview dialog box, and returns you to the Letters form.

An invoice or a pre-bill letter

Cancels the creation of the invoice, closes the preview dialog box, and returns you to the Release & Invoice form.

A Release letter

Cancels only the creation of the letter(s) and returns you to the Release dialog box, where you may select a new letter template or deselect adding a letter to the release of the documents.
