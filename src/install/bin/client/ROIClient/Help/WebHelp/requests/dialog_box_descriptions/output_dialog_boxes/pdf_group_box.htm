PDF group box

Allows you to specify the intended media type, password, and name of a file to generate. Also allows you set file properties.

Note: Currently, only the PDF file type is supported.

  • For Media
    Displays a list of media types and allows you to select one.
    Options are     CD, DVD, and Other / Unlimited Size.
    Your choice determines the maximum allowable file size in Mbytes, and the system creates multiple files, if needed.
    Defaults to     Other / Unlimited Size.

  • Password
    Allows you to enter a password (up to 64 characters) to encrypt the PDF file.
    If flagged as required, then defaults to the password specified for the queue (if any), and can be overridden but cannot be left blank. If not required, then can be blank or set to a value of your choice.
    Must manually be given to the requestor to use when opening the file.
    Read-only if you are resubmitting a PDF job from the Monitor Jobs function.

  • Name
    Lists all save-as-file queues that have been configured, in alphanumeric order.
    Defaults to the first save-as-file queue in the list for your first-time save-as-file job or when the last-used save-as-file queue has been deleted; otherwise defaults to the last save-as-file queue you selected.

  • Status, Type, Where, and Comment
    Read-only.
    Displays information for the queue you selected, including the queue status, the queue type, the destination where the output will go, and any comment associated with the queue.
