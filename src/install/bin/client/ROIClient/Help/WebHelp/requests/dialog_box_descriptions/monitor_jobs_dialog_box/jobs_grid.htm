Jobs grid

Displays on both the Job View tab and the Queue View tab. Displays jobs filtered by the criteria you select on the tab.

Initially sorted by Type, then by Status in alphanumeric ascending order. Can be sorted by any column.

Allows you to select one or more jobs on which to perform an action. An administrator user can act on all jobs. A non-administrator user can act only on jobs submitted by that user.

Displays the following job information:

  • Status
    The status of the job.  Based upon the type of job, the system displays a warning or error icon.  Hover your mouse over the icon for more information. You can also click the     Job Details button.

Note: For disc jobs, "Warning" messages can indicate there is no CD found in the bin or the drive tray needs to be closed manually.  "Error" messages can indicate the device is shutdown or there is no connection to the device.  For more detailed information on disc errors refer to the Rimage User Guide.

  • Type
    The type of job.

  • Date Submitted
    The date and time the job was submitted.
    Date and time format are controlled by the region setting of your operating system.

  • Job Name
    The name assigned by the system to this job using <originating_app_name>.<jobID> format.

  • Pages
    The number of pages in a job.

  • Queue
    Identifies the queue where the job resides.

  • Job ID
    The ID assigned by the system to the job.

  • User ID
    The user ID of the person who created the job.
