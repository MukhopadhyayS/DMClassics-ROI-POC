Add Mapping button

Validates the MPF user ID you provided, associates it with your Windows account, and displays a confirmation message. At the confirmation dialog box, you can choose to either map another MPF user ID or launch the application.

This is the only opportunity the system allows for you to map other MPF user IDs to the Windows account you entered. If you skip this opportunity and launch the application, only a system administrator can map additional MPF IDs to the Windows account.
