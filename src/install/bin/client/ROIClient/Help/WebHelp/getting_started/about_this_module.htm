About this module

The McKesson Patient Folder Release of Information (ROI) Module provides a tool for managing all requests for copies of medical records, including a billing component.  The ROI Module enables you to select and release patient information to entities who request patient records, and to easily generate letters, billing notices, and invoices for that service. You can manage and account for the release of both MPF and non-MPF records.

Release of Information features enable users to:  

  • Configure and manage administrative functions including user profile permissions, perform maintenance and configure output settings

  • Create and manage patients and view patient information, records and request history

  • Create requestors and view their profile and request and account histories

  • Manage requestor accounts including adding payments and adjustments, generating statements, and issuing refunds

  • Create requests specifying billing and shipping information, output method, and sales tax

  • Generate cover letters, statements, and invoices (both current and past due)

  • Generate reports

  • Find overdue invoices

Auditing

The ROI Module can maintain an audit trail history of releases and other activity. Audit trail events are configured in the MPF Workstation application for each facility.

Record views in ROI

The Patient Records and Patient Information forms allow you to select record views for subsets of documents associated with the patient’s encounters. Record views identify standardized packets of documents for specialized functions such as coding, business office, or clinical. Record views are maintained by the McKesson Patient Folder system administrator. Only one record view can be associated with a request as a default record view. All other record views available to the user are available for selection.

ROI requests on file

The Request History form contains a list of all existing requests for a selected patient. You can select a request to view its details, and can add, change, or delete a request.
