Directory Services Self-Mapping dialog box

The Directory Services Self-Mapping dialog box allows you to associate your Windows account with your McKesson Patient Folder user ID. This dialog box displays when all of the following conditions are true:

  • The Directory Services Authentication feature is enabled at your site AND

  • The user self-mapping feature is enabled AND

  • No MPF user ID is currently associated with the Windows account you entered.

This feature is effective only with ROI Release 15.0 and later.

To open the dialog box

  1. At the Logon dialog box, enter your Windows user name, password, and domain.

  2. Click Logon.
