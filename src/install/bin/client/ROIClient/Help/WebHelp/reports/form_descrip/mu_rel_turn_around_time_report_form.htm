MU Release Turn Around Time Report form

The MU Release Turnaround Time Report form allows you to generate a report that shows request turnaround times for Meaningful Use documents for MPF patients (non-MPF patients are excluded). The report includes only the first release of an MU document. Subsequent releases of the same document are not included. You can select criteria for the report and specify the date range. Security rights are required.

This report shows request turnaround times as measured in days.

When viewed by Totals Per Facility, the report shows the total number of first releases that were completed within the government-mandated turnaround period. It also shows compliance as a percentage of total MU documents that were released.

Default sort order

The report is initially sorted as follows.

  • Turnaround Hours. Highest total turnaround time in descending order

  • Business Days. Highest total business days in descending order within turnaround time

  • Date Received. Latest MU Request receipt date in descending order within Total business days

  • Date Released. Latest date released in descending order within date received

  • Facility. Ascending alphabetic order within date released

  • User Name. Ascending alphabetic order within Facility

  • Patient Name (Last, First). Ascending alphabetic order within Username

Note: All the requests with a blank Turnaround time value and Date Released are listed at the beginning of the report sorted by the latest date received. They are followed by the highest turnaround time requests as defined in the above sort order.

To open the form

  1. Select the Reports tab.

  2. In the Report Name box, select MU Release Turnaround Time Report.  

When the form first displays, the main content pane is blank. To view a report, select the report options in the left pane and click Run Report.
