About generating reports

The ROI Reports function provides the ability to generate reports related to ROI requests.

  • To generate reports, the Crystal Reports product must be installed and running.

  • You must have the appropriate security permissions.

  • Printing longer reports can tie up your system for some time. This is largely dependent on your system resources and the date range you specify, for example, more data takes longer.

For information about request statuses, see About requests.

Security rights

  • You must have the ROI Reports security right to generate ROI reports. See your administrator if you need this right.

  • If you are logged into ROI when the right is granted, you must log out and log back in for the change to take effect.

  • If you do not have VIP permissions, the system displays VIP patient names in reports but omits all other information related to those patients.

  • Reports that can display VIP names include Accounting of Disclosure, Documents Released by MRN, Request Detail, and Requests with Logged Status reports.

Report options and functionality

  • For most reports, you can select report criteria, specify a date range, and select the level of detail.

  • You can print the reports and export (save) them to a file (provided you have the appropriate security rights).

  • The available file formats for saving reports are PDF, XLS, RTF, CSV, and HTML.

  • You cannot fax reports directly from the ROI interface, but you can save the generated reports to a file and attach the file to an email.

Report results viewer

This view contains the report results.

The View By drop-down list changes how the results display.

The button bar contains the following items:

  • Copy. Copies the displayed report information into the Windows clipboard.  The information can then be pasted into another application such as Microsoft Excel or Word.  Click and drag the box to highlight the text to copy. Click the Copy button, open the desired application, and paste the information.

  • Navigation buttons.  Moves forward or backward one page.
