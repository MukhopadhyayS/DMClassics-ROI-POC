Apply Payment to

  • Invoice #
    List of all open invoices, sorted chronologically with the oldest first.

  • Request ID
    Request ID associated with the invoice.

  • Date
    Date the invoice was created.

  • Charges
    Original charges when invoice was generated.

  • Pay/Adj
    Total payments/adjustments for the invoice.

  • Applied
    Amount of the payment previously applied to the invoice. Click the red X (if applicable) to remove the applied payment.

  • Balance
    Current invoice balance (charges - payments + adjustments).

  • Apply
    Manual override of amount of the total payment to apply to individual invoice. Also displays the amount to apply
