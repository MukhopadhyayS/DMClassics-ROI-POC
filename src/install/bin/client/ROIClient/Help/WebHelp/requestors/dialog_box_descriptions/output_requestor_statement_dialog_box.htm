Output Requestor Statement dialog box

The Account Requestor Statement function allows you to issue a requestor statement.  You select a pre-configured template, date range and enter any notes.

To open the form

  1. Find a requestor.

  2. In the left pane, select Account Management.

  3. Click Statements.
