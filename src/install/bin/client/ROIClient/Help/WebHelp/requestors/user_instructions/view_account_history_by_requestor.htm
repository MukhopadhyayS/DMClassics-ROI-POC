View account history by requestor

This function enables authorized users to view Pre-Bills, Invoices, and Statements submitted for a specific requestor and generate a document that can be printed, faxed, emailed, and saved.  

  1. Find a requestor.

  2. Go to the Account History form.

  1. Follow the instructions below for the task you want to perform.

To view a request

The Account History form lists each request for the requestor in summarized form. To view more information for a request:

  1. Select the request from the Request grid.

Note: If a request contains records from facilities that you are not allowed to access, the detail pane lists the request in red text and masks data in the Patients column. If you select the request, the system disables the View/Edit Request button.

  1. Click View. The system displays the appropriate Preview dialog box. You can print, fax, email or save the file.

To sort requests

Requests initially display in chronological order, beginning with the most recent.

To sort requests in a different order, click on any column heading in the Requests grid to sort the list by that column.
