Account Register grid

Contains the following information:

  • Date
    The transaction date. For invoices, this is the creation date.  For unapplied payments/adjustments and refunds, this is the date the transaction was initiated.

  • Type
    The type of event.  Types include Open/Closed Invoice, Unapplied Payment, Unapplied Adjustment, and Refund.

  • Description
    The description of the event.  Invoices show the invoice number.  Payments, adjustments and refunds show a note.

  • Request ID
    The system-assigned ID of the request.

  • Charges
    The amount of the invoice before adjustments and payments are applied.

  • Adjustments
    The value/amount of the adjustment. Negative values are shown in ( ).

  • Payment
    The value/amount of the payment. Negative values are shown in ( ). Refunds show as a positive value.

  • Billable
    Specifies if the request is billable. A checkmark indicates applicable invoices.

  • Balance
    The total charges minus payments and adjustments.
