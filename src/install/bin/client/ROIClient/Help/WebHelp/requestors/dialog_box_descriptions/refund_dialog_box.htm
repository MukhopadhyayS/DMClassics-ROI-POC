Refund dialog box

The Refund dialog box enables you to refund a credit balance on a requestor's account.  

Refunds cannot be issued to accounts that have open invoices.

To open the form

  1. Find a requestor.

  2. From the left pane, select Account Management.

  3. In the Account Management form, click Refund.
