Permissions

Any McKesson OneContent Patient Folder user can log in to the Index Correction Utility and view or process documents with index errors.

Access to inbound exceptions is controlled by permissions in your OneContent Patient Folder user profile. The system disables the inbound exception option for users who do not have appropriate permission.

Note: Administrator users can always access inbound exceptions.
