Using ICU within a Multi-facility Installation

The Index Correction Utility can be used:

• Within a single facility, where it provides error document information for only that facility.
• Across a multi-facility enterprise, where it can be either dedicated to a specific facility or configured to access all facilities within the enterprise. This is useful when, for example, your organization operates out of more than one physical location. A cross-facility ICU provides error document information for all facilities within the enterprise.

System Administrators can configure single or multi-facility at any time.
