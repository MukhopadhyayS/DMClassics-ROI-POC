Display Workstation module

The Display Workstation, also called the WinDSS viewer, displays documents in a viewer window. The viewer is launched from these applications:

  • From the Index Correction Utility application
    When you launch the viewer from ICU, the viewer allows you to view pages from a single record, scroll document pages, or manipulate windows. You can also choose to simultaneously display an inbound exception document and an existing document in two viewer windows for comparison.
  • From the Release of Information (ROI) Module application
    The ROI Module is an add-on component that is installed separately from the basic McKesson OneContent Patient Folder system. Your site may or may not have the module installed. When you launch the viewer from the ROI Module, the viewer allows you to view pages from a single record, scroll document pages, or manipulate windows.

File formats

The viewer supports displaying color, bitonal (black and white), or gray scale images in a variety of image formats including IMNET, version 3, 4 and LZW compression TIFF, BMP, GIF, ASCII, PDF, and JPEG.

The viewer can display text-based documents (those that enter the system through an electronic feed) as text or as images. It displays scanned documents or those acquired as images (such as through Batch Compiler) only as images.

Screen display setting

To enable the system to correctly display icons, highlighted text, document images, and so on, the color display for your monitor should be set to at least Medium (16-bit). (This setting may also be called High Color or 65536 Colors.)

Log files

By default, the WinDSS application logs fatal and error messages to the Windows Event Log and generates a series of up to 21 log files in the working directory for all error messages. Each log file is limited to a maximum file size of 1 MB. See the Messages and Codes Reference guide for more information.

Basic viewer functions

These viewer functions are available regardless of how the viewer has been launched. You can:

  • View up to two pages from one document
  • Manipulate document pages by zooming, rotating, or scrolling
  • Scroll documents or document pages
  • Manipulate viewer windows (save settings, set default position, or keep in front)
