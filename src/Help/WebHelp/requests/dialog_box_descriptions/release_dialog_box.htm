Release dialog box

The Release dialog box enables you to release documents associated with a request.

To open the form

  1. On the Requests tab, search for, select, and open a request.
  2. Select the Release & Invoice function. The system displays the Release dialog box.

Item descriptions

Update Status option

Allows the system to automatically set the request status to "Complete" when the documents are successfully released.
Options are:

  • Yes (selected by default; sets request status to "Complete" when the documents are successfully released)
  • No (leaves current status as-is when the documents are successfully released)

Cover Letter group box

Enables you to generate a cover letter and notes for the request to output with the documents selected for release.

Items are:

  • Add Cover Letter to Release
    Triggers the system to create a cover letter to output with the documents selected for release.
  • Cover Letter Template
    Lists all cover letter templates configured in the system and allows you to select one.
    Defaults to the template configured as the default selection.
    If no default is configured, then defaults to Please Select.
  • Select/Create Notes box and Create Freeform Note button
    Displays only when the selected template is configured to support notes.
    Displays all preconfigured notes associated with the template.
    Allows you to select one or more notes to print with the cover letter.
    Allows you to create additional freeform notes.

Statements group box

Enables you to generate a statement and notes for the request to output with the documents selected for release.

Items are:

  • Output Statement
    Triggers the system to create a statement to output.
    When selected, enables the Statement Template and Date Range fields.
  • Statement Template
    Lists all statement templates configured in the system and allows you to select one.
    Defaults to the template configured as the default selection.
    If no default is configured, then defaults to Please Select.
  • Date Range
    Specifies the date range to include in the statement.
    Preconfigured list of choices includes: 30, 60, 90, or 120 days, Year to Date, and All.
  • Select/Create Notes to add to the Invoice box and Create Freeform Note button
    Displays only when the selected template is configured to support notes.
    Displays all preconfigured notes associated with the template.
    Allows you to select one or more notes to print with the statement.
    Allows you to create additional freeform notes.

Current Invoice group box

Enables you to generate an invoice and notes for the request to output with the documents selected for release.

Items are:

  • Add New Invoice to Release
    Triggers the system to create a cover letter to output with the documents selected for release.
  • Invoice Template
    Lists all cover invoice templates configured in the system and allows you to select one.
    Defaults to the template configured as the default selection. If no default is configured, then defaults to Please Select. Displays None if the user has enabled the Invoice Optional checkbox on the Requestor Types screen.
  • Invoice Due [number] Days
    Displays the default number of days by which the invoice must be paid.  
    Allows you to override the default.
  • Select/Create Notes box and Create Freeform Note button
    Displays only when the selected template is configured to support notes.
    Displays all preconfigured notes associated with the template.
    Allows you to select one or more notes to print with the invoice.
    Allows you to create additional freeform notes.

Past Invoice group box

Enables you to generate a list of past due invoices (if applicable) for the request to output with the documents selected for release.  

Items are:

  • Add Past Invoice to Release
    Populates the grid with all past due invoices.  
  • Invoice grid
    Lists all past due invoices.  Select each invoice to include with the release documents.

Continue

Continues with the release of documents.

Launches a preview of the cover letter, statement, current and past due invoices.

Cancel

Cancels the creation of the release of documents and returns you to the Release & Invoice form.
