Select Requestor dialog box

The Select Requestor dialog box enables you to search for and select a requestor to associate with a request.

To open the form

  1. Select the Requests tab.
  2. Click New Request.
  3. On the Request Information form, click Select Requestor. The dialog box displays two tabs: Find Requestor and Requestor Information.

Item descriptions

Find Requestor tab

  • The Find Requestor tab displays by default.
  • This tab uses the same items as the Find Requestor form, with the exception of the Create Request button.
  • See Find Requestor form for item descriptions.

Requestor Information tab

  • The Requestor Information tab can be accessed only if you click either New Requestor or View/Edit Requestor on the Find Requestor tab.
  • This tab uses the same items as the Requestor Information form, with the exception of the Create Request button.
  • See Requestor Information form for item descriptions.

Additional items

This dialog box contains the following additional items.

Select Requestor for Request

Saves any unsaved data, records the selected requestor as the requestor for the request, and closes the dialog box.

If you selected a patient requestor, launches the Select Patients dialog box. Otherwise, displays the Requestor Information form.

Enabled only after you select an existing requestor or when you are creating a new requestor.

Cancel

Discards unsaved changes, cancels the selection of a requestor, and returns you to the Request Information form.
