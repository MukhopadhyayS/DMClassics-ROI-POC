Output Disc Dialog Box

The Output Disc dialog box enables you to output documents to a disc such as a CD or DVD.

Note: The DVD size cannot exceed 1GB.

To open the form

The system launches the dialog box when you do any of the following:

  • Click Continue on the Release & Invoice form and make any of the following selections:
  • Release
  • Re-Release
  • Invoice
  • Pre-Bill
  • Click Resend from the Patient Information form.

Item descriptions

Name

Name of the Disc output queue. Lists all Disc queues that have been configured.

Disc type

Type of disc, either CD or DVD.

Password

Default password for the selected queue.  You can edit the password.

Template

Name of selected template.  Open the drop-down list to display a list of all available templates.

Select/Create Notes to Add to label

Select a note from the list or click the Create Freeform note to create your own note.

Add to Release Document group box

Allows you to print a header, footer, and/or watermark on all released documents (except letters).

Displays only when the output dialog box has been launched from the Release dialog box and the header, footer, and/or watermark options are enabled in the Configure Output Settings function (any disabled option does not display). Displayed options are enabled only if a corresponding value has been configured.

  • Headers and footers can display an absolute page number, a relative page number, the patient name, and the encounter number, MRN, or EPN.
  • Watermarks can contain either the default value or any text string you specify up to 20 characters.

Print Release Letters group box

Displays only when the Output dialog box has been launched from the Release/Re-Release dialog box.

When releasing documents to disc, the release letters (invoices, statements, or cover letter) will be printed.

Name

Lists all printers that have been configured in alphanumeric order.

Defaults to the first printer in the list for first-time print or when the last-used printer has been deleted; otherwise defaults to the last printer you selected.

Status, Type, Where and Comment

Read-only.

Displays information for the queue you selected, including the queue status, the queue type, the destination where the output will go, and any comment associated with the queue.

Note: If the release contains only non-OneContent Patient Folder documents, a message displays at the top of the dialog box indicating that no documents will be output to disc.

OK

Submits the job to the selected queue and closes the dialog box.

If an error occurs, the system alerts you.

Cancel

Discards the job, closes the dialog box, and returns you to the calling form or dialog

box.
