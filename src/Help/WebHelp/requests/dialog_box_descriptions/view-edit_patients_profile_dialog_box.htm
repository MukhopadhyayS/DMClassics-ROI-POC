View/Edit Patient's Profile dialog box

The View/Edit Patient's Profile dialog box enables you to view demographic information for patients. For non-OneContent patients, the dialog box allows you to edit information. You can access this dialog box when you are creating a request or when you are viewing an existing request.

To open the form

  1. Select the Requests tab.
  2. Create a request or open an existing request.
  3. At the Patient Information form, select a patient and click View / Edit Patient Profile.

Item descriptions

This dialog box uses the same items as the Patient Information form. Exceptions are the Patient Requesting and Create Request buttons, which do not display in this dialog box.

See Patient Information form for item descriptions.

Save

Saves changes to the patient's information.

Enabled only for non-OneContent patients and only when the Name field contains a value and unsaved data exists.

  • Validates all fields and displays alerts or messages if problems exist
  • If patient name is non-unique, prompts whether to save and continue

Revert

Reverts all information on the Patient Information form to the last saved state.

Displays only for an existing patient.

Enabled only after you edit information for a non-OneContent patient. Remains disabled for an OneContent patient.

Close

Closes the dialog box and returns you to the Patient Information form.

If unsaved changes exist, prompts whether to discard the changes.
