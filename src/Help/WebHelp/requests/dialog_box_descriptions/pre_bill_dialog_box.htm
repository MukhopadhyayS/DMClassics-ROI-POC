Pre-Bill dialog box

The Pre-Bill dialog box allows you to select a template to use for generating a pre-bill letter. If the template supports notes, you can also select and/or create notes to include.

To open the form

On the Release & Invoice form, click Pre-Bill.

Item descriptions

Template

Lists all pre-bill templates in the system and allows you to select one.

Defaults to the template configured as the default, if one is specified. If not, then defaults to Please Select.

Notes/Create Freeform Note

Displays only if the template supports notes.

Lists all predefined notes in the ROI system and allows you to select one or more to include on the pre-bill letter.

Also allows you to add, change, select, and delete one or more notes of your own.

Continue

Launches the Preview of Pre-Bill dialog box.

Cancel

Cancels the pre-bill letter, closes the dialog box, and returns you to the Release & Invoice form.
