Create New Request with Patient Requestor dialog box

The Create New Request with Patient Requestor dialog box allows you to select patient(s) to include on a request when you are creating a request for a patient requestor.

To open the form

This dialog box displays when you display a patient requestor on the Requestor Information form and click the Create Request button.

Item descriptions

Patients grid

Displays an entry for each patient name that could be matched to the patient requestor and allows you to select one or more patients to include on the request.

Initially sorted by Patient Name in alphanumeric ascending order. Can be sorted by any column. By default, the first row is auto-selected.

Information includes:

  • Patient Name
    The name of the matching patient.
    Sorts on the first characters displayed, regardless of which name (first or last) occurs first.
  • Date of Birth
    The date of birth of the patient.
    Initial sort is date ascending.
  • SSN
    The patient's Social Security Number.
    Initial sort is numeric ascending.
  • Facility
    Displays only for non-EPN systems.
    The facility where patient care occurred.
    Initial sort is alphanumeric ascending.
  • MRN
    Displays only for non-EPN systems.
    The patient's medical record number.
    Initial sort is alphanumeric ascending.
  • EPN
    Displays only for EPN systems.
    The patient's enterprise person number.
    Initial sort is alphanumeric ascending.

Continue with Selected Patient(s)

Creates a new request for the patient requestor and with the selected patients added to the request.

Enabled only when at least one patient is selected.

Disabled when EPN is configured and more than one EPN patient is selected.

Launches the Request Information form.

Continue with No Patient

Creates a new request that includes only the selected patient requestor.

Always enabled.

Launches the Request Information form.

Cancel

Stops the process of creating a new request, closes the dialog box, and returns you to the calling form.
