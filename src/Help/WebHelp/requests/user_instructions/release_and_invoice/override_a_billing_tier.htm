Override a billing tier

This function enables an authorized user to assign a specific billing tier to documents selected for release on a request. This function is effective for only non-OneContent documents. A default  billing tier is assigned to documents on a request. When you change the assigned billing tier, the system automatically recalculates any Balance Due total displayed on the Release & Invoice form.

You must drill down to the page level to change a billing tier.

  1. Find a request.
  2. Go to the Requests > Patient Information form.  
  1. In the Document(s) Selected for Release box, expand the document tree and select one or more non-OneContent document(s) you want.  
  1. Click Modify Billing Tier. (You cannot modify the billing tier for a previously released document.)
  2. In the Modify Billing Tier dialog box, select the tier you want to assign to the document. You can assign either an OneContent tier or a non-OneContent tier to the documents. Then click Save.
  3. In the Patient Information form, click Save to apply the change.
