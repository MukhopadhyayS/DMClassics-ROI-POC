Pre-bill a request

This task enables an authorized user to create a pre-bill letter to give an estimate of the total cost of a request.

You can create a pre-bill on-demand any time during the life of a request, as long as no documents have been released for the request and the request status is not Denied or Canceled. You can create a pre-bill even if no documents have been specified for release, although certain subtotals will be zero.

While not required, your site may have configured the ROI system to automatically include a cover letter with the pre-bill invoice.

Note: To retrieve or recreate a previously created invoice for viewing or output, see Resend documents.

Instructions assume you are accessing an existing request.

  1. Find a request.
  2. Go to the Requests > Release & Invoice form.
  3. Click CreatePre-bill.

Note: Create Pre-bill is only available for billable requests.

  1. In the Pre-Bill dialog box, select the template you want to use.
  1. Optional: The Notes box displays only if your site has configured the Notes feature. Select one or more standard notes (if any exist) to include on the invoice, or add a free-form note.

Note: Note: Use only letters and/or numbers in free-form notes or descriptions. Do not include special characters. Special characters (such as apostrophes, dashes, ampersands, and so on) may cause the system to error when generating letters. Click Continue.

  1. At the Preview of Pre-Bill dialog box, review the information, and then select the output method you want to use. The system displays only the output options for which you have permission.
  2. Provide information as needed at the print, save as file, fax, or email dialog box, and complete the output operation.
- If you choose to output to a file, the system automatically assigns a file name based on request ID. If you choose to fax the output, the system automatically populates the requestor's fax information and allows you to change it.
- To help you monitor your output jobs, the system assigns an output job name using the request ID appended with "- Pre-bill Sent".

The pre-bill information will display in the Payments dialog box and Account Management screen.
