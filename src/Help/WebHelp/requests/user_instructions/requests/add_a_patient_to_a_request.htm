Add a patient to a request

This function enables you to add one or more patients to a request. You can add as many patients to the request as needed.

1. With the request open, go to the Patient Information form. (For an existing request, find and open it.)  
2. Click Add Patient (for a new request) or Add Another Patient (for an existing request). The system displays the Add Patient(s) dialog box.  
3. In the Add Patient(s) dialog box:
- Perform a search. Results show all patient records that match the search criteria and include all facilities that provided care for the same patient.
- Select one or more patients.

CAUTION: For patient requestors, if you select multiple patients to include on the request, the system displays a list of potential mismatches and allows you to evaluate each patient record to determine whether the record is appropriate for this request. It may be appropriate, for example, when parents request records for their children.

4. Optional: To add the selected patients, but perform another search and select more patients, click Add Patient & Continue.
5. If you selected all the patients you wanted and are ready to return to the Patient Information form, click Add Patient & Close.
6. Click Save. The request now contains the patients for whom you want to release documents. You can continue to build the request now or return to it later.
7. To continue building the request, see Add documents to a request
