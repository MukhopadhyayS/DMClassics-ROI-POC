Create a request from the Requestors tab

This task assumes you want to create a request for a specific requestor (either patient or non-patient).

  1. Perform a requestor search.
  2. In the detail pane on the Find Requestor form, select the requestor that you want to associate with the new request. Then click Create Request.
- For a non-patient requestor, the system displays the Request Information form. Continue with the next step.
- For a patient requestor, the system displays the Create New Request with Patient Requestor dialog box. Do one of the following:
- Select the patient you want to assign as the requestor and click Continue with Selected Patient(s). The system will create the request with records for the patient you select.
- Do not select any patient and click Continue with No Patient. The system uses the requestor that was selected when you clicked Create Request and will create the request with no patient records.

Note: Click Cancel to close the dialog box and return to the Find Requestor form without creating the request.  

  1. At the Request Information form, select a Request Reason and Receipt Date, then click Save.

The request has been created. You can continue building the request now or later.

To continue building the request, see:

- Add a patient to a request (if you want to add more patients to this request)
- Add documents to a request (if you want to add documents to this request)
