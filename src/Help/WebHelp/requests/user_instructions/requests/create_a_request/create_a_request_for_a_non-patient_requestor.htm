Create a request for a non-patient requestor

This task assumes you want to create a request for a non-patient requestor. Non-patient requestors (such as insurance companies, attorneys, or other entities) often request records for more than one patient at a time. You can select one or more patients to include on the request.

This task describes how to create the request from the Patients tab.  

  1. Perform a patient search.
  2. On the Find Patient form, select one or more patients whose records are being requested.

Note: If you selected a single patient and want to verify that you have selected the right patient, click View/Edit Patient to see details on the Patient Information form.  

  1. Click Create Request. The system displays the Request Information form.
  2. At the Request Information form, select a Request Reason and Receipt Date.
  3. Click Select Requestor and specify a requestor at the Select Requestor dialog box.
  4. When you return to the Request Information form, click Save & Continue.

The request has been created. You can continue building the request now or later.

To continue building the request, see:

- Add a patient to a request (if you want to add more patients to this request)
- Add documents to a request (if you want to add documents to this request)
