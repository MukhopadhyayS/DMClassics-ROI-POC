Create a request with patient requesting

This task assumes that you want to create a request for a patient who is requesting his or her own records.

1. Perform a patient search.
2. On the Find Patient form, select the patient who is requesting records and click Patient Requesting. The system searches for a matching requestor record.

Note: If the patient has never requested records before, the displayed dialog box omits the following items: Address, Phone, Fax, and New Requestor button. Also, it labels the Select Requestor button as Select Patient as Requestor.

3. At the Create a New Request with the Selected Patient Requesting dialog box:
- If the system finds one or more requestors that potentially match the patient, the system displays all found names in the grid. Select the matching entry from the list and click Select Requestor. The system proceeds to the Requests > Request Information form.
- If no match is found, the system displays and highlights the patient's name in the grid. Click Select Patient as Requestor. The system then displays the Save New Patient Requestor dialog box.
4. Select Pre-payment Required and/or Certification Letter Required (when appropriate), review the patient's information and correct it, if needed, and then click Save New Requestor. The system adds the requestor record and proceeds to the Requests > Request Information form.
5. At the Request Information form, select a Request Reason and a Receipt Date, then click Save.

The request has been created. You can continue building the request now or later.

To continue building the request, see Add documents to a request.
