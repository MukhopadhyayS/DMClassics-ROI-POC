Re-release documents

This function enables an authorized user to re-release documents that have already been released at least once and to generate billing charges for the service.

Note: To re-send released documents without generating charges, see Resend documents.

The system calculates charges for re-released documents depending on the billing tiers set up by your enterprise, adds the new charges to any existing balance due, and creates an invoice. All invoices are displayed in the invoice history.

The system audits the event only when the "R" (ROI Released) audit code is enabled in the OneContent application.

  1. Find a request.
  2. Go to the Requests > Patient Information form.
  3. At the Document(s) Selected for Release box, click the Released filter option to list only those documents that have been released.
  1. If no pre-bill is required, click Re-Release. (Otherwise, see Pre-bill a request for further instructions.)
  2. At the Release dialog box, specify a cover letter template and an invoice template, and click Continue. When the system displays the Preview of Letter(s) dialog box, click Continue. After output is successfully generated, the system returns you to the Release & Invoice form.
  3. The release process is complete within the ROI system. Complete any manual steps (such as handing off to the shipper). You will be able to access this request later to record payments, to re-send or re-bill, or enter notes as needed.
