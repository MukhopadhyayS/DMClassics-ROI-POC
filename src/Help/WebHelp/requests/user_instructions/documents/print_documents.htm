Print documents

This function enables you to print one or more OneContent patient documents or an authorization request.  Output is sent to a local printer, not to a print queue, and the printer must be attached to your computer.

Note: You cannot print non-OneContent documents because the system does not store images of non-OneContent documents; it stores only information about them. If you access a non-OneContent document, the system displays a dialog box that allows you to edit document information. See Maintain non-MPF document information for details.

Also, to print letters, bills, or invoices, you must select the Print option from the Preview dialog box. See About printing, faxing, saving or emailing documents for more information.

  1. Find and view one or more documents. (See View documents for instructions.)
  1. In the viewer application, select Local Print from the File menu, and enter information required by the Print dialog box. Then click OK. The system prints the pages you specified. See online help for the viewer application if you need more instructions about how to print documents you are viewing. If the Local Print option is disabled, then the feature is not properly configured in the Windows registry. Contact your system administrator for help.
