Letters and output

Generate a letter

This function enables authorized users to generate a cover letter or a letter of type "Other" for a request. You can generate a letter regardless of the status of the request.

Note: Use only letters and/or numbers in free-form notes or descriptions. Do not include special characters. Special characters (such as apostrophes, dashes, ampersands, and so on) may cause the system to error when generating letters.

You can generate a cover letter either when you release documents (see Release documents for details) or as an activity separate from releasing documents. Instructions assume that you are not releasing documents and that appropriate letter templates exist.

The ROI Module is delivered with a predefined template for a cover letter, a Pend letter, a Denial letter, and a Cancellation letter. Your enterprise may customize the templates or may create other letter templates for you to use.

  1. Find a request.
  2. Select the Letters function.
  1. Select the type of letter you want to generate (Other or Cover).
  2. Specify the template to use.
  1. Conditional: If the template supports notes, the system displays them. Select notes and/or create free-form notes as desired to include on the letter.
  2. Click Create Letter. The system displays the preview dialog box for the template you chose.
  1. In the Preview dialog box, select the output method you want to use.
  2. Provide information as needed at the print, save as file, fax, or email dialog box, and complete the output operation.
