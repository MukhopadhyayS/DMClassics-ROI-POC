ROI Billing Report form

This report includes financial information of all the requests for all billing location(s) during a particular time.  It provides a snapshot of income, expenses, assets and liabilities for a billing location.

To open the form

  1. Select the Reports tab.
  2. In the Report Name box, select ROI Billing Report.  When the form first displays, the main content pane is blank. To view a report, select the report options in the left pane and click Run Report.

Item descriptions

Report Name

A drop-down list of all available reports. Allows you to select a report to generate.

Date Range

Specifies the time span for the report. You can select from the following options:

  • Today's Date
  • Custom Date.  If selected, allows you to enter the applicable dates in the Start Date and End Date fields.
  • Current Week
  • Current Month
  • Previous Month
  • 3 Months Prior
  • 12 Months Period
  • Year to Date
  • Previous Year

Reset

Clears the pane of any selection criteria and displays only the drop-down field with the list of reports.

Run Report

Generates the report based on the specified report criteria and date range.

Main content pane

The system displays the first page of the generated report. If more than one page exists, the system enables the scroll buttons to allow you to page through the report.

Print Preview

Displays the Print Preview dialog box so you can preview the report before printing it.

Print

Displays the Print dialog box so you can select a printer and printing options.

Export

Displays the Export Report dialog box so you can select a file name, file format, and location for saving the report. The available file formats are PDF, XLS, RTF, HTML, and CSV.
