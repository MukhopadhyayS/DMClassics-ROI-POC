Request Status Summary form

The Request Status Summary form allows you to create the Request Status Summary Report. You can select criteria for the report and specify the date range. Security rights are required.

This report provides a summary of all request statuses.

To open the form

  1. Select the Reports tab.
  2. In the Report Name box, select Request Status Summary.  

When the form first displays, the main content pane is blank. To view a report, select the report options in the left pane and click Run Report.

Item descriptions

Left pane

Facility

Specifies the facility or facilities on which you want to report.

You can select or deselect one or more facilities, click Select All to select all facilities, or click Clear All to clear all previous selections.

Request Status

Specifies the request status or statuses on which you want to report. The available statuses are:

  • Logged
  • Completed
  • Denied
  • Canceled
  • Pended
  • Pre-Billed

You can select or deselect one or more request statuses, click Select All to select all request statuses, or click Clear All to clear all previous selections.

Requestor Type

Specifies the requestor type on which you want to report. Examples are Physician, Attorney, and Patient.

Date Range

Specifies the time span for the report. You can select from the following options:

  • Today's Date
  • Custom Date.  If selected, enter the applicable dates in the State Date and End Date fields.
  • Current Week
  • Current Month
  • Previous Month
  • 3 Months Prior
  • 12 Months Period
  • Year to Date
  • Previous Year

Reset

Clears the pane of any selection criteria and displays only the drop-down field with the list of reports.

Run Report

Generates the report based on the specified report criteria and date range.

Main content pane

View By

Provides filtering options to change the information displayed in the report. You do not need to re-run the report to see the new view. The system automatically changes the view when you make a selection. Options include:

  • Requestor Type
  • Requestor Type, Requestor Name
  • Requestor Type, Requestor Name, Request ID
  • Summary

Print Preview

Displays the Print Preview dialog box so you can preview the report before printing it.

Print

Displays the Print dialog box so you can select a printer and printing options.

Export

Displays the Export Report dialog box so you can select a file name, file format, and location for saving the report. The available file formats are PDF, XLS, RTF, HTML, and CSV.
