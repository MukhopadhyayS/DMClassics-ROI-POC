About this information

Overview

The OneContent Release of Information (ROI) Module is a software application that is designed to meet the information management needs of modern organizations.  The ROI Module is designed to provide an electronic method of managing requests for patient records.

• The administration component of the ROI Module enables ROI administrators to configure and manage informational building blocks that are used to process and manage ROI requests.
• The end-user functions of the ROI Module enable ROI users to create, maintain, and manage ROI requests.

Note: This information is effective with the OneContent™ family of products except where specifically noted. In addition, readers should assume that information for the OneContent™ product is generally applicable to the OneContent™ Business Folder product as well. For easier readability, such passages may cite only "OneContent."

Purpose

This information describes how to use the ROI Module to create and manage ROI records for patients, requestors, and requests. It also tells you how to generate ROI reports.

Scope

This information contains instructions and screen descriptions for users who need to learn and use the OneContent Release of Information (ROI) module. Installation instructions are provided under separate cover.

Audience

This information is intended for administrators and non-administrator users.
