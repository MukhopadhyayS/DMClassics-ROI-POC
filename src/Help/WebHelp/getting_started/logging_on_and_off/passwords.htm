Passwords

When the Directory Services Authentication feature is enabled, you cannot change your OneContent password because the system uses your Windows credentials for security. However, administrators can still reset OneContent passwords.

When your site uses standard authentication, you can change your OneContent password at any time. Occasionally, the system may prompt you to change your password.  

You are prompted to change your OneContent password when:

  • You log on for the first time.
  • Your password has been reset by the system administrator.
  • Your account is locked (such as after a number of unsuccessful login attempts).

Your password must comply with current OneContent security requirements, and if it fails the security checks, the system displays an informative message. When planning your password, ask your system administrator for password rules currently in effect.

Change your password

This function allows you to change your application-based password on demand or when the system requires you to change it.  

You are required to change your application-based password when you have logged on with a temporary password provided by your system administrator. Your system administrator generates a temporary password for you when:

  • You are a new user.
  • Your account is locked (such as when the number of failed logon attempts on your account exceeds the maximum number permitted).
  • Your password has expired.

This action changes your application password. The new password must be different from the current one and must meet password security rules used by the system.

Note: If your site has enabled the Directory Services Authentication feature, you cannot change your system-based password. However, the system administrator can reset your password, if necessary.

  1. If you are already logged on, select the Security > Change Password menu option. Then continue with the next step.

If you are in the process of logging on, enter your current password and click Change Password. Then continue with the next step.

If the system is requiring you to change your password, click OK to acknowledge the notification message, and continue with the next step.

  1. On the Change Password dialog box, enter your current password. Then enter your new password twice for validation purposes. Passwords are case sensitive.
  2. Click OK.
• If the new password fails password security policies, the system informs you. Click OK to try again.
• If the new password meets the password security policies, the system saves your entry and displays a message. Click OK to proceed.
