Output Statements and/or Invoices form

The Output Letters and/or Statements form enables you to generate statements or invoices during the Find Overdue Invoices process.

To open the form

  1. Select Overdue Invoices > Find Overdue Invoices.
  2. In the Find Overdue Invoices form, search for and select one or more invoices.
  3. Click Output.  The system displays the Output Statements and/or Invoices form.

Item descriptions

Statements group box

Enables you to generate a statement and notes.  

Items are:

  • Output Statement
    Triggers the system to create a statement to output.
    When selected, enables the Statement Template and Date Range fields.
  • Statement Template
    Lists all statement templates configured in the system and allows you to select one.
    Defaults to the template configured as the default selection.
    If no default is configured, then defaults to Please Select.
  • Date Range
    Specifies the date range to include in the statement.
    Displays a pre-configured list of choices: 30, 50, 90, or 120 days, Year to Date, and All.  All if the default.
  • Select/Create Notes to add to the Invoice box and Create Freeform Note button
    Displays only when the selected template is configured to support notes.
    Displays all preconfigured notes associated with the template.
    Allows you to select one or more notes to print with the cover letter.
    Allows you to create additional freeform notes.

Preview Before Output

Displays a preview of the statement or invoice prior to printing or saving.

Selected by default.

Allows you to override the selection.

Print

Launches the preview dialog box.

Enabled only if at least one template is selected.

Select Continue to open the Print dialog box.

Select Email to open the Email dialog box.

Save

Launches the Preview of Output dialog box.

Enabled only if at least one template is selected.

Select Continue to open the Save as File dialog box.

Select Email to open the Email dialog box.

Cancel

Cancels the generation of the statement or invoices and returns to the Find Overdue Invoices form.
