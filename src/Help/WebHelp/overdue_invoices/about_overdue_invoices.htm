About Overdue Invoices

The Overdue Invoices tab enables you to search for, view and output overdue invoices by billing location, requestor name, requestor type and aging days.

Invoice Due Days

You can set a default number of days after generation that an invoice is due. There are pre-configured values (for example, 30, 60, and so on) as well as the ability to create a custom value. This value can be shown on the invoice as well as in the invoice history. At the time of generating an invoice, you can change the default.

Searching
• Search results can be sorted, and the invoices can be viewed or output.
• Statements can be generated and output based on the results.
Invoices and Statements

You can generate a requestor statement or invoice independently or at the time of releasing and invoicing records. You can also generate invoices and statements for multiple billing locations or requestors at the same time.

When you select an output method, you can select a template for formatting printouts.

If the invoices are searched for by facility, a summary letter for the requestor cannot be generated: it can only be generated if the Requestor Name is specified. Pre-configured letter templates are provided, however, these templates can be changed to meet your site's needs. When generated, printouts are grouped by:

• Requestor, if you selected the requestor statement template.
• The existing sort order on the Find Overdue Invoices form, if you did not select the requestor statement.

If you do not select a template, the system generates printouts based upon the last sort order selected in the search results.
