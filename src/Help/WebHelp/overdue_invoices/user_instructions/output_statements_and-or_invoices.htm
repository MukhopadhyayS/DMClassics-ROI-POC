Output statements and/or invoices

A statement or invoice can be generated reflecting any overview invoices.

  1. Find an overdue invoice.
  2. On the Find Overdue Invoices form, select the invoice(s) requiring a statement or invoice. The system displays the Output Statements and/or Invoices form.
  3. Make your selections and choose an action such as preview before print.
