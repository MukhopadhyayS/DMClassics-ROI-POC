Find a requestor

This task enables you to search for and select a specific requestor. You can use any search criteria in any combination. However, the system enables or disables data entry fields dynamically, depending on the Requestor Type setting. Basically, patient and non-patient fields are mutually exclusive.

  1. Select Requestors > Find Requestor to access the Find Requestor form.
  1. In the main content pane, set a Filter by option and specify search criteria. The Filter by options control the search operation as follows:
- The Recent Requestors option limits the search to only those requestors who have submitted a request within the last 90 days. Selected by default.
- The All Requestors option allows the search to include all requestors.
  1. Click Find Requestor. The system lists all requestors that meet the search criteria you entered.

Note: The result set is limited to a maximum of 500 records. If more records exist, you can enter more search criteria to narrow the search.

  1. In the detail pane, select the requestor you want and click View/Edit Requestor. The system displays the Requestor Information form.
