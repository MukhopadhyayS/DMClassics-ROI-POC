View request history by requestor

This function enables authorized users to view a list of all requests submitted by a specific requestor and to access more details about a specific request. All information is read-only.

  1. Find a requestor.
  2. Go to the Request History form.
  1. Follow instructions below for the task you want to perform.

To view a request

The Request History form lists each request for the requestor in summarized form. To view more information for a request:

  1. Select the request from the Request grid.

Note: Note: If a request contains records from facilities that you are not allowed to access, the detail pane lists the request in red text and masks data in the Patients column. If you select the request, the system disables the View/Edit Request button.

  1. Click View / Edit Request. The system displays the Requests > Request Information form. You can navigate the forms on the Requests tab to view more information.

To sort requests

Requests initially display in chronological order, beginning with the most recent.

To sort requests in a different order, click on any column heading in the Requests grid to sort the list by that column.
