Maintain adjustments

This task enables authorized users to add, edit, or remove adjustments. Adjustments are used to credit the requestor's account, for example, to offer the requestor a discount.

To add an adjustment

  1. Find a requestor.
  2. Select Requestors > Account Management.
  3. In the Account Management form, click Add Adjustment. The system displays the Adjustments dialog box.
  4. Enter the adjustment details.
  5. Click Save.

To edit an adjustment

You can only edit applied adjustments.

  1. Find a requestor.
  2. Select Requestors > Account Management.
  3. In the Account Management form, open an invoice to edit.  The system displays the Adjustments dialog box.
  4. In the Adjustments dialog box, click the red   to delete the adjustment.
  5. In the Apply field, type the correct adjustment amount.
  1. Click Save.

To remove an adjustment

You can only remove unapplied adjustments.

  1. Find a requestor.
  2. Select Requestors > Account Management.
  3. In the Account Management form, open the unapplied adjustment to remove.  The system displays the Adjustments dialog box.
  4. Click Remove Adjustment.
