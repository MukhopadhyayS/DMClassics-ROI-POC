Set Encounter Filter dialog box

The Set Encounter Filter dialog box allows you to select/deselect one or more encounters to display on the calling form.

If you perform a search for a patient and do not specify an encounter, the system displays all encounters specific to the selected patient and selects all encounters by default. If you do specify an encounter, the system defaults to the specified encounter and deselects all other encounters.

In addition:

  • If EPN is turned on for your system, all encounters are displayed.
  • If EPN is turned off for your system, only the selected encounters for the selected patient record (MRN and Facility) are displayed.

To open the form

Click the Set/Update button on the Patient Records form or the Patient Request History form.

Item descriptions

Encounters grid

Displays a list of encounters for the current patient and allows you to select/deselect encounters.

Initially sorted by Admit Date, most recent to oldest. Can be sorted on all columns.

Contains the following information:

  • Checkbox
    Checked state indicates that items relating to the encounter in this row will appear in the results (documents in the record tree or requests in the request history) when this filter is on.
    Initial sort is checked followed by unchecked.
  • Encounter
    Lists encounter numbers.
    Initial sort order is alphanumeric ascending.
    Self-pay indicates the primary payor is the patient.
  • Patient Status
    Patient status stored in OneContent.
    Initial sort is alphanumeric ascending.
  • Facility
    Identifies the facility where the encounter occurred.
    Initial sort is alphanumeric ascending.
  • Patient Type
    Identifies the type of encounter (for example, Inpatient, Outpatient, or Emergency Room).
    Initial sort order is alphanumeric ascending.
  • Admit Date
    Displays the date and time the encounter occurred; format is mm/dd/yyyy.
    Initial sort order is most recent date to oldest.

Filter By Encounters

Turns filtering on and allows only the selected encounters to display on the calling form.

Enabled only when at least one encounter in the grid is selected.

Defaults to On for EPN systems.

Turn Off Filter

Turns off filtering and allows all encounters to display on the calling form.

Cancel

Leaves the current status of the filter unchanged, closes the dialog box, and returns you to the calling form.

Always enabled.
