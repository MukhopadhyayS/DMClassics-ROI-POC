Create New Attachment dialog box

The Create New Attachment dialog box allows you to acquire a file from one of two locations and to associate it with the patient's

 

OneContent record.

Note: Currently, this function supports attaching only Clinical Summary (CCR/CCD) files.

To open the form

  1. From the Patients tab, find a patient.
  2. Go to the Patient Records form.
  3. Click Add Attachment.

OR

  1. From the Requests tab, create a request or open an existing request.
  2. Go to the Patient Information form. If more than one patient exists on the request, select the patient you want.
  3. Click Add Attachment.  The system displays the Create New Attachment dialog box.

Item descriptions

Patient information box

Displays the patient's name, gender, date of birth, the facility where care occurred, Social Security Number (masked, when enabled), EPN, and MRN. Also contains "VIP" if the patient has VIP status.

Attachment Location

Allows you to select the current location of the attachment file you want to add to the patient's record. You can select eligible files from:

  • A local file location Use this option when the file initially resides in a location on your local hard drive, a network drive, or on transportable media like a thumb drive. It is copied to a secure location in the system when you associate it with a patient record. For item descriptions, see Local File option.
  • An external clinical system uses this option when you want to check a remote system for eligible files. The ROI Module automatically queries the source clinical system associated with the patient's facility and lists all encounters with documents that are eligible to be attached to the patient's record. The association between a facility and a source clinical system is configured using the OneContent Admin > Maintenance > Facility function.

    This option is available only when the dialog box is launched from the Requests > Patient Information form. For item descriptions, see Query Clinical System option.

Save

Effective only for Local File attachments.

Associates your selection(s) with the currently selected patient, and saves the file(s) to the location configured for your system.

Cancel

Discards unsaved entries and returns you to the parent form.
