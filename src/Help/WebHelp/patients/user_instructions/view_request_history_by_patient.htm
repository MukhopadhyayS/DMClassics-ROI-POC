View request history by patient

This function enables you to view a list of all requests for a specific patient and to access more details about a specific request. All information is read-only.

  1. Find a patient.
  2. Go to the Request History form.
  1. Follow instructions below for the task you want to perform.

To sort requests

The system initially displays requests in chronological order, beginning with the most recent.

To sort requests in a different order, click in any column heading in the Requests grid to sort the list by that column.

To view a request

The Request History form summarizes each request for this patient's records.

Note: If a request contains records from facilities that you are not allowed to access, the detail pane lists the request in red text and masks data in the following columns: Requestor Name and Requestor Type. If you select the request, the system disables the View/Edit Request button.

To view more information for a request, select the request from the Request grid and click View / Edit Request, or double-click a selected request. The system displays the Requests > Request Information form. You can navigate the options on the Requests tab to view more information.
