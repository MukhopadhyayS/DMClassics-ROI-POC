Create a non-OneContent patient

You can create records for only non-OneContent patients. A non-OneContent patient does not exist in the OneContent database. This occurs, for example, when you cannot match a patient chart to an OneContent patient. Consequently, you must create a patient record to associate with the chart for the purposes of managing the ROI request for that person.

  1. Select Patients > Find Patient. The system displays the Find Patient form.
  1. In the main content pane, click New Patient. The system displays the Patient Information form.
  1. Indicate whether this patient has VIP status.
  2. Enter the patient's name (required), gender, and date of birth.
  3. Optional: Enter the patient's SSN, EPN, Facility, and/or MRN information, and the patient's address and contact information.
  4. Click Save.
