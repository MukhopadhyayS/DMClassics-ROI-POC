View deficiencies

This task enables you to view deficiency details for all deficiencies associated with a selected encounter. All information is view-only.

  1. Find a patient.
  2. Navigate to the Patient Records form and find the document in the document tree. Use the Expand button to quickly drill down to the document or page level.

You can filter the list to reduce the volume of documents in the list.

Use these options in any combination:

• Set Update
• Record View
• Sort By options
• Only Disclosure Docs
• Show Versions
  1. In the document tree, double-click an encounter that displays the deficiencies icon (), or click the icon itself. The system displays the List of Deficiencies dialog box.
  1. Click Close when you are ready to return to the calling form.
