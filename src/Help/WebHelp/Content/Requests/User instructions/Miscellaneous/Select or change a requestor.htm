Miscellaneous

Select or change a requestor

This function enables you to:

  • Select a requestor for a new ROI request
  • Change a requestor for an existing ROI request

If you change the requestor or the requestor type for an existing request, the system automatically recalculates the billing charges. Additionally, if billing tiers are changed for an existing requestor, those changes are effective only on a day-forward basis and are applied to documents (even those on an existing request) that are released for the first time after the change occurs.

Note: Dialog boxes displayed by this function also permit you to create a requestor, if you don't find the requestor you want. Click the New Requestor button when available, and see Create a requestor for instructions.

To select a requestor for a new request

  1. Create a request.
  2. At the Request Information form, click Select Requestor. The system displays the Select Requestor dialog box.  
  1. In the Select Requestor for this Request dialog box, enter the search criteria you want to use.
- Select a Filter by option.
- Select a requestor type to narrow the search to a specific type. If you set Requestor Type to a non-patient requestor, the system disables the fields for patient information.
- Enter at least three characters at Requestor Name (for non-patient requestor) or at Last Name and/or First Name (for patient requestor).
- Select a status (Active or Inactive).
- For Patient or All Requestor types only: Enter patient information.
  1. Click Find Requestor. The system may display additional dialog boxes, depending on different factors, such as whether you select a patient requestor and whether the patient you select already exists as a requestor. Follow on-screen instructions.
  2. Select a requestor from the list and click Select Requestor for Request. The system applies your choice and returns to the calling form.

To change a requestor for an existing request

If any documents on the existing request have already been released, you cannot change the requestor.

In addition, the system clears any existing patient information if you change the current requestor to a patient requestor. The system retains existing patient information if you change the current requestor to a non-patient requestor.

  1. Find a request.
  2. At the Request Information form, click Change Requestor. The system displays the Change Requestor dialog box.  
  1. In the Change Requestor for this Request dialog box, enter the search criteria you want to use.
- Select a Filter by option.
- Select a requestor type to narrow the search to a specific type. If you set Requestor Type to a non-patient requestor, the system disables the fields for patient information.
- Enter at least three characters at Requestor Name (for non-patient requestor) or at Last Name and/or First Name (for patient requestor).
- Select a status (Active or Inactive).
- For Patient or All Requestor types only: Enter patient information.
  1. Click Find Requestor. The system may display additional dialog boxes, depending on different factors, such as whether you select a patient requestor and whether the patient you select already exists as a requestor. Follow on-screen instructions.
  2. Select a requestor from the list and click Select Requestor for Request. The system applies your choice and returns to the calling form.
