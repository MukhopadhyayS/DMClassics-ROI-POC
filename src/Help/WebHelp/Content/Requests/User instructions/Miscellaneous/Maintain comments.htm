Miscellaneous

Maintain comments

This function enables an authorized user to view or add comments for a request. Once a comment is saved, you cannot change or delete it. You can work with comments either while you are creating a request or when you access an existing request.

The system displays all existing comments for the request in chronological order, beginning with the most recent. You can sort the comments by date/time, author, or actual comment text. In addition, you can filter the comments to display only those entered by a specific author and/or those created on a specific date or within a date range.

  1. Find a request.
  2. Go to the Requests > Comments form.
  1. Follow instructions below for the task you want to perform.

To add a comment

  1. Type your comment in the Enter New Comment box.
  2. Click Save Comment.
  3. At the confirmation prompt, click OK. The system adds your comment as the first item in the Comments grid.

To view a comment

View comments in the Comments grid. Comments initially display in chronological order, beginning with the most recent.

To sort comments

To sort comments in a different order, click in any column heading to sort the list by that value. Initial sort order for Date | Time is most recent to oldest. Initial sort order for Author and Comment is alpha ascending.

To filter comments

To filter the list of comments, do any of the following, in any combination:

  • Select a specific author in the Author box. The default selection is All.
  • Select a standard date range in the Date box: 30 days, 60 days, or 90 days. The default selection is All.
  • To set a specific date or a date range of your choice, select Custom Range in the Date box, then select dates for the From and To boxes. Once you select a date, you can change the month, day, or year value using either the keyboard or the drop-down calendar.
