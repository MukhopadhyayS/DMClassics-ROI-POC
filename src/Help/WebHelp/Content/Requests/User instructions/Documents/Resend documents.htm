Resend documents

This function enables an authorized user to re-send documents on an existing request that have already been released at least once. Use this function when you do not want to generate billing charges for this service. You may need to do this when, for example, a shipment was lost or pages were otherwise destroyed.

Resending documents does not generate a new request, nor does it generate billing charges, Accounting of Disclosures entries, page count updates, release history entries, or invoices.

Note: To re-release documents and generate charges, see Re-release documents.

The system audits the event only when the "R" (ROI Released) audit code is enabled in the OneContent application.

  1. Find a request.
  2. Go to the Requests > Patient Information form.

Note: The system will use the output method currently specified on the Shipping tab on the Release & Invoice form. If you need to specify a different shipping method for this shipment, go to the Output tab, modify information, and then save it.

  1. At the Document(s) Selected for Release box, click the Released filter option to list only those documents that have been released. You must uncheck any documents highlighted in red.  Those documents are moved/deleted from the patient and cannot be released.

Then select the encounter(s), folder(s), document(s), or document page(s) that you want to re-send. You can select an encounter or folder to include all documents it contains, or you can select specific documents or pages from an encounter or folder.  

  1. Click Resend. The system displays a dialog box for the output method currently specified on the Output tab: Print, Fax, Email or Save As File.
  1. Enter data required by the output method and click OK. After output is successfully generated, the system returns you to the Patient Information form.
  2. The release process is complete within the ROI system. Complete any manual steps (such as handing off to the shipper). You will be able to access this request later to record payments, to re-send or re-bill, or enter notes as needed.
