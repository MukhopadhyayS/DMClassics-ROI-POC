Create a request from the Requests tab

This task describes how to create a request from the Requests tab.

1. Select Requests > Find / Create Request.
2. If the Find/Create Request form currently displays patient information, the request you create will be associated with that patient. To create a request for a different patient, click Reset to clear the data.
3. Click New Request. The system displays the Request Information form.
4. At Request Reason, select a request reason.
5. Enter the date the request was received.
6. Click Select Requestor and specify a requestor.
7. Click Save & Continue. The request has been created. You can continue to build the request now or can return to it later.

To continue building the request, refer to the ROI Online help topic, "Add patient to a request."

To continue building the request, see Add a patient to a request.
