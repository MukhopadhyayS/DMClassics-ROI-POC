Delete an attachment from a request

This task enables you to delete an attachment from a request, but the behavior of the Delete function depends on whether the attachment has been released.

If the attachment has not been released, the system:

  • Removes the selected file from the DSR list for this request.
  • Keeps the attachment available in the patient's document tree.

If the attachment has been released, the system:

  • Keeps it on the DSR list and can re-release it.
  • Grays it out on the patient's document tree. You can still view it or edit comments for it, but you cannot select it for any future request.

Note: To delete an unreleased attachment from the system entirely, see Delete attachments from a patient record.

  1. Find a request.
  2. Go to the Requests > Patient Information form.
  3. At the document tree, double-click the attachment you want to delete.
  1. At the Edit Attachment dialog box, click Delete.
  2. Perform one of the following:
- If the file can be deleted, the system prompts for confirmation. Click Delete to complete the action.
- If the file cannot be deleted, the system informs you. Click OK. The system continues to list the file in the document tree but prevents you from adding it to any future request.
