History

View event history

This function enables an authorized user to view the history of all recorded events for a request and see event date/time, originator, and remarks.

The system records the following events for each request:

  • Comment Added
  • Status Reason Added
  • Change of Status
  • Documents Released
  • Documents Resent
  • Invoice Sent
  • Pre-Bill Sent
  • Letter Sent
  • Payment Posted
  • Payment Deleted
  • Adjustment Applied
  • Adjustment Deleted
  • Write Off

The system displays all events in chronological order, beginning with the most recent. You can sort the event list by date/time, event, originator, or event remarks text. In addition, you can filter the events to display only those of a specific type, those triggered by a specific originator, and/or those that occurred on a specific date or within a date range.

  1. Find a request.
  2. Go to the Requests > Event History form.
  1. Follow instructions below for the task you want to perform.

To view an event

View event details in the Events grid.

To sort events

Events initially display in chronological order, beginning with the most recent.

To sort events in a different order, click any column heading in the Events grid to sort the list by that column.  

To filter events

To filter the list, do any of the following, in any combination:

  • Select a specific event in the Event box. The default selection is All Events.
  • Select a specific originator in the Originator box. The default selection is All.
  • Select a standard date range in the Date box: 30 days, 60 days, or 90 days. The default selection is All.
  • To set a specific date or a date range of your choice, select Custom Range in the Date box, then select dates for the From and To boxes. Once you select a date, you can change the month, day, or year value using either the keyboard or the drop-down calendar.
