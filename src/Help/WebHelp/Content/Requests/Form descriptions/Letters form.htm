Letters form

The Letters form enables an authorized user to generate a letter for the currently selected request. You can select from any available letter template, and can elect to print or fax the output, or export it to a PDF file.

To open the form

  1. Select the Requests tab.
  2. Search for and select a request.
  3. Click View/Edit Request.
  4. Select the Letters function.

Note: You can also access this form after you save a new request.

Item descriptions

Requestor information box

Identifies the requestor type, the requestor's name and demographic information, and the requirements for pre-payment and certification letters (if any). Also displays the request ID, current status, and receipt date and time.

Letter Type

Allows you to select the type of letter you want to generate for the request. Drives the values populated in Letter Template.

Options are:

  • Other (selected by default)
  • Cover Letter

Letter Template

Allows you to select a template to use for generating a letter for the request.

Options depend on the value of Letter Type.

Displays Please Select, except when type is Cover Letter and a default template has been designated. In this case, defaults to the name of the default template.

Notes

Displays only if the selected letter template supports notes.

Lists all notes currently defined in the ROI Module and allows you to select one or more notes to include in the letter.

Create Freeform Note

Displays only if the selected letter template supports notes.

Allows you to create a note.

Create Letter

Triggers the creation of the specified letter and displays the result in a viewer window.

Enabled only after you select a letter template.

Cancel

Cancels the creation of the letter, discards your entries, and reverts all fields to their default settings.

Enabled only after you have made a change to any default setting.
