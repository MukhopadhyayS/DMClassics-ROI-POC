Preview of Invoice dialog box

The Invoice preview displays the invoice you are generating, and outputs the form using the selected method.

To open the form

  1. On the Release & Invoice form, click Release & Invoice.  
  2. In the Release dialog box, select a template, enter the invoice due days and any notes, and click Continue.  The system displays the applicable Preview dialog box.

Note: If there are unapplied adjustments or payments, the Unapplied Adjustments / Payments dialog box displays. Click Yes or No, then Save & Continue.

Item descriptions

Continue

Closes the Preview dialog box and launches the dialog box for the output method (Print, Fax, Save as File or Email) specified on the Output tab of the Release & Invoice form.

Note: You must output the documents in order for them to be released.  

Cancel

Cancels the creation of the cover letter and the release of the documents, closes the Release dialog box, and returns you to the Release & Invoice form.

Page controls

Functions (such as save the file, navigate pages, and zoom) that are available from the third-party product are used to display previews.
