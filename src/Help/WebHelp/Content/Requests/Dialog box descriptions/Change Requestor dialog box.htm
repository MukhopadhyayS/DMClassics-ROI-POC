Change Requestor dialog box

The Change Requestor dialog box enables you to search for and select a requestor to associate with a request.

To open the form

  1. Select the Requests tab.
  2. Search for and select a request.
  3. Click View/Edit Request.
  4. On the Request Information form, click Change Requestor. The dialog box displays two tabs: Find Requestor and Requestor Information.

Item descriptions

Find Requestor tab

The Find Requestor tab displays by default.

This tab uses the same items as the Requestors > Find Requestor form with the exception of the Delete Requestor and Create Request buttons, which do not display on this tab.

See Find Requestor form for item descriptions.

Requestor Information tab

The Requestor Information tab can be accessed only if you click either New Requestor or View/Edit Requestor on the Find Requestor tab.

This tab uses the same items as the Requestors > Requestor Information form with the exception of the Delete Requestor and Create Request buttons, which do not display on this tab.

See Requestor Information form for item descriptions.

Additional items

This dialog box contains the following additional items.

Select Requestor for Request

Saves any unsaved data, records the selected requestor as the requestor for the request, and closes the dialog box.

If you selected a patient requestor, launches the Select Patient dialog box. Otherwise, displays the Requestor Information form.

Enabled only after you select an existing requestor or when you are creating a new requestor.

Note: If you updated contact and shipping information on the request, and then updated the requestor’s contact and address information, the system prompts you to confirm that you want to apply the same changes to the corresponding information on the request. You have the option of applying updates to the request or leaving the request information unchanged.

Cancel

Discards unsaved changes, cancels the selection of a requestor, and returns you to the Request Information form.
