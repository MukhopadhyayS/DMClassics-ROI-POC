Select Patient dialog box

The Select Patient dialog box enables you to add patients to a request when the requestor is a patient. The system displays a list of patients that most closely match the requestor.

To open the form

  1. Select the Requests tab.
  2. Create a request or open an existing request.
  3. At the Request Information form, click Select Requestor (for a new request) or Change Requestor (for an existing request).
  4. At the Select Requestor or Change Requestor dialog box, search for and select a requestor and click Select Requestor for Request.

Item descriptions

Note: Date and time format are controlled by the regional settings of your operating system.

Patients grid

Displays an entry for each patient record that closely matches the patient requestor's information and allows you to select one or more records to add to the request. You are not required to select a record.

Initially sorted by Patient Name in alphanumeric ascending order. Can be sorted by any column. By default, the first row is auto-selected.

Information includes:

  • Checkbox column
    Indicates whether to include the corresponding record on the request.
    Can be checked or unchecked as desired.
    Unchecked by default.
  • Patient Name
    The name of the patient.
    Sorts on the first characters displayed, regardless of which name (first or last) occurs first.
  • DOB
    The date of birth of the patient.
    Initial sort is date ascending.
  • SSN
    The patient's Social Security Number.
    Initial sort is numeric ascending.
  • Facility
    The facility where patient care occurred.
    Initial sort is alphanumeric ascending.
  • MRN
    The patient's medical record number.
    Initial sort is alphanumeric ascending.
  • EPN (displays only for EPN systems)
    The patient's enterprise person number.
    Initial sort is alphanumeric ascending.

Continue with Selected Patient(s)

Adds the selected patient(s) to the request, closes the dialog box, and returns you to the Request Information form, where you can continue to work on the request.

Enabled only after you select at least one patient.

Continue with No Patient

Allows you to continue working on the request even though you have not selected any patients to add to the request. Closes the dialog box and returns you to the Request Information where you can continue to work on the request.

Always enabled.

Cancel

Discards any unsaved actions and returns you to the calling form.
