Email dialog box

The Email dialog box enables you to email OneContent items and supplemental documents, invoices, pre-bills, letters, statements and attachments.

To open the form

The system launches the dialog box when you do any of the following:

  • Click Continue on the Release & Invoice form and make any of the following selections:
  • Release
  • Re-Release
  • Invoice
  • Pre-Bill
  • Click Resend from the Patient Information form.
  • Click Create Letter from the Letters form.

Item descriptions

Email group box

  • Name
    Lists all configured email queues.
    Defaults to the first queue in the list the first time you access this function or when the last used queue has been deleted. Otherwise, defaults to the last queue selected.
  • Password
    Allows you to specify the password required to open the PDF document, or displays the pre-configured password for the PDF file.
  • To
    Displays the email address of the requestor if contained in the requestor's profile. If no address is shown, enter the applicable email address.
  • Subject
    Displays any pre-configured text.  You must have the "Allow user" option selected in OneContent to add or edit text.
  • Message
    Displays any pre-configured text.  You must have the "Allow user" option selected in OneContent to add or edit text.
  • Status
    Displays the status of the queue.
  • Type
    Displays the type of queue.

Add to Released Document group box

Allows you to print a header, footer, and/or watermark on all released documents (not on letters).

Displays only when the output dialog box has been launched from the Release dialog box and the header, footer, and/or watermark options are enabled in the Configure Output Settings function (any disabled option does not display). Displayed options are enabled only if a corresponding value has been configured.

  • Headers and footers can display an absolute page number, a relative page number, the patient name, and the encounter number, MRN, or EPN.
  • Watermarks can contain either the default value or any text string you specify up to 20 characters.

OK

Submits the job to the selected email queue and closes the dialog box.  If an error occurs, the system alerts you.

Cancel

Discards the job, closes the dialog box, and returns you to the calling form or dialog box.
