Print dialog box

The Print dialog box enables you to print cover letters, invoices, pre-bills, statements and patient documents that are included in a release request.

To open the form

The system launches the dialog box when you do any of the following:

  • Click Continue from the Release dialog box without a letter selected
  • Click Print from the Letters preview dialog box
  • Click Print from the Invoice preview dialog box
  • Click Print from the Pre-Bill preview dialog box

Item descriptions

Printer group box

Allows you to select a printer and displays printer attributes. Also allows you set printer properties.

  • Name
    Lists all printers that have been configured in alphanumeric order.
    Defaults to the first printer in the list for first-time print or when the last-used printer has been deleted; otherwise defaults to the last printer you selected.
  • Properties
    Allows you to view and update properties for the selected printer to output the selected job.
    Changes apply to the current job only.
    Options depend on the printer you select and cannot be predicted.
  • Status, Type, Where, and Comment
    Read-only.
    Displays information for the queue you selected, including the queue status, the queue type, the destination where the output will go, and any comment associated with the queue.
  • Page Separator
    Displays only when the Page Separator option is enabled in the Configure Output Settings function; otherwise, does not display.
    Defaults to unchecked.
    When enabled, inserts a separator sheet between released documents.
    When used, the separator sheet contains the user name (lastname, firstname), user department, printer name, and date and time according to system settings.

Add to Released Document group box

Allows you to print a header, footer, and/or watermark on all released documents (not on letters).

Displays only when the output dialog box has been launched from the Release dialog box and the header, footer, and/or watermark options are enabled in the Configure Output Settings function (any disabled option does not display). Displayed options are enabled only if a corresponding value has been configured.

  • Headers and footers can display an absolute page number, a relative page number, the patient name, and the encounter number, MRN, or EPN.
  • Watermarks can contain either the default value or any text string you specify up to 20 characters.

Copies group box

Contains these controls:

  • Number of Copies
    Allows you to specify the number of copies to print, from one to 99 copies.
  • Collate
    Enabled only when Number of Copies is more than one.

OK

Submits the job to the selected printer and closes the dialog box.

If an error occurs, the system alerts you.

Cancel

Discards the job, closes the dialog box, and returns you to the calling form or dialog box.
