Fax dialog box

The Fax dialog box enables you to fax cover letters, invoices, pre-bills, statements and patient documents that are included in a release request.

To open the form

The system launches the dialog box when you do any of the following:

  • Click Continue from the Release Documents dialog box without a letter selected
  • Click Fax from the Letters preview dialog box
  • Click Fax from the Invoice preview dialog box
  • Click Fax from the Pre-Bill preview dialog box

Item descriptions

Fax group box

Allows you to specify a fax number and queue and displays fax attributes. Also allows you set fax properties.

  • Number
    Automatically displays the requestor's fax number, when available.
    Allows you to override the pre-populated number.
  • Name
    Lists all fax queues that have been configured, in alphanumeric order.
    Defaults to the first fax queue in the list for first-time fax or when the last-used fax queue has been deleted; otherwise defaults to the last fax queue you selected.
  • Status, Type, Where, and Comment
    Read-only.
    Displays information for the queue you selected, including the queue status, the queue type, the destination where the output will go, and any comment associated with the queue.

Add to Released Document group box

Allows you to print a header, footer, and/or watermark on all released documents (except letters).

Displays only when the output dialog box has been launched from the Release dialog box and the header, footer, and/or watermark options are enabled in the Configure Output Settings function (any disabled option does not display). Displayed options are enabled only if a corresponding value has been configured.

  • Headers and footers can display an absolute page number, a relative page number, the patient name, and the encounter number, MRN, or EPN.
  • Watermarks can contain either the default value or any text string you specify up to 20 characters.

OK

Submits the job to the selected fax queue and closes the dialog box.

If an error occurs, the system alerts you.

Cancel

Discards the job, closes the dialog box, and returns you to the calling form or dialog box.
