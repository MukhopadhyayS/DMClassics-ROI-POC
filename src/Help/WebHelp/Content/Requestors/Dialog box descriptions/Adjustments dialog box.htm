Adjustments dialog box

The Adjustments dialog box allows you to manage adjustments for a requestor's account.

To open the form

  1. Find a requestor.
  2. Select Requestors > Account Management.
  3. Click Add Adjustment.  The system displays the Adjustments dialog box.

Item descriptions

Adjustment attributes

  • Type
    The type of adjustment: Options are Billing Adjustment, Customer Goodwill, and Bad Debt.
  • Amount
    The amount of the adjustment.
  • Reason
    The reason for the adjustment.  Select a pre-defined reason from the list (configured in Administration > Reasons Maintenance).
  • Note
    The note explaining the reason for the adjustment.  Type the note text.
  • Date
    The date of the adjustment.  The default is today's date. Click the calendar icon to change the date.
  • Adjustment Amount
    The total amount of all adjustments before being applied to any invoice(s).
  • Applied
    The amount of all applied adjustments (manual and auto apply).
  • Refunded
    The amount of all refunds associated with the adjustment.

Note: You cannot apply a refund in this dialog box.  See Refunds.

  • Unapplied
    The amount of the adjustment not applied to an invoice.

Auto Apply

Applies the adjustment amount to any open invoices, starting with the oldest open invoice first.  The system continues to apply the remaining adjustment amount to the next open invoice until all invoices are closed.

Apply Adjustment

  • Invoice #
    List of all open invoices, sorted chronologically with the oldest first.
  • Request ID
    Request ID associated with the invoice.
  • Date
    Date the invoice was created.
  • Charges
    Original charges when invoice was generated.
  • Pay/Adj
    Total payments/adjustments for the invoice.
  • Applied
    Amount of the adjustment previously applied to the invoice.  When in "modify mode," the applied field has a red X used to unapply the adjustment.
  • Balance
    Current invoice balance (charges - payments + adjustments)
  • Apply
    The amount of the adjustment to apply to the individual invoice.

Save

Saves the new or modified adjustment.

Cancel

Cancels the adjustment you were adding or any unsaved changes made to an existing adjustment.

Remove Adjustment (conditional display)

Removes the selected adjustment.  

Enabled only when the full adjustment amount is unapplied.
