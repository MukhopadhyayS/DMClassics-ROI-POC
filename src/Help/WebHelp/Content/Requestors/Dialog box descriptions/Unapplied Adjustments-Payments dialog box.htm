Unapplied Adjustments/Payments dialog box

The Unapplied Adjustments/Payments dialog box allows you to apply any unapplied adjustments or payments on the requestors account when generating an invoice.

To open the form

  1. Create a request.
  2. On the Release & Invoice form, click Release & Invoice.  If there are unapplied adjustments/payments, the system displays the Unapplied Adjustments/Payments dialog box.

Item descriptions

Yes/No

Indicates if you want to apply any unapplied adjustments or payments.  Yes is selected by default and any unapplied amount is automatically shown in the Amount to be applied field.  

Unapplied Adjustments/Unapplied Payments

The total amount of all unapplied adjustments and payments.

To be applied

The total amount to be applied to the invoice being created.

Cancel

Closes the dialog box and returns to the Release dialog box.

Save & Continue

Saves the selection (either Yes or No) and opens the Preview of Invoice dialog box.
