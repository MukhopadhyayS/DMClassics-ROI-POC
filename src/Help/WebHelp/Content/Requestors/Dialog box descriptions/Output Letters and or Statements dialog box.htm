Output Letters and/or Statements dialog box

The Output Letters and Statements form enables you to generate a letter or statement for a refund.

Note: The Refund button is enabled only when a credit balance exists for the requestor's account.

To open the form

  1. Find a requestor.
  2. In the left pane, select Account Management.
  3. In the Account Management form, click Refund.  The system displays the Refund dialog box.
  4. Complete the dialog box information and click Save + Output Letter. The system displays the Output dialog box.

Item descriptions

Refund Letter group box

Enables you to generate a refund letter and notes to include with the refund.

Items are:

  • Output Refund Letter
    Triggers the system to create a refund letter to output.
  • Template
    Displays a list of all configured templates.
  • Select/Create Notes to add to the Letter box and Create Freeform Note button
    Displays only when the selected template is configured to support notes.
    Displays all preconfigured notes associated with the template.
    Allows you to select one or more notes to print with the refund letter.
    Allows you to create additional freeform notes.

Statements group box

Enables you to generate a statement and notes to include with the refund.

Items are:

  • Output Statement
    Triggers the system to create a statement to output.
    When selected, enables the Statement Template and Date Range fields.
  • Statement Template
    Lists all statement templates configured in the system and allows you to select one.
    Defaults to the template configured as the default selection.
    If no default is configured, then defaults to Please Select.
  • Date Range
    Specifies the date range to include in the statement.
    Preconfigured list of choices includes: 30, 50, 90, or 120 days, Year to Date, and All.
  • Select/Create Notes to add to the Invoice box and Create Freeform Note button
    Displays only when the selected template is configured to support notes.
    Displays all preconfigured notes associated with the template.
    Allows you to select one or more notes to print with the statement letter.
    Allows you to create additional freeform notes.

Print

Launches the Preview of Refund dialog box.

Save

Saves the current edits.

Cancel

Cancels printing of the letter/statement.
