Create a requestor statement

A statement showing a summary of all transactions for a requestor can be generated at the time the request is released and invoiced.

To generate a requestor statement:

  1. Find, select and open a request.  
  2. On the Release & Invoice form, make the applicable selections including the output method.
  3. Click Release & Invoice.  
  4. In the Release dialog box, click Output Statement, select a template, specify a date range, enter any notes, and click Continue.  The system displays the Preview of Statements dialog box.
  5. Click Continue.  This displays the appropriate output method dialog box, for example, print, fax, email or save as file.
  6. Complete the dialog box and click OK.
