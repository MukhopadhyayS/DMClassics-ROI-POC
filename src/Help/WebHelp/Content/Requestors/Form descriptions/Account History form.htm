Account History form

The Account History form allows authorized users to view Pre-Bills, Invoices and Statements for the selected requestor.

To open the form

  1. Select Requestors > Find Requestor.
  2. Search for and select a requestor. Then click View/Edit Requestor.
  3. Select Account History in the left pane.

Note: You can access this form at any time while viewing and/or editing requestor information.

Item descriptions

Requestor information box

Identifies the requestor type, the requestor's name and demographic information, and the requirements for pre-payment and certification letters (if any).

Pre-Bills grid

Contains the following information:

  • Status
    Displays the status of Inactive or Active.
  • Pre-Bill Number
    Displays the pre-bill number.
  • Request Id
    Displays system-assigned ID of the request.
  • Date Created
    Displays the date the pre-bill was created.
  • Aging (Days)
    Displays number of days since the pre-bill was generated.
  • Created by
    Identifies the user who generated the pre-bill.
  • Template Used
    Displays the name of the template used to create the pre-bill.
  • Request Password
    Displays the system-generated password that protects documents for a specific request.
  • Queue Password
    Displays the user-defined password to protect documents for a specific request.
  • Balance
    Displays the total charges (in dollars) for the request.

Invoices grid

Contains the following information:

  • Status
    Displays the current status of the invoice.
  • Invoice Number
    Displays the number of the invoice.
  • Request Id
    Displays the system-assigned ID of the request.
  • Date Created
    Displays the date the invoice was created.
  • Due Date
    Indicates the date the invoice is due.
  • Created by
    Identifies the user who created the invoice.
  • Template Used
    Displays the name template used for creating the invoice.
  • Request Password
    Displays the system-generated password that protects documents for a specific request.
  • Queue Password
    Displays the user-defined password to protect documents for a specific request.
  • Balance
    Displays the total charges (in dollars) for the request.

Statements Grid

Contains the following information:

  • Statement ID
    Displays the system-assigned ID.
  • Date Created
    Displays the date the statement was generated.
  • Created by:
    Identifies the user who created the statement.
  • Template Used
    Displays the name of the template used to generate the statement.

View

Allows you to view a selected item.

Enabled after you select a request.
