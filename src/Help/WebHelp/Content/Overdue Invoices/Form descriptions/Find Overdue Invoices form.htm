Find Overdue Invoices form

The Find Overdue Invoices form lists all overdue invoices based upon selection criteria.

To open the form

Select Overdue Invoices > Find Overdue Invoices.

Item Descriptions

Find Overdue Invoices

Contains the following items:

  • Billing Location
    The billing location for the selected invoice.
  • Requestor Type
    The requestor type.
  • Requestor Name
    The requestor's name.
  • Days Overdue
    The number of days the invoice payment is overdue. Use the slider or enter values in the text boxes.
  • Search
    Searches the system for results matching the selection criteria.
  • Reset
    Resets all the current selection criteria.

Invoices grid

Contains the following items:

  • Number of invoices
    Displays the number of invoices matching the search criteria.
  • Billing Location
    Displays the billing location for the selected invoice.
  • Requestor Type
    Displays the requestor type.
  • Requestor Name
    Displays the requestor's name.
  • Invoice Number
    Displays the invoice number.
  • Request ID
    Displays the system request ID.
  • Overdue Amount
    Displays the amount overdue.
  • # Days Overdue
    Displays the number of days the invoice is overdue.
  • Phone Number
    Displays the phone number of the requestor.

View Highlighted

Opens the Preview of Invoice dialog box.

Output

Launches the Output Statements and/or Invoices form.
