Delete attachments from a patient record

This task enables you to delete attachments from the patient record, but the behavior of the Delete function depends on whether the attachment has been released on a request:

If the attachment has not been released

The system removes the selected file from the patient's document tree and deletes it from the storage folder.

If the attachment has been released

The system:

  • Keeps it on the DSR list and can re-release it.
  • Grays it out on the patient's document tree. You can still view it or edit comments for it, but you cannot select it for any future request.

Note: To delete an attachment from a request but leave it in the patient record, see Delete an attachment from a request.

  1. Find a patient.
  2. Go to the Patient Records form, and double-click the attachment in the document tree.
  3. On the Edit Attachment dialog box, click Delete.
  4. Do one of the following:
• If the file can be deleted, the system prompts for confirmation. Click Delete to complete the action.
• If the file cannot be deleted, the system informs you. Click OK. The system continues to list the file in the document tree but prevents you from adding it to any future request.
