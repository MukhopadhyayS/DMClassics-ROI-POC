Create New Patient Requestor dialog box

The Create New Patient Requestor dialog box enables you to quickly add a new patient requestor.

To open the form

The dialog displays when you have selected to create a new patient requestor with the patient selected.

It is triggered from the New Requestor button on the Create a New Request with the Selected Patient Requesting (Matching Requestors Found) dialog.

Item descriptions

Patients grid

Displays an entry for each patient name you selected on the calling form and allows you to select only one patient as the requestor.

Initially sorted by Patient Name in alphanumeric ascending order. Can be sorted by any column. By default, the first row is auto-selected.

Information includes:

  • Patient Name
    The name of the patient selected in the calling form.
    Sorts on the first characters displayed, regardless of which name (first or last) occurs first.
  • DOB
    The date of birth of the patient.
  • SSN
    The patient's Social Security Number.
  • Facility
    The facility where patient care occurred.
  • MRN
    The patient's medical record number.
  • EPN (displays only for EPN systems)
    The patient's enterprise person number.

Select Patient as Requestor

Creates a new requestor record, using information for the patient you selected, and inserts the patient's information on the request you are creating.

Always enabled.

Launches the Save New Patient Requestor dialog box.

Cancel

Stops the process of creating a new request, closes the dialog box, and returns you to the calling form.
