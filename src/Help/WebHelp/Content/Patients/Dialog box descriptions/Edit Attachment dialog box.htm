Edit Attachment dialog box

The Edit Attachment dialog box enables you to modify a subset of information for an existing attachment file.

To open the form

  1. Select the Patients tab and find a patient.
  2. Go to the Patient Records form.
  3. Select an existing attachment in the patient's document tree and click Edit Attachment.

OR

  1. Select the Requests tab.
  2. Open an existing request.
  3. Go to the Patient Information form and, if more than one patient exists on the request, select the patient you want.
  4. Select an existing attachment in the patient's document tree and click Edit Attachment.

Item descriptions

Patient information box

Displays the patient's name, gender, date of birth, the facility where care occurred, Social Security Number (masked, when enabled), EPN, and MRN. Also contains "VIP" if the patient has VIP status.

Attachment Location

Display only. The original location of the attachment (local file or clinical system).

Cannot be modified.

Title

The title you want to give to this attachment. Displays in the document tree.

Free-form. For existing attachments, can be edited until the file is released.

Date of Service

The date that patient care occurred.

For existing attachments, can be edited until the file is released.

File Name

The name of the file, including the extension. Auto-populated by concatenating file type, patient name, date, and extension

Cannot be modified.

Page Count

The number of pages in the document.

Can be edited until the file is released.

Document Source

The name of the assigning authority or the source system (for example, Paragon).

For existing attachments, can be edited until the file is released.

Facility

The facility where patient care occurred.

For existing attachments, cannot be edited.

Enc/Acct

The encounter number or account number for this patient.

For existing attachments, can be edited until the file is released.

Date Uploaded

The date and time that the attachment was associated with the patient's record.

Cannot be modified.

Comment

Free-form comment.

For existing attachments, can be edited even after the file is released.

Save

Effective only for Local File attachments.

Saves any changes you made to file metadata.

Cancel

Discards your entries and closes the dialog box.

Delete

Deletes the attachment.

An attachment can be deleted even after it has been released. The attachment file itself is not deleted from the system; it is deleted only from the request. The attachment will still be listed in the patient's record. However, you cannot add it to future release requests.
