Create New Request with Patient Requesting dialog box

The Create New Request with Patient Requesting dialog box allows you to select a requestor for a record request, using a patient as the requestor. The content of this dialog box depends on whether the patient already exists in the system as a requestor.  

To open the form

  1. Go to Patients > Find Patients and perform a search.
  2. Select a single patient (in either an EPN or non-EPN system) or multiple patients (in a non-EPN system) from the search results.
  3. Click Patient Requesting.

Item descriptions

When the patient does not already exist as a requestor

Patients grid

Displays an entry for each patient name you selected on the calling form and allows you to select only one patient as the requestor.

Initially sorted by Patient Name in alphanumeric ascending order. Can be sorted by any column. By default, the first row is auto-selected.

Information includes:

  • Patient Name
    The name of the patient selected in the calling form.
    Sorts on the first characters displayed, regardless of which name (first or last) occurs first.
  • DOB
    The date of birth of the patient.
  • SSN
    The patient's Social Security Number.
  • Facility
    The facility where patient care occurred.
  • MRN
    The patient's medical record number.
  • EPN (displays only for EPN systems)
    The patient's enterprise person number.

Select Patient as Requestor

Creates a new requestor record, using information for the patient you selected, and inserts the patient's information on the request you are creating.

Always enabled.

Launches the Save New Patient Requestor dialog box.

Cancel

Stops the process of creating a new request, closes the dialog box, and returns you to the calling form.

When the patient already exists as a requestor

Requestors grid

Displays the requestor search result set and allows you to a select requestor.

Initially sorted by Requestor Name in alphanumeric ascending order. Can be sorted by any column.

Information includes:

  • Requestor Name
    The name of the patient requestor.
  • Address
    The street address, city, state, and zip code of the patient requestor.
  • Phone
    The home phone for a patient requestor.
  • Fax
    The fax number associated with the requestor's main address.
  • DOB
    The patient requestor's date of birth in mm/dd/yyyy format.
  • SSN
    The patient's Social Security Number.
  • EPN (displays only for EPN systems and Requestor Type is Patient)
    Displays the patient's enterprise person number.

Select Requestor

Creates a new request with the selected patient requestor.

Enabled only after you have selected a requestor.

Closes the dialog box, moves you to the Requests tab, and displays the Select Requestor dialog box with a new request displayed.

New Requestor

Creates a new requestor with the selected patient.

Closes the dialog box.

Cancel

Stops the process of creating a new request, closes the dialog box, and returns you to the calling form.
