Edit Non-MPF document dialog box

The Edit Non-MPF document dialog box allows you to edit information for a non-OneContent document that has not been released.

Note: The Edit Non-MPF Document dialog box shares fields with the Create New Non-MPF Document dialog box. Field descriptions include information for both dialog boxes.

To open the form

On the Patient Records form, double-click a non-OneContent document.

Item descriptions

Enc/Acct

Allows you to enter or change the encounter or MRN number for the non-OneContent document.

If left blank, the system uses the encounter / MRN number for the currently selected patient.

Document Type

Displays a list of all document types defined for your system and allows you to select a type.

  • For a new non-OneContent document, defaults to Select Document Type
  • For an existing non-OneContent document, displays the document type currently associated with the document

Date of Service

Identifies the date of service for the non-OneContent document. Format is mm/dd/yyyy. Can be edited.

  • For a new document, defaults to blank
  • For an existing document, displays the date currently in effect for the document

Facility

Displays a list of all facilities for which you have permission and allows you to select a facility as the source for the non-OneContent document.

  • For a new document, defaults to Select Facility
  • For an existing document, displays the facility currently associated with the document

Department

Displays a list of all departments associated with the facility identified in the Facility box and allows you to select a department that is responsible for the non-OneContent document.

  • For a new document, defaults to Select Department
  • For an existing document, displays the department currently associated with the document, or, if you select a different facility, resets to Select Department

Comment

Allows you to enter comments about the non-OneContent document.

  • For a new document, is blank
  • For an existing document, displays comments currently associated with the document

Save

Saves your entries for the non-OneContent document.

Enabled only after a valid entry exists in all required fields.

  • For a new document, adds the document to the patient's record on the Patient Records form
  • For an existing document, updates the document and returns to the Patient Records form

Cancel

Discards unsaved data, closes the dialog box, and returns you to the calling form.

Delete

Deletes the selected non-OneContent document.

  • Displays only if you are editing an existing non-OneContent document and only if you have Create/Modify permissions. Prompts for confirmation to delete
  • Enabled only when the document is eligible for deletion
