Encounter Details dialog box

The Encounter Details dialog box displays information about encounters for the currently selected patient.

Note: To view or edit custom fields implemented by your facility, you must have the View Custom Info or Custom Info Maintenance permission enabled on your OneContent user profile.

To open the form

  1. Search for and select a patient.
  2. At the Patient Records form, click Enc. Details.

Item descriptions

Encounters grid

Displays details about all encounters for the selected patient.

Default sort is by Admitted, with the most recent encounters first. Can be sorted by any column.

Read-only format. Displays the following information:

  • Encounter
    The encounter number.
  • MRN
    The patient's Medical Record Number.
  • Admitted
    The date and time when the patient was admitted.
  • Discharge
    The date and time when the patient was discharged.
  • Patient Service
    The type of hospital service rendered.
  • Disposition
    Discharge disposition.
  • Class
    The financial class of the patient.
  • Balance
    Total current balance (in dollars).
  • Clinic
    The site (or clinic) within the facility where patient care occurred.
  • Self-Pay
    The primary plan for this encounter is self-pay.
  • Custom Fields
    Columns of data that display only if any custom fields exist for your site and you have permission to view them.

Close

Closes the dialog box and returns you to the calling form.
