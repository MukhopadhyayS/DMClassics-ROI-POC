Maintain request reasons

To add, modify, or delete the reasons available for an ROI request, do the following.

  1. Access the Request Reasons form.
  2. Follow the instructions below for the task you want to perform.

To add a new request reason

  1. In the main content pane, click Create New.
  2. In the detail pane, enter a Reason Name and Display Text for the request reason.
  3. Select the Attribute (TPO or Non-TPO).
  4. Click Save.

To modify an existing request reason

  1. In the main content pane, select the request reason you want to modify.
  2. In the detail pane, change the information as needed.
  3. Click Save.

To delete a request reason

  1. In the main content pane, select the request reason you want to delete.
  2. Click Delete.
  3. In the confirmation message, click Delete.
