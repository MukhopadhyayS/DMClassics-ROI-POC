Billing Templates form

The Billing Templates form allows you to view, add, modify, or delete billing templates that can be used when billing for ROI requests.

Billing templates can be associated with one or more fee types and one or more requestor types such as Attorney, Hospital, or Patient.

To open the form

Select the Administration tab > Billing Maintenance > Billing Templates.

Item descriptions

Main Content Pane

Billing Templates grid

Displays the billing templates and associated information currently defined in the system. The grid title is preceded by the number of existing billing templates.

The Billing Templates grid contains the following columns:

  • Template Name
    Names of the billing templates currently defined in the system.
  • Fee Types
    Fee types associated with the billing templates.

Create New

Allows you to enter the information needed to add a new billing template.

To create a billing template, the fee types you need must already be defined in the system.

Delete

Deletes the selected billing template.

Note: You cannot delete billing templates that are associated with requestor types.

Detail Pane

Template Name

Specifies the name for a billing template. They must be unique. They are not case sensitive.

Fee Types

Allows you to select or deselect one or more fee types to associate with the billing template.

Save

Saves the new or modified billing template.

Cancel

Cancels the billing template you were adding or any unsaved changes you made to an existing billing template.
