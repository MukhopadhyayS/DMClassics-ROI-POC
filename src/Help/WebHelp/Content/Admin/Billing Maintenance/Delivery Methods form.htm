Delivery Methods form

The Delivery Methods form allows you to view, add, modify, or delete methods for shipping ROI requests.  Examples include courier, FedEx, UPS, and the US Postal Service.

To open the form

Select the Administration tab > Billing Maintenance > Delivery Methods.

Item descriptions

Main Content Pane

Delivery Methods grid

Displays the delivery methods and associated information currently defined in the system. The grid title is preceded by the number of existing delivery methods.

The Delivery Methods grid contains the following columns:

  • Default
    If checked, indicates the default delivery method for shipping ROI requests.
  • Name
    Names of the delivery methods currently defined in the system.
  • Description
    Descriptions of the delivery methods (optional).
  • URL
    Website URLs for the delivery methods (optional).

Create New

Allows you to enter the information needed to add a new delivery method for shipping ROI requests.

Delete

Deletes the selected delivery method for shipping ROI requests.

Detail Pane

Default

Indicates whether this is the default delivery method for shipping ROI requests.

Name

Specifies a name for the delivery method for shipping ROI requests. Delivery method names must be unique. They are not case sensitive.

URL

Provides the URL for the delivery method website for shipping ROI requests. ROI users can access this website to determine shipping costs. You can change the URL at any time. They are optional.

Description

Provides a description for the delivery method for shipping ROI requests. For example, you might want to enter FedEx Overnight or UPS Ground. Descriptions are optional. You can change them at any time.

Save

Saves the new or modified delivery method.

Cancel

Cancels the delivery method you were adding or any unsaved changes you made to an existing delivery method.
