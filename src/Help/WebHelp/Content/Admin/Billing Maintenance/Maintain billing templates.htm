Maintain billing templates

To add, modify, or delete billing templates for ROI requests, do the following:

  1. Access the Billing Templates form.
  2. Follow the instructions below for the task you want to perform.

To add a new billing template

  1. In the main content pane, click Create New.
  2. In the detail pane, enter a name for the billing template and select one or more fee types to associate with the billing template. The fee types you need must already be defined.
  3. Click Save.

To modify an existing billing template

  1. In the main content pane, select the billing template you want to modify.
  2. In the detail pane, change the name for the billing template and/or the fee types to associate with it.
  3. Click Save.

To delete a billing template

Note: You cannot delete billing templates that are associated with requestor types.

  1. In the main content pane, select the billing template you want to delete.
  2. Click Delete.
  3. In the confirmation message, click Delete.
