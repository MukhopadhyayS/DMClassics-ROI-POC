Billing Maintenance

About Billing Maintenance

The Billing Maintenance function provides ROI administrators the ability to define options for:

  • Billing tiers, fee types, and related information used for determining invoice amounts for ROI requests.
  • Delivery methods, payment methods, and shipping costs.

These features are accessed by selecting Billing Maintenance on the left pane of the Administration tab.

Sequential tasks

Because certain items must exist before you can define others, you must define the following items in the order they appear on the Administration tab. The order is:

  • Media Types
  • Billing Tiers
  • Fee Types
  • Billing Templates
  • Requestor Types

You can define the following items in any order:

  • Page Weight
  • Delivery Methods
  • Payment Methods
  • Invoice Due Days
  • Billing Location
