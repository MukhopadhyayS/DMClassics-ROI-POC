Maintain adjustments

This task enables authorized users to add, edit, or remove adjustments. Adjustments are used to credit the requestor's account, for example, to offer the requestor a discount.

To add an adjustment

Do this... In this form, dialog box, or screen
1. Find a requestor.

Find Requestor form
2. Select Requestors > Account Management.

 
3. Click Add Adjustment.

Account Management form
4. Enter the adjustment details

Adjustments dialog box.
5. Click Save.

 

To edit an adjustment

You can only edit applied adjustments.

Do this... In this form, dialog box, or screen
1. Find a requestor.

Find Requestor form
2. Select Requestors > Account Management.

 
3. Open an invoice to edit.

Account Management form
4. Click the red to delete the adjustment.

Adjustments dialog box
5. In the Apply field, type the correct adjustment amount.
Adjustments dialog box
6. Click Save.
Adjustments dialog box


To remove an adjustment

You can only remove unapplied adjustments.

Do this... In this form, dialog box, or screen
1. Find a requestor.

Find Requestor form
2. Select Requestors > Account Management.

 
3. Open the unapplied adjustment to remove.

Account Management form
4. In the Apply field, type the correct adjustment amount.

Adjustments dialog box
5. Click Remove Adjustment.

Adjustments dialog box.
