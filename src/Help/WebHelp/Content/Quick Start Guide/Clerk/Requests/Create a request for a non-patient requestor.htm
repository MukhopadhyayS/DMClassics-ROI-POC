Create a request for a non-patient requestor

This task assumes you want to create a request for a non-patient requestor. Non-patient requestors (such as insurance companies, attorneys, or other entities) often request records for more than one patient at a time. You can select one or more patients to include on the request.

This task describes how to create the request from the Patients tab.
