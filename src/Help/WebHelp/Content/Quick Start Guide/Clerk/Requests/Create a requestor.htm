Create a requestor

The system permits you to create patient and non-patient requestors and prompts for information based on the requestor type you specify. More...

You can create requestors by clicking the New Requestor button from these screens:

  • The Requestors > Find Requestors form (this method is described below)
  • The Select Requestor (or Change Requestor) dialog box

All methods allow you to enter the same data. However, the system enables or disables data entry fields dynamically, depending on the Requestor Type setting. Basically, patient and non-patient fields are mutually exclusive.

Note: Refer to this online help topic for complete details:
Create a requestor

To create a patient requestor

Do this... In this form, dialog box, or screen
1. On the Requestor tab, set Requestor Type to Patient.

 
2. Enter the search criteria.

 
3. Click New Requestor.

 
4. Select the patient record and click Select Patient as Requestor.

Select a Patient to Create New Patient Requestor dialog box
5. Enter requestor information.

Requestor Information form
6. Click Save.

 

To create a non-patient requestor

Do this... In this form, dialog box, or screen
1. On the Requestor tab, set Requestor Type to All Requestor Types or to a non-patient type.

 
2. Click New Requestor.

 
3. Enable the prepayment and/or certification flags, when appropriate.

Requestor Information form
4. Perform one of the following:
• If you are creating a patient requestor, enter any missing information.
• If you are creating a non-patient requestor, select the type of requestor and enter a name.

 
5. Click Save.
