Find a request

This task enables you to search for and select a specific request. The Find function supports a wide variety of search criteria that can be used in any combination, enabling you to narrow results to a concise list of requests.

For example, you can search by:

  • Request status (to list all requests with a specific status)
  • Patient identifiers (to list all requests for a specific patient, date of birth, SSN, encounter number, MRN, or EPN)
  • Requestor identifiers (to list all requests for a specific requestor or requestor type)
  • Balance amounts (to list all requests with a balance that is less than, equal to, or more than a specific dollar amount)
  • Facility, received date, invoice number, request ID, or completed date

Note: Refer to this online help topic for complete details:
Find a request

Do this... In this form, dialog box, or screen
1. Select Requests > Find / Create Request.

 
2. Enter the search criteria.

Find/Create Request form
3. Click Find Request.

 
4. In the detail pane, select the request you want and click View/Edit Request.
