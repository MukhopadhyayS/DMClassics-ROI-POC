Create a request from the Requestor's tab

This task assumes you want to create a request for a specific requestor (either patient or non-patient).

Note: Refer to this online help topic for complete details:
Create a request for a non-patient requestor

For a non-patient requestor

Do this... In this form, dialog box, or screen
1. Perform a requestor search.

 
2. Select the requestor to associate with the new request and click Create Request.

Find Requestor form
3. Select a Request Reason and Receipt Date.

Request Information form
4. Click Save.

 

For a patient requestor

Do this... In this form, dialog box, or screen
1. Perform a requestor search.

 
2. Select the requestor to associate with the new request and click Create Request.

Find Requestor form
3. Do one of the following:
• Select the patient to assign as the requestor and click Continue with Selected Patient(s).
• Do not select any patient and click Continue with No Patient.

Create New Request with Patient Requestor dialog box
4. Select a Request Reason and Receipt Date.

Request Information form
5. Click Save.
