Create a request with patient requesting

This task assumes that you want to create a request for a patient who is requesting his or her own records.

Note: Refer to this online help topic for complete details:
Create a request with patient requesting

Do this... In this form, dialog box, or screen
1. Perform a patient search.

 
2. Select the patient who is requesting records and click Patient Requesting.

Find Patient form
3. If the system finds one or more requestors that potentially match the patient, the system displays all found names in the grid. Select the matching entry from the list and click Select Requestor. The system proceeds to the Requests > Request Information form.
Create a New Request with the Selected Patient Requesting dialog box
4. If no match is found, the system displays and highlights the patient's name in the grid. Click Select Patient as Requestor.
 
5. Select Pre-payment Required and/or Certification Letter Required (when appropriate), review the patient's information and correct it, if needed, and then click Save New Requestor.
Save New Patient dialog box
6. Select a Request Reason and a Receipt Date

Request Information form
7. Click Save.
