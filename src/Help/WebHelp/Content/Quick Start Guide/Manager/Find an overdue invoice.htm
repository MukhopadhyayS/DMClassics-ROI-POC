Find an overdue invoice

This task allows you to search for and view overdue invoices by billing location, requestor type, number of days overdue and requestor name.

Do this... In this form, dialog box, or screen
1. Select Overdue Invoices > Find Overdue Invoices.

 

2. Enter the search criteria.

Find Overdue Invoices form

3. Click Search.

Find Overdue Invoices form
4. View or output the highlighted invoice(s).
