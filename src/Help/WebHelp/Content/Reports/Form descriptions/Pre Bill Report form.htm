Pre-Bill Report form

This report displays information related to all active pre-bills as of the current date.

Note: Inactive pre-bills are not shown in the report results.

To open the form

  1. Select the Reports tab.
  2. In the Report Name box, select Pre-Bill Report.  When the form first displays, the main content pane is blank. To view a report, select the report options in the left pane and click Run Report.

Item descriptions

Billing Location

Specifies the billing location on which you want to report.

You can select or deselect one or more locations, click Select All to select all locations, or click Clear All to clear all previous selections.

Requestor Type

Specifies the requestor type for the report. Examples are Physician, Attorney, or Patient.

You can select or deselect one or more requestor types, click Select All to select all requestor types, or click Select All to clear all previous selections.

Requestor Name

Specifies the name of the person making the request.

Request Status

Specifies the request status or statuses on which you want to report. The available statuses are:

  • Logged
  • Completed
  • Denied
  • Canceled
  • Pended
  • Pre-Billed

You can select or deselect one or more request statuses, click Select All to select all request statuses, or click Clear All to clear all previous selections.

Aging

The number of days since the pre-bill was generated.  If the aging criteria is set at 0 to 0, the report does not generate.

Balance

Specifies the balance amount using At Least, At Most or Equal criteria

Reset

Clears the pane of any selection criteria and displays only the drop-down field with the list of reports.

Run Report

Generates the report based on the specified report criteria and date range.

Main content pane

View By

Provides filtering options to change the information displayed in the report. You do not need to re-run the report to see the new view. The system automatically changes the view when you make a selection. Options include:

  • Summary - Requestor Type
  • Detailed - Requestor Name
  • Detailed - Request ID

Print Preview

Displays the Print Preview dialog box so you can preview the report before printing it.

Print

Displays the Print dialog box so you can select a printer and printing options.

Export

Displays the Export Report dialog box so you can select a file name, file format, and location for saving the report. The available file formats are PDF, XLS, RTF, HTML, and CSV.
