Turn Around Times form

The Turn Around Times form allows you to generate the Turn Around Times Report for processing requests as they have changed from one status to another.  This report shows request turnaround times as measured in days.

Since the status can be changed several times, use the earliest and latest status change dates for calculating the average turnaround time. When the report is run for such requests whose status is often changed back to a previous state, the system calculates and displays higher average turnaround time value.  This assists in analyzing workflow process.

You can select criteria for the report and specify the date range. Security rights are required.

To open the form

  1. Select the Reports tab.
  2. In the Report Name box, select Turn Around Times.  

When the form first displays, the main content pane is blank. To view a report, select the report options in the left pane and click Run Report.

Item descriptions

Left pane

Facility

Specifies the facility or facilities on which you want to report.

You can select or deselect one or more facilities, click Select All to select all facilities, or click Clear All to clear all previous selections.

From Status

Specifies the starting status. Options are:

  • Auth Request Received
  • Auth Request Denied
  • Received (receipt date)
  • Logged
  • Pended
  • Canceled
  • Denied
  • Pre-billed
  • Completed

To Status

Specifies the ending status. Options are:

  • Logged
  • Pended
  • Canceled
  • Denied
  • Pre-bill
  • Completed

Requestor Type

Specifies the requestor type for the report. Examples are Physician, Attorney, or Patient.

You can select or deselect one or more requestor types, click Select All to select all requestor types, or click Clear All to clear all previous selections.

Date Range

Specifies the time span for the report. You can select from the following options:

  • Today's Date
  • Custom Date.  If selected, enter the applicable dates in the State Date and End Date fields.
  • Current Week
  • Current Month
  • Previous Month
  • 3 Months Prior
  • 12 Months Period
  • Year to Date
  • Previous Year

Reset All

Clears the pane of any selection criteria and displays only the drop-down field with the list of reports.

Run Report

Generates the report based on the specified report criteria and date range.

Main content pane

View By

Provides filtering options to change the information displayed in the report. You do not need to re-run the report to see the new view. The system automatically changes the view when you make a selection.

Print Preview

Displays the Print Preview dialog box so you can preview the report before printing it.

Print

Displays the Print dialog box so you can select a printer and printing options.

Export

Displays the Export Report dialog box so you can select a file name, file format, and location for saving the report. The available file formats are PDF, XLS, RTF, HTML, and CSV.
