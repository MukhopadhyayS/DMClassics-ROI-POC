Accounting of Disclosure Report form

The Accounting of Disclosure Report is provided at the request of the patient. It reveals the information that was disclosed, to whom it was disclosed, and for what purpose it was disclosed.

As of 16.2, the Accounting of Disclosures report will also include legacy ROI requests such as those created in version 15.0 and earlier.

Note: Patients can have one free report annually. After that, any charges are at the discretion of the facility.

To open the form

  1. Select the Reports tab.
  2. In the Report Name box, select Accounting of Disclosure.  

When the form first displays, the main content pane is blank. To view a report, select the report options in the left pane and click Run Report.

Item descriptions

Left pane

Select a Patient

Displays the Find Patient dialog box so you can enter search criteria to search for and select the patient on which you want to report.

Patient Information box

Displays key patient information returned from the Find Patient dialog box (which is accessed by clicking the Select a Patient button on the Accounting of Disclosure form). The information returned is the patient name, date of birth, gender, SSN, facility, MRN, and EPN (Enterprise Person Number). If SSN masking is in effect, asterisks will display instead of the SSN.

Request Type

Specifies the request type for the report. You can select TPO (related to treatment, payment, or healthcare operations), Non-TPO, or Both.

Date Range

Specifies the time span for the report. You can select from the following options:

  • Today's Date
  • Custom Date.  If selected, enter the applicable dates in the State Date and End Date fields.
  • Current Week
  • Current Month
  • Previous Month
  • 3 Months Prior
  • 12 Months Period
  • Year to Date
  • Previous Year

Reset

Clears the pane of any selection criteria and displays only the drop-down field with the list of reports.

Run Report

Generates the report based on the specified report criteria and date range.

Main content pane

View By

Provides filtering options to change the information displayed in the report. You do not need to re-run the report to see the new view. The system automatically changes the view when you make a selection. Options include:

  • Disclosure Date
  • Disclosure Date, Documents Released

Print Preview

Displays the Print Preview dialog box so you can preview the report before printing it.

Print

Displays your standard Print dialog box so you can select a printer and printing options.

Export

Displays the Export Report dialog box so you can select a file name, file format, and location for saving the report. The available file formats are PDF, XLS, RTF, HTML, and CSV.
