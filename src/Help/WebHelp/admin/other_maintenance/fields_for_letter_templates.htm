Fields for letter templates

The following fields can be included in letter templates you choose to create or modify.

You must know how to use Microsoft® Word and the merge field function. Merge instructions are beyond the scope of this document. See the Microsoft documentation if you need help.

Note: A "Yes" in the Grid field column indicates that this field, if included in the template, must be located in the grid section on the template. Otherwise, the template will not work properly.
