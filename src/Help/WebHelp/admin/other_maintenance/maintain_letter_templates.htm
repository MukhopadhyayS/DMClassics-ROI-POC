Maintain letter templates

To create, modify, upload, or delete letter templates, or to change their attributes, do the following:  

  1. Access the Letter Templates form.
  2. Follow the instructions below for the task you want to perform.

Note: Users that are logged in must log out and log back in for any newly uploaded or deleted templates to be reflected at their station.

To create a new letter template

The easiest way to create a new letter template is to download and modify an existing template, upload the modified template, and change its attributes to suit its new purpose.

Note: You can also create templates from scratch using Microsoft Word. Template files must be saved in .docx format and can use specific fields from the ROI database. You must then upload the file to make it available to users.

To create a new letter template using an existing template as its basis, do the following:

  1. Modify an existing template and save it with a new name. Follow the instructions below for the task, To modify a letter template.
  2. Upload the template. Follow the instructions below for the task, To upload a letter template.
  3. Change the attributes of the template to something appropriate for its new purpose. Follow the instructions below for the task, To change the attributes of a letter template.

To modify a letter template

Use this task to modify the template itself. If you need to modify the attributes of a template, such as to change its name or description, follow the instructions for the task, To change the attributes of a letter template.

To modify a letter template, do the following:

  1. In the main content pane, select the template you want to modify.  
  2. Click Download.

The system opens the Save As dialog box.

  1. Browse to the location of the file you want to modify and select it.

The system populates the File Name box with the name of the file you selected.

  1. If you do not want to overwrite the existing template, change the file name. You must keep the .docx extension.
  2. Click Save.

The system closes the Save As dialog box.

  1. Using Windows Explorer, go to the file and open it in Microsoft Word.
  1. Edit the file as needed. You can edit boilerplate text passages and can modify the database fields that are used by the template. To modify the fields included on the template, you must know how to use the Microsoft Word merge function, and you must know valid mapped database fields. See Fields for letter templates for a list of valid fields.
  2. From the Windows menu bar, select File / Save.
  1. Close the file.

To upload a letter template

  1. In the main content pane, click Create New.
  2. In the detail pane, select the Letter Type.

Note: Cover Page, Invoice, and Pre-Bill templates should be associated with letter types of the same name. Other templates should be associated with the letter type, Other.  Requestor Letter templates should be associated with the letter type, Statement.

  1. If you selected Cover Letter, Invoice, Pre-Bill or Requestor Letter, and you want this template to be the default template for the letter type, select Default. Otherwise, leave the Default field blank.
  2. Enter a name for the template and, optionally, a description.
  3. Click Browse.

The system opens the Open dialog box.

  1. Go to the template location, select the template you want to upload, and click Open.

The system populates the Upload File field on the Letter Templates form with the path and file name.

  1. Click Save.

The system uploads the file.

To change the attributes of a letter template

  1. In the main content pane, select the letter template whose attributes you want to change.
  2. In the detail pane, change the name, description, or other information as needed.
  3. Click Save.

To delete a letter template

  1. In the main content pane, select the letter template you want to delete.
  2. Click Delete.
  3. In the confirmation message, click Delete.
