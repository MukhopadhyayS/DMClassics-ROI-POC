Maintain meaningful use documents

ROI Module administrators can flag document types to include in Meaningful Use turnaround time statistics (turnaround time refers to the number of hours or days it takes to fulfill an ROI request). The document types to flag are dictated by the source clinical system. Only Clinical Summaries (CCRs/CCDs), Discharge Summaries, and Discharge Instructions are included in Meaningful Use statistics.

To designate document types as Meaningful Use documents for purposes of the ROI Module, do the following:

  1. Select Administration > Other Maintenance > Document Types.
  2. On the Document Types form, select the Code Set you want to maintain.
  3. In the Document Types grid, go to the MU Document Type column and identify any document type that qualifies as one of the following:
• Clinical Summary
• Discharge Summary
• Discharge Instructions

Click Save.

Note: Code sets are maintained in the base OneContent system. For information, see OneContent online help.
