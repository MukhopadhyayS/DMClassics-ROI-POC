Configure Location form

The Configure Location function allows you to specify a default country to associate with your application enterprise installation. This provides an additional reference point in the ROI Module application for several areas including billing, and patient and requestor maintenance. You must be an ROI Administrator to configure this option.

Note: The system audits the country code configuration event.

To open the form

Select Administration > Other Maintenance > Configure Location.

Item descriptions

Default Country

Indicates the country selected as the default value for address information.

Save

Saves the new or modified default country.

Cancel

Cancels the default country value you were adding or any unsaved changes you made to an existing value.
