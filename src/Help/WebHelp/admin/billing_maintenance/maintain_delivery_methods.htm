Maintain delivery methods

To add, modify, or delete delivery methods for shipping ROI requests, do the following:

  1. Access the Delivery Methods form.
  2. Follow the instructions below for the task you want to perform.

To add a new delivery method

  1. In the main content pane, click Create New.
  2. In the detail pane, select Default if you want the delivery method you are adding to be the default delivery method.
  3. Enter a unique name for the delivery method.
  4. Optionally, enter a description and the URL for the delivery method website.
  5. Click Save.

To modify an existing delivery method

  1. In the main content pane, select the delivery method you want to modify.
  2. In the detail pane, select or deselect Default as needed.
  3. Change the name and/or other information as needed.
  4. Click Save.

To delete a delivery method

  1. In the main content pane, select the delivery method you want to delete.
  2. Click Delete.
  3. In the confirmation message, click Delete.
