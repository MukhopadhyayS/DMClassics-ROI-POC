Page Weight form

The Page Weight form allows you to specify the number of pages it takes to equal an ounce. This value is used for calculating shipping charges for ROI requests. Each time a request is opened, the system retrieves this figure and re-calculates the shipping weight.

Keeping this value current is an ongoing, manual exercise.

To open the form

Select the Administration tab > Billing Maintenance > Page Weight.

Item descriptions

pages = 1 ounce

Specifies the number of pages it takes to equal an ounce.

The number must be a whole number greater than zero but less than 100.

Save

Saves the number of pages per ounce that you entered.

Cancel

Cancels any unsaved changes you made to the number of pages per ounce.
