Media Types form

The Media Types form allows you to view, add, modify, or delete media types such as hardcopy, microfilm, and electronic media.

Media types are associated with billing tiers and are used for calculating the cost of ROI requests for billing purposes.

To open the form

Select the Administration tab > Billing Maintenance > Media Types.

Item descriptions

Main Content Pane

Media Types grid

Displays the Media Types and associated information currently defined in the system. The grid title is preceded by the number of existing media types.

The Media Types grid contains the following columns:

  • Icons that indicate the type, or source, of the data:
    The system icon indicates media types that are system defined. You cannot delete system-defined media types. You can edit the description.
    The user icon indicates media types that are sample data or user created. You can delete the media type or change the description.
  • Media Name
    Names of the media types currently defined in the system.
  • Description
    Descriptions of the media types (optional).

Create New

Allows you to enter the information needed to add a new media type.

Delete

Deletes the selected media type.

Note: You cannot delete the names of system-defined media types. They are indicated by the system icon in the Media Types grid. You also cannot delete media types that are associated with billing tiers.

Detail Pane

Media Name

Specifies the name for the media type. Media type names must be unique. They are not case sensitive.

Note: You cannot change the names of system-defined media types. They are indicated by the system icon in the Media Types grid. You also cannot change the names of media types that are associated with billing tiers.

Description

Provides the description for a media type. You can change the description at any time. Descriptions are optional.

Save

Saves the new or modified media type.

Cancel

Cancels the media type you were adding or any unsaved changes you made to an existing media type.
