Maintain media types

To add, modify, or delete media types for processing ROI requests, do the following:

  1. Access the Media Types form.
  2. Follow the instructions below for the task you want to perform.

Note: You cannot change or delete system-defined media types or media type names that are associated with billing tiers. System-defined media types are indicated by theicon.

To add a new media type

  1. In the main content pane, click Create New.
  2. In the detail pane, enter a name and, optionally, a description for the new media type.
  3. Click Save.

To modify an existing media type

  1. In the main content pane, select the media type you want to modify.
  2. In the detail pane, change the information as needed.
  3. Click Save.

To delete a media type

  1. In the main content pane, select the media type you want to delete.
  2. Click Delete.
  3. In the confirmation message, click Delete.
