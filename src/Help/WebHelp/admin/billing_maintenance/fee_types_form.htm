Fee Types form

The Fee Types form allows you to view, add, modify, or delete miscellaneous expenses that can be added to an ROI request. These can include items such as a research or certification fee.

Fee types can be associated with one or more billing templates.

To open the form

Select the Administration tab > Billing Maintenance > Fee Types.

Item descriptions

Main Content Pane

Fee Types grid

Displays the fee types and associated information currently defined in the system. The grid title is preceded by the number of existing fee types.

The Fee Types grid contains the following columns:

  • Fee Name
    Names of the fee types currently defined in the system.
  • Amount
    Charges associated with the fee types.
  • Sales Tax
    Indicates if sales tax is applied to this item by default.
  • Description
    Descriptions of the fee types (optional).

Create New

Allows you to enter the information needed to add a new fee type.

Delete

Deletes the selected fee type.

Note: You cannot delete fee types that are associated with billing templates.

Detail Pane

Fee Name

Specifies the name for the fee type. Fee type names must be unique. They are not case sensitive.

You can change a fee type name at any time. If the fee type is associated with a billing template, the template automatically reflects the new name going forward.

Amount

Specifies the amount for a fee type.

You can change the amount for a fee type at any time. If the fee type is associated with a billing template, the template automatically reflects the new amount going forward.

Sales Tax

Indicates if sales tax is applied to this item by default.  Check to select this option.  The default is unchecked.

Description

Provides the description for the fee type. You can change the description at any time. Descriptions are optional.

Save

Saves the new or modified fee type.

Cancel

Cancels the fee type you were adding or any unsaved changes you made to an existing fee type.
