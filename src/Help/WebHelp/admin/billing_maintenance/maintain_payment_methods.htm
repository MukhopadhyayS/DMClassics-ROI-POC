Maintain payment methods

To add, modify, or delete payment methods for processing ROI requests, do the following:

  1. Access the Payment Methods form.
  2. Follow the instructions below for the task you want to perform.

To add a new payment method

  1. In the main content pane, click Create New.
  2. In the detail pane, enter the name for the payment method you want to add.
  3. Optionally, enter a description for the payment method.
  4. If you want to display a Note in the Payments dialog box, select Display for the Form Field option.
  5. Click Save.

To modify an existing payment method

  1. In the main content pane, select the payment method you want to modify.
  2. In the detail pane, change the information as needed.
  3. Click Save.

To delete a payment method

  1. In the main content pane, select the payment method you want to delete.
  2. Click Delete.
  3. In the confirmation message, click Delete.
