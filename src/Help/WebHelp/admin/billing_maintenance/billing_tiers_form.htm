Billing Tiers form

The Billing Tiers form allows you to view, add, modify, or delete billing tiers.   

Billing tiers are used for calculating the charges for ROI requests. They consist of a base charge plus any per-page charges that you specify. Each billing tier is associated with a media type such as paper, microfilm, or electronic media.

To open the form

Select the Administration tab > Billing Maintenance > Billing Tiers.

Item descriptions

Main Content Pane

Billing Tiers grid

Displays the billing tiers and associated information currently defined in the system. The grid title is preceded by the number of existing billing tiers.

The Billing Tiers grid contains the following columns:

  • Icons that indicate the type, or source, of the billing tier data:

    The system icon indicates billing tiers that are system defined. You cannot delete system-defined billing tiers or change the media type associated with them. You can edit the description and rates per page.

    The user icon indicates that the data is sample data or is user created. You can delete the billing tier or change the associated information.
  • Tier Name
    Names of the billing tiers currently defined in the system.
  • Media Type
    Names of the media types associated with the billing tiers.
  • Description
    Descriptions of the billing tiers (optional).

Create New

Allows you to enter the information needed to add a new billing tier.

To create a billing tier that is associated with a specific media type, the media type must already be defined in the system.

Delete

Deletes the selected billing tier.

Note: You cannot delete system-defined billing tiers. They are identified by the system iconin the Billing Tiers grid.

Detail Pane

Tier Name

Specifies the name for a billing tier. Billing tier names must be unique. They are not case sensitive.

Note: You cannot change the names of system-defined billing tiers. They are identified by the system iconin the Billing Tiers grid.

Media Type

Specifies the media type associated with the billing tier.

Note: You cannot change the media type associated with system-defined billing tiers. They are identified by the system iconin the Billing Tiers grid.

Description

Provides the description for a billing tier. You can change the description at any time. Descriptions are optional.

Sales Tax

Indicates if sales tax is applied to this item by default. Check to select this option.  The default is unchecked.

Rates section

Specifies the base charge and page-level rate tiers (per-page charges) for the billing tier. You cannot delete the base charge or page-level rate tiers that are delivered with the system, but you can set the amounts to $0.00.

Rate information consists of the following items:

  • Base Charge: $ <amount>
    A flat rate to which any per-page charges are added.
  • Page <number> through page <number> =$ <amount> per page
    The page range and per-page charge for the page range you specify. For example, if you enter that Page 1 through page 10 = $ 0.50 per page, the charge for pages 1 through 10 would be $5.00. You can add as many page-level rate tiers as you need. The starting page number for each new page-level rate tier must be the ending page number of the previous tier plus one.
  • Amount for every page thereafter = $ <amount> per page
    The per-page charge for all pages after the page-level rate tier or tiers.
  • Add a new rate tier
    Clicking this button displays the fields needed to add a new page-level rate tier. Any page ranges you add must follow the previous page range sequentially.  You can delete any page-level rate tiers that you add by clicking the delete page tier icon   next to the item.

Save

Saves the new or modified billing tier.

Cancel

Cancels the billing tier you were adding or any unsaved changes you made to an existing billing tier.
