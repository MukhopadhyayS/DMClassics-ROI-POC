Maintain request status reasons

To add, modify, or delete the reasons users can select when adding or changing the status of an ROI request, do the following:

  1. Access the Request Status Reasons form.
  2. Follow the instructions below for the task you want to perform.

To add a new request status reason

  1. In the main content pane, click Create New.
  2. In the detail pane, select the Request Status for which you want to define a new reason.
  3. Enter the Reason Name and Display Text for the request status reason.
  4. Click Save.

To modify an existing request status reason

  1. In the main content pane, select the request status reason you want to modify.
  2. In the detail pane, change the information as needed.
  3. Click Save.

To delete a request status reason

Note: Note: Some reasons are created automatically by the system and cannot be deleted.

  1. In the main content pane, select the request status reason you want to delete.
  2. Click Delete.
  3. In the confirmation message, click Delete.
