Release & Invoice form

The Release & Invoice form enables an authorized user to view, enter, or change billing charges for a currently selected request. If you do not have required permissions, all fields are read-only.

The form also enables an authorized user to generate pre-bill letters and invoices, and to release the requested information.  

You can access the Release & Invoice form only after at least one document has been selected for release on the Patient Information form.

To open the form

Create a request from the Patients, Requestors, or Requests tab.

Item descriptions

Requestor information box

Identifies the requestor type, the requestor's name and demographic information, and the requirements for pre-payment and certification letters (if any). Also displays the request ID, current status, and receipt date and time.

Billing & Shipping for Release

This section displays information and controls for document charges (that is, billing tiers for documents selected for release) and Fee Charges.

Note: If the request has a status of Cancelled or Denied, all information is read-only.

Unbillable Request

Indicates that the charges for the request are unbillable.  Non-billable charges display as a 100% discount under the Totals & Balances Due group box.

Billing Location

Identifies the billing location associated with this request.

Document Charges group box

Copies

The number of copies of the released documents to bill for.

Allows you to enter the number of copies.

Defaults to 1. Valid range is 1 to 99.

Can be modified.

Pages

The number of pages for the corresponding billing tier.

System-generated and read-only.

Billing Tier

The name of the billing tier.

For an OneContent document, always displays 'Electronic' in the first row.

For a non-OneContent document, displays the name of the tier associated with the requestor type.

Amount

The amount of the charges for this line item.

Total

The total number of pages being released and the total dollar charge for the documents.

Automatically calculated based on copies, pages, and the base charge for each billing tier.

Billing Tier Total (pages and billing tier charges)

Displays the total number of pages to bill for and the total billing tier charges.

Fee Charges group box

Billing Template

Displays the name of the billing template on which to base fee charges.

If a default billing template has been configured for the current requestor type, defaults to that template.

Otherwise, defaults to Please Select, lists all billing templates assigned to the current requestor type, and allows you to select one.

Fee Types

Displays all the fee types currently configured for the template specified in the Billing Type box, and displays a total of the charges generated by the fees.

Defaults to the amount configured on the template.

Allows you to modify the amount to charge.

Custom Fees grid and Create Custom Fee button

Custom Fees

Displays the name of any custom fee created for this request.

Allows you to modify the name.

Amount

Displays any current charge for the associated custom fee.

Allows you to modify the amount.

Format is dollars and cents.

Delete icon

When clicked, displays a confirmation prompt to delete the Custom Fee line item.

Custom Fees Total

Displays the total charges for all custom fees.

Read-only.

Create Custom Fee button

Allows you to create a new Custom Fee line item.

Output tab

The Output tab contains information and controls for setting the release method and shipping information.

Note: After the system successfully releases documents, all information on this tab is cleared or reset to its initial state, except for the shipping information, which retains the address of the last release (and can be changed).

Output Method for this Release

Important: You must select an output method to continue with the release process.

Allows you to select the method to use for releasing the documents. Options are based on your user permissions.

Initially, no option is selected. After you select an option, it remains selected until you change it or you successfully release the documents. If you cancel the release, the selection is cleared.

Options are:

  • Print
    Generates a print job when documents are released.
  • Save as File
    Generates output to a PDF file in a file location that you specify when documents are released.
  • Fax
    Generates output to a fax job using a fax number you specify, when documents are released.
  • Email
    Generates output to an email job when documents are released.
  • Disc
    Generates output to disc (such as CD or DVD) when documents are released.

Will the Release be Shipped?

Allows you to specify whether you will manually ship the released documents.  

Enabled only when Output Method is either Print or Save as File.

Initially, no option is selected. After you select an option, it remains selected until you change it or you successfully release the documents. If you cancel the release, the selection is cleared.

When set to Yes, enables all fields in the Shipping Information group box.

Non-Printable Attachment Queue

Lists all "Save As File" queues. This field is enabled when the request has non-printable attachments added to it and the output type is either Print or Fax. The application submits two output tasks for release.  Depending upon the contents, there is either a Print job, a Save as File job, or both.

Defaults to Please Select.

Populate Shipping Address for the Requestor's

Specifies the address to send the released documents to. Options are Main Address or Alternate Address.

Controls are enabled only if you set Will the Release be Shipped? to Yes.

Allows you to select an address to use and edit address information as needed.

Changes apply to only this request (edits do not affect the requestor's main or alternate address on record). If you enter changes, the system:

  • Disables the address option buttons
  • Displays a prompt to confirm that you want to use the modified address

Reset Address

Discards any edits you made to address information, enables the Main Address or Alternate Address options, and selects Main Address by default.

Enabled only if you have edited the address information.

Shipping Method

Displays all shipping methods configured for your system.

Defaults to the shipping method specified as the default for your system (if one exists). Otherwise, defaults to Please Select.

Tracking Number

Allows you to enter the tracking number provided by the shipping agent.

Shipping Charges

Allows you to enter the shipping charges (in dollars) calculated by the shipping agent for this shipment of documents.

Shipping Weight

Displays only if Output Method is set to Print.

Displays the weight for the printed pages using the shipping weight formula configured for your system.

Go To URL

Displays only if your administrator has specified a URL for the shipping provider.

Provides a link to the shipping provider's Web site to calculate shipping charges.

Sales Tax tab

The Sales Tax tab displays information about sales tax for various elements (for example, billing tier type, custom fee, etc.)

Apply Tax

Selected if sales tax is enabled by default for the Billing Location.

The user can override this setting.

Sales Tax Percentage

Displays the sales tax percentage if configured in Billing Location.

If there is no sales tax percentage associated with the billing location, no percentage displays.

If there is a sales tax percentage associated with the billing location, the percentage displays even if Apply Tax is not selected.

Select/Deselect Tax

Indicates if sales tax is included for an item. Unchecked is the default.

Charge Type

The element to which sales tax applies.

Charge

The amount of the charge.

Sales Tax

The amount of the sales tax calculated as the charge times the tax percentage.

Totals Line

Sum total of the selected items for both charge and sales tax items.

Reason for override

An explanation as to why the tax was or was not applied.

Date/Reasons for tax change

Lists all dates for tax overrides and the corresponding reason. When a reason is entered, today's date is auto populate in the reason for change grid.

Totals & Balance Due group box

Displays a summary of payments, adjustments, and discounts for the current request.

Charge History

Launches the Charge History dialog box.

Previously Released Charges

Displays the sum cost of all previous releases for this request.

Applied Payments & Adjustments

Displays the sum total for adjustments and payments for this request.

Unbillable Amount

Displays the total amount for all unbillable requests.

Request Balance

Displays the total balance for this request including all previous releases and any applied payments/adjustments.  

Pending Release Charges

Displays the sum of the pending release cost. Also shows the total tax for the request.

Request Charges

Displays the sum of both the Pending Release and Previously Released Costs.

Unapplied Payments & Adjustments

Displays all payments and adjustments that have posted to the requestor's account but not applied to any open invoice.

Buttons

Account Management

Opens the Account Management form for the current requestor.

Add Payment

Opens the Payments dialog box. Only invoices associated with this request are shown.

Add Adjustment

Opens the Adjustments dialog box. Only invoices associated with to this request are shown.

Save

Saves any changes made to billing, shipping, adjustment, or payment information for this request.

Enabled only if you have changed any information.

Revert

Reverts all information to the last saved state.

Enabled only if you have changed any information.

Create Pre-Bill

Allows you to generate a pre-bill letter for the request.

Enabled only if you have saved or reverted any changes you have made on either tab.

Disabled if documents have been released, the request status is Denied or Cancelled, or the request is unbillable.

Release & Invoice

Allows you to release the selected documents and generate an invoice.

Enabled only after you have:

  • Saved or reverted any changes you have made, AND
  • Specified an output method on the Output tab.

If self-pay encounter documents are being released, the systems displays an information message.  Click Continue to release the documents.  Click Cancel to stop the release and invoice process.

Re-Release

Allows you to release the selected documents to the requestor.

Enabled only after you have:

  • Saved or reverted any changes you have made on either tab AND
  • Selected at least one previously released document to release (cannot include any unreleased documents) AND
  • Specified and saved an output method on the Output tab.
