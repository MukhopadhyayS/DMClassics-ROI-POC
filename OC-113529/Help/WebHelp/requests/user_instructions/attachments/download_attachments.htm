Download attachments

This task enables you to electronically download files from an external clinical system. The files are added as attachments to a patient record. You can add attachments for both OneContent and non-OneContent patients. Currently, only Clinical Summaries can be downloaded.

  1. With a request open, go to the Requests > Patient Information form.
  2. If more than one patient is on the request, select the patient you want.
  3. Click Add Attachment. The system displays the Create New Attachment dialog box.
  1. At Attachment Location, select Query Clinical System. The system displays a list of encounters for this patient, beginning with the most recent Admit Date.  

Note: You can also acquire files from a local file system. Select Local File and enter information as needed. Then click Save. For more details, see Add attachments to a patient record.  

  1. Select one or more encounters and click Add Selected. The system attaches any Clinical Summaries it finds to the patient's record and lists them in the patient's document tree like any other patient document.

Note: For each document downloaded from a Paragon system, the ROI Module generates a unique reference number and sends it to the Paragon system. This ensures traceability for each download. The number is formatted as MPFROIReqID-YYYYMMDDHHMMSSMMM.
