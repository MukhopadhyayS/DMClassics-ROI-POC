Print, Fax, and Save as File buttons

  • Print

Closes the preview dialog box and launches the print output dialog box.

Displays only if you have print/fax permission.

  • Fax

Closes the preview dialog box and launches the fax output dialog box.

Displays only if you have print/fax permission.

  • Save as File

Closes the preview dialog box and launches the Save As File output dialog box.

Displays only if you have the Export to PDF permission.
