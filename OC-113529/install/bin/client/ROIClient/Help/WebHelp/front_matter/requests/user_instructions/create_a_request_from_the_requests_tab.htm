Create a request from the Requests tab

This task describes how to create a request from the Requests tab.

 

  1. Select Requests > Find / Create Request.

  1. If the Find/Create Request form currently displays patient information, the request you create will be associated with that patient. To create a request for a different patient, click Reset to clear the data.

  2. Click New Request. The system displays the Request Information form.

  1. At Request Reason, select a request reason. Then enter the date the request was received.

  2. Click Select Requestor and specify a requestor.

  3. Click Save & Continue. The request has been created. You can continue to build the request now or can return to it later.

To continue building the request, see Add a patient to a request.
