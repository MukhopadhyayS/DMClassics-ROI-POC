Configure notes

To add, modify, or delete standard notes that users can add to ROI letters such as cover letters and invoices, do the following:

  1. Access the Configure Notes form.

  2. Follow the instructions below for the task you want to perform.
     

To add a new note

  1. In the main content pane, click Create New.

  2. In the detail pane, enter a Name for the note and the associated Display Text.

  3. Click Save.

 

To modify an existing note

  1. In the main content pane, select the note you want to modify.

  2. In the detail pane, change the note Name and/or the associated Display Text.

  3. Click Save.

 

To delete a note

  1. In the main content pane, select the note you want to delete.

  2. Click Delete.

  3. In the confirmation message, click Delete.
