Configure headers, footers, and watermarks

To configure headers, footers, and watermarks that ROI users can add to release documents, do the following:

  1. On the menu bar, select Administration > Configure Output Settings.

The system displays the Configure Output Settings dialog box.

  1. Select the Headers, Footers and Watermarks tab.

  2. Follow the instructions below for the task you want to perform.

 

To configure watermarks

  1. On the Headers, Footers and Watermark tab, do one of the following for the Display Watermark item:

    • To provide users the option to display watermarks on release documents, select Yes.

    • To disallow the option to display watermarks, select No.

  2. If you select Yes, you also need to enter the watermark text you want to display in the Watermark Text box.

  3. Click Save.

 

To configure headers and footers

  1. On the Headers, Footers and Watermark tab, do one or both of the following for the Header and Footer options:

    • To provide users the option to display headers and/or footers on release documents, select Header and/or Footer.  

    • To disallow adding headers and/or footers, deselect one or both options.

  1. If you select either or both options, you also need to select the information you want to display in the headers and/or footers of the release documents.

If you select both options, you also need to select whether to include the information in the headers or in the footers. The same information cannot be displayed at both locations. The information items that can be displayed are:

    • Absolute page numbers (For each new document in a release, the page numbering restarts at 1 of x.)

    • Relative page numbers (Page numbering does not restart at 1 of x for each new document. Instead, each page number increments by 1, whether a new document is encountered or not.)

    • Patient name

    • Encounter number

    • MRN (medical record number)

  1. Click Save.

Note: Patient name, encounter, and MRN are treated as a unit and must be displayed at the same location (header or footer).
