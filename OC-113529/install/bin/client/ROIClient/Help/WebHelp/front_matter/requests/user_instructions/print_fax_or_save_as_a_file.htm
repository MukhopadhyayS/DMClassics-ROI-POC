Print, fax, or save as a file

This function enables you to print or fax letter output or save a letter as a PDF file.

Note: Not all preview dialog boxes contain all buttons described below. Instructions are general in nature.

At the Preview dialog box, do any of the following.

Click

To

Then

Print

Print the letter to any network printer connected to your workstation

At the Print dialog box, select a printer, enter other print options as desired,  and click OK to send the job to the print queue.

Fax

Fax the letter

At the Fax dialog box, make changes to the requestor's information as needed and click OK to send the job to the fax queue.

Save as File

Save the letter as a PDF file

At the Save as File dialog box, make any changes needed and click OK to send the job to the output queue.

Continue

Launch the dialog box for the output method (Print, Fax, or Save as File) specified on the Shipping tab on the Billing & Payment form

Note: The Continue button displays only if you are releasing documents.

Make any changes as needed and click OK to send the job to the appropriate queue.

Cancel

Close the preview window and return to the calling form
