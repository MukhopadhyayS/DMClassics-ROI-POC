Find a patient

This task enables you to search for and select a specific patient. The search returns records for both MPF and non-MPF patients, but hides patients flagged as VIP if you do not have permission to view VIP records. In addition, for a patient who has received care at multiple facilities, the system returns a separate entry for each facility.

You can select more than one entry at a time from the result set. This is convenient when, for example, you want to create a request. All entries you select will be added to the request.

  1. Select Patients > Find Patient to access the Find Patient form.

  1. In the main content pane, enter search criteria.

    You can use search criteria in any combination. However, if you use the following fields, observe these rules:

    • You must enter at least three characters in Last Name or First Name or across both fields.

    • Date of Birth must be combined with at least one other field.

    • Facility must be combined with at least one other field.

    • When used, the following fields individually require at least three characters: SSN, MRN, or Enc/Acct.

  2. Click Find Patient.

The system lists all patients who meet the search criteria you entered.

Note: The result set is limited to a maximum of 500 records. If more records exist, you can enter more search criteria to narrow the search.

  1. In the detail pane, select the patient you want and click View/Edit Patient.
