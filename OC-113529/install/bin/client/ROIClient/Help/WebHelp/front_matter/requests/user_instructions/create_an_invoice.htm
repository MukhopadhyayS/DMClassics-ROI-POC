Create an invoice

This task enables an authorized user to create an invoice on-demand for a request.

You can create an invoice on-demand any time during the life of a request, as the total cost and balance due may change over time. You can create an invoice even if no documents have been specified for release, although certain subtotals will be zero.  

While not required, your site may have configured the ROI system to automatically include a cover letter with an invoice.

Notes: You can also create an invoice when you release documents; see Release documents for details. To retrieve or recreate a previously created invoice for viewing or output, see View pre-bill and invoice history or Resend documents.

Instructions assume you are accessing an existing request.

 

  1. Find a request.

  2. Go to the Requests > Billing & Payment form.

  3. Click Invoice.

  4. At the Invoice dialog box, select the template you want to use.

  1. Optional: The Notes box displays only if your site has configured the Notes feature. Select one or more standard notes (if any exist) to include on the invoice, or add a free-form note.

Note: Use only letters and/or numbers in free-form notes or descriptions. Do not include special characters. Special characters (such as apostrophes, dashes, ampersands, and so on) may cause the system to error when generating letters.

  1. Click Continue.

  2. At the Preview of Invoice dialog box, review the information, and then select the output method you want to use. The system displays only the output options for which you have permission.

Note: The Total Pages figure displayed on each invoice represents the total number of pages released so far for the request. The Total Pages calculation includes only those pages that have actually been released and does not consider any re-released pages.

  1. Provide information as needed at the print, fax, or save as file dialog box, and complete the output operation.

If you choose to output to a file, the system automatically assigns a file name based on request ID. If you choose to fax the output, the system automatically populates the requestor's fax information and allows you to change it.

To help you monitor your output jobs, the system assigns an output job name based on request ID appended with "- Invoice Sent".
