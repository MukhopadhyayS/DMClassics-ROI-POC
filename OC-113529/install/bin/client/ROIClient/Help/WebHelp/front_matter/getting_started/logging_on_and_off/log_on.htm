Log on

This procedure tells you how to log on to the McKesson Patient Folder ROI Module.

 

  1. Launch the ROI Module.

OR

If the application is already open, continue with the next step.

  1. At the ROI Logon dialog box, enter your user ID and password. Then click Log On.

If

Then

Your logon was successful

The system displays the default ROI view.

You can begin using the ROI Module.

Your logon failed because your User ID and/or password are invalid

The system alerts you and allows you to try again.

Click OK and try again.

Your logon failed because you used a default password for logon

The system alerts you and requires you to change your password.

Click OK to close the message box and continue to the Change Password dialog box.

Your logon failed because:

  • You do not have permission to use the ROI Module

  • You have exceeded the maximum number of logon attempts allowed

  • Your account is locked

  • Your account is inactive

  • The server is down

The system alerts you to the reason, and you cannot log on.

Contact your system administrator.
