Display: Header Footer group

Specifies whether to allow users the option to display headers and footers on release documents and which of the items listed below to display at each location. If you enable headers and footers, and users select to display them on release documents, headers and footers will be added to each document page. They will not be added to letters.

You can select both Header and Footer, but the same items cannot be displayed at both locations. For example, you can display relative page numbers in headers or footers, but not in headers and footers.

Note: Patient information items (patient name, encounter, and MRN) are treated as a group. Therefore, if you select to add patient name to the header, you will not be able to add patient name, encounter, or MRN to the footer. You will be able to add the encounter and MRN to the header, where the patient name is located. If one item of this group is selected, you do not have to select the others. If selected for display, this information is centered in the header or footer.

If you select Header and/or Footer, the following additional options are provided:

  • Display Absolute Page Number:  Yes  No
    Specifies whether to display absolute page numbers on the document pages of a release. Absolute page numbers restart at 1 for each new document. So, for example, the page numbering would be 1 of 5, 2 of 5, and so on, through page 5 of 5 for a 5-page document. Then, for a second document in the same release, they would be 1 of 6, 2 of 6, and so on, through page 6 of 6 for a 6-page document.

    Absolute Page Number Location:     If you select Yes, specifies whether to display the page numbers in the header or footer.  At either location, the page numbers will be left justified.
     

  • Display Relative Page Number:  Yes  No
    Specifies whether to display relative page numbers on the document pages of a release. Relative page numbers increment by 1 for each subsequent page, regardless the number of documents. So, in the current example, the page numbering would be 1 of 11, 2 of 11, and so on, through page 11 of 11.

    Letters such as cover letters and invoices are not included in the page count. This option provides the total number of billable pages.

    Relative Page Number Location: If you select Yes, specifies whether to display the page numbers in the header or footer. At either location, the page numbers will be right justified.
     

  • Display Patient Name:  Yes  No
    Specifies whether to display the patient name on release documents.

    Patient Name Location: If you select Yes, specifies whether to display the patient name in the header or footer.
     

  • Display Encounter:  Yes  No
    Specifies whether to display the encounter number on release documents.

    Encounter Location: If you select Yes, specifies whether to display the encounter number in the header or footer.
     

  • Display MRN:  Yes  No
    Specifies whether to display the medical record number on release documents.

    MRN Location:     If you select Yes, specifies whether to display the MRN in the header or footer.

    MRN Masking:     If you select Yes, specifies whether to mask the MRN. You can mask from 0 to 9 characters. Masking is usually done when the MRN is the same as the Social Security Number.
