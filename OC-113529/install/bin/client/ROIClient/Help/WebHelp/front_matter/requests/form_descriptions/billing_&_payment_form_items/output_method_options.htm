Output Method options

Allows you to select the method to use for releasing the documents. Options are based on your user permissions.

Initially, no option is selected. After you select an option, it remains selected until you change it or you successfully release the documents. If you cancel the release, the selection is cleared.

Options are:

  • Print
    Generates a print job when documents are released.

  • Save as File
    Generates output to a PDF file in a file location that you specify when documents are released.

  • Fax
    Generates output to a fax job, using a fax number you specify, when documents are released.
