Revert / Cancel button

Discards unsaved data.

Rules

  • If you are viewing existing patient information:

    • Button label is Revert

    • Enabled only for non-MPF patients and only if you revise patient data

    • Discards unsaved data and returns field values to the last saved state

  • If you are creating a new patient:

    • Button label is Cancel

    • Displays only for non-MPF patients

    • Is always enabled

    • Discards unsaved data
