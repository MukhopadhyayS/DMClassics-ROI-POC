Add a patient to a request

This function enables you to add one or more patients to a request. You can add as many patients to the request as needed.

  1. With the request open, go to the Patient Information  form. (For an existing request, find and open it.)   

  1. Click Add Patient (for a new request) or Add Another Patient (for an existing request).  

  1. At the Add Patient(s) dialog box:

    1. Perform a search. Results show all patient records that match the search criteria and include all facilities that provided care for the same patient.

    2. Select one or more patients (use caution).

      Caution: Do not release records for different people to a patient who is requesting records. For patient requestors, if you select multiple patients, the system displays a list of potential mismatches and allows you to evaluate each patient record to determine whether the selected records are appropriate for this request.  

    3. Optional: To add the selected patients, but perform another search and select more patients, click Add Patient & Continue.

    4. If you selected all the patients you wanted and are ready to return to the Patient Information form, click Add Patient & Close.

  2. Click Save. The request now contains the patients for whom you want to release documents. You can continue to build the request now or can return to it later.

To continue building the request, see Add documents to a request.
