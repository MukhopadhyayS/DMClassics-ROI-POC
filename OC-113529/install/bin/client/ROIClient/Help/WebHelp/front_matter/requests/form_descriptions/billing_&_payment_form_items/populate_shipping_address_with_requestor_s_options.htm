Populate Shipping Address with Requestor's options

Specifies the address to send the released documents to. Options are Main Address or Alternate Address.

Controls are enabled only if you set Will the Release be Shipped? to Yes.

Allows you to select an address to use and edit address information as needed.

Changes apply to only this request (edits do not affect the requestor's main or alternate address on record). If you enter changes, the system:

  • Disables the address option buttons

  • Displays a prompt to confirm that you want to use the modified address
