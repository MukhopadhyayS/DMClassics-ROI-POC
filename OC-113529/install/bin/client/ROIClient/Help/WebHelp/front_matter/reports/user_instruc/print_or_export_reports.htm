Print or export reports

After generating a report, you can print or export (save) the report to file in your choice of various formats (PDF, XLS, RTF, HTML, or CSV).

Printing or exporting reports requires specific security rights that must be granted by the ROI administrator. If you have just been granted these rights, you must log out and log back in for them to take effect.

To print ROI reports

  1. Select the Reports tab.

  2. Generate the report you want to print.

  3. Optionally, click Print Preview to preview the report.

  4. Click Print.

The system displays the Print dialog box.

  1. Select the print options you need for printing the report and click OK.

To export ROI reports

  1. Select the Reports tab.

  2. Generate the report you want to export.  

  3. Click Export.

The system displays the Export Report dialog box.

  1. In the File Name box, enter a file name for the report.

  2. In the File Format box, select the format for the file (PDF, XLS, RTF, HTML, or CSV).

  3. In the Save In box, select the location where you want to save the report.

  4. Click Save.
