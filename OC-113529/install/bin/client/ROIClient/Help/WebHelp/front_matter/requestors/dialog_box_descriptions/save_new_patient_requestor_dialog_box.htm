Save New Patient Requestor dialog box

The Save New Patient Requestor dialog box enables you to review and edit information for the patient requestor record you are creating. This dialog box displays when you are creating a request with a patient as the requestor and no matching requestor record exists.

To open the form

  1. On the Patients tab, search for or create a new patient.

  2. Click Patient Requesting.

  3. At the Create New Request with Patient Requesting dialog box, click Select Patient as Requestor.

Item descriptions

This dialog box uses the same items as the Requests > Requestor Information form, except for the Delete Requestor and Create Request buttons, which do not display on this dialog box. In addition, the Last Name and First Name fields on this dialog box appear as a single field labeled Name on the Requestor Information form.

See Requestor Information form for item descriptions.
