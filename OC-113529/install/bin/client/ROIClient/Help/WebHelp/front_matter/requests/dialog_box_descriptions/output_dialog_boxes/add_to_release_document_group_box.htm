Add to Release Document group box

Allows you to print a header, footer, and/or watermark on all released documents (except letters).

Displays only when the output dialog box has been launched from the Release dialog box and the header, footer, and/or watermark options are enabled in the Configure Output Settings function (any disabled option does not display). Displayed options are enabled only if a corresponding value has been configured.

  • Headers and footers can display an absolute page number, a relative page number, the patient name, and the encounter number, MRN, or EPN.

  • Watermarks can contain either the default value or any text string you specify up to 20 characters.
