Requestor grid

Displays the requestor search result set and allows you to a select requestor.

Can be sorted on all columns.

Contains the following information:

  • Requestor Name
    Used as the default sort in alphanumeric ascending order.

  • Address
    Displays street address, city, state, and zip code.

  • Phone
    Displays the work phone for a non-patient requestor.
    Displays the home phone for a patient requestor.

  • Fax
    Displays the fax number associated with the requestor's main address.

  • DOB
    Displays only if Requestor Type is Patient or All Requestor Types.
    Displays the patient requestor's date of birth.

  • SSN
    Displays only if Requestor Type is Patient or All Requestor Types.
    Displays the patient's Social Security Number

  • MRN
    Displays only if     Requestor Type is Patient or All Requestor types.
    Displays the patient requestor's medical record number.

  • Facility
    Displays only if Requestor Type is Patient or All Requestor Types.
    Displays the code for the facility where patient care occurred.

  • EPN
    Displays only if EPN is configured and Requestor Type is Patient or All Requestor Types.
    Displays the patient's enterprise person number.

  • Status
    Displays requestor's status (active or inactive).
