Patient Requesting button

Allows you to create a new request for the selected patient, with the patient as the requestor.

Displays only if you have ROI Create Request permissions.

  • For a new patient, enabled only after you save information to create the new patient

  • For existing patients, enabled when:

    • EPN is not configured and you have selected at least one patient OR

    • EPN is configured and you have selected only one patient

  • Launches the Create New Request with Patient Requesting dialog box
