Potential Patient Mismatch dialog box

The Potential Patient Mismatch dialog box displays when:

  • You create a new request with the patient as the requestor and some of the selected patients do not match the requestor.

  • You have added patients to a request in which the requestor is a patient and the other patient records do not match the requestor.

The dialog box lists patients you selected on the calling form, highlights any potential mismatch(es), and allows you to select no patient, one patient, or multiple patient records to continue.

To open the form

  1. Select Patients > Find Patient and perform a search.

  2. From the search results grid, select more than one patient.

  3. Click Patient Requesting.

Note: This dialog box is also launched from the Requests function at the Add Patient dialog box when the selected patient(s) do not match the requestor.
