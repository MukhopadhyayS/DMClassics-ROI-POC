Patient Information grid

Displays the Patient Name and associated information for all patients matching the search criteria. The grid title is preceded by the number of patients found.

Except for the Locked and VIP columns, you can sort on any column by clicking the column header and sort in reverse order by clicking the column header again.

The Patients grid contains the following columns:

  • Locked
    An icon indicates patients that are locked or that have an encounter that is locked. If locked and you do not have security rights to view this information, the system displays the patient's name in the report, but masks all other patient-related or encounter-related information. This information comes from the HIS system.

  • VIP
    An icon indicates patients that have VIP status. If you do not have security rights to view this information, the system displays the patient's name in the report, but masks all other patient-related information. This information is passed from the HIS system to the McKesson Patient Folder system.

  • Patient Name
    First and last names of patients matching the search criteria.

  • Date of Birth
    Date of birth of patients matching the search criteria.

  • SSN
    Social Security Numbers of patients matching the search criteria.

  • Facility
    Facilities where the patient care occurred for patients matching the search criteria.

  • MRN
    Medical Record Numbers of patients matching the search criteria.
