Patients grid

Displays an entry for each patient name you selected on the calling form and allows you to select one or more patient records to include on the request.

Initially sorted by Patient Name in alphanumeric ascending order. Can be sorted by any column. By default, the first row is auto-selected.

Information includes:

  • Patient Name
    The name of the patient selected in the calling form. Sort is on the first characters displayed, regardless of which name (first or last) occurs first.

  • DOB
    The date of birth of the patient.
    Initial sort is date ascending. Format is mm/dd/yyyy.

  • SSN
    The patient's Social Security Number.
    Initial sort is numeric ascending.

  • Facility
    The facility where patient care occurred.
    Initial sort is alphanumeric ascending.

  • MRN
    The patient's medical record number.
    Initial sort is alphanumeric ascending.

  • EPN (displays only for EPN systems)
    The patient's enterprise person number.
    Initial sort is alphanumeric ascending.
    Displays only if the dialog box was launched by the Add Patient function when maintaining requests.
