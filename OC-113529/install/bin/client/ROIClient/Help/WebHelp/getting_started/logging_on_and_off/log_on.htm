Log on

This procedure tells you how to log on to the McKesson Patient Folder ROI Module. Logon procedures vary depending on whether your site uses the Directory Services Authentication feature:

  • Pre-Release 15.0 systems always use standard MPF authentication.

  • Release 15.0 and later systems have the option of using Directory Services authentication. Directory Services authentication allows you to use your Windows credentials for logging on to McKesson Patient Folder applications.

Note: If you are unsure of which release you are using, at the ROI Module window select Help > About from the menu. The system displays ROI release information. If you are using Release 15.0 (or later) and are unsure whether Directory Services Authentication is enabled, follow instructions for the logon dialog box that matches your screen or ask your supervisor or McKesson Patient Folder system administrator.

Follow instructions for the authentication method used by your site.

  • Standard McKesson Patient Folder authentication

  • Directory Services authentication
