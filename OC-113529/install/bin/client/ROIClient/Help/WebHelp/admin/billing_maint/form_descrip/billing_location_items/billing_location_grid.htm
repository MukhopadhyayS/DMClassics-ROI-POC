Billing Location grid

Displays the billing location and associated information currently defined in the system. The grid title is preceded by the number of existing billing locations.

The Billing Location grid contains the following columns:

  • Default
    Indicates if this is the default location.

  • Billing Location
    Name of the billing location.

  • Tax Percentage
    Amount of the tax percentage.

  • Description
    Descriptions of the billing location (optional).
