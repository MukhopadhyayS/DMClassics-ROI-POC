Configure save and retry options

To configure the length of time to save completed output jobs for users to access and/or the length of time to retry failed output jobs, do the following:

  1. On the menu bar, select Administration > Configure Output Settings.

  1. Select the Save and Retry tab.

  2. Do one or both of the following:

    • In the Save completed output jobs for: <number of> Hours box, enter the length of time you want the system to save completed jobs for users to access.

    • In the Cease system retry attempts after: <number of> Minutes box, enter the length of time you want the system to retry failed jobs before ceasing.  

  1. Click Save.
