Configure masking

To enable or disable the masking of Social Security Numbers during online ROI sessions and in reports, letters, and invoices generated by ROI users, do the following:

  1. Access the Masking form.

  2. If you want to:

    • Mask SSNs, select Apply SSN Masking.

    • Display SSNs, do not select (or deselect) Apply SSN Masking.

  3. Click Save.
