Request Status group box

Contains the following items.

  • Current Status
    For existing requests, displays the current status of the request.
    For new requests, is blank until after the request is saved, and then defaults to Auth-Received (or Auth-Received Denied) if the system created the request or to Logged if a user created the request.
    Can be changed by an authorized user.

  • Current Reason
    The reason(s) associated with a status change (if the status requires a reason to be given).
    Displays the latest recorded reason(s) from the Change of Status and the Status Reason Added events in the event history.
    Can be changed by an authorized user.

  • Update Request Status button
    Launches the     Update Request Status dialog box so that an authorized user can change the request status and/or status reason(s).
