Complete a request

This function allows an authorized user to complete a request. You can complete a request in one of two ways:

  • Release the documents for the request

  • Manually set the status to Complete or Denied

For Auth-Received requests, at the Request Information form you should click View Authorization Request to verify information. After you verify information, proceed as usual.

To complete a request by releasing documents

See Release documents for details.

To complete a request by changing its status

You cannot set the status to Complete if no documents have been released for the request.

  1. Find a request.

  2. On the Request Information form, click Update Request Status.

  3. In the Update Request Status dialog box, select Completed or Denied for the status.

  4. Select one or more reasons for the status change. To create a free-form reason, click Add Freeform Reason and enter text.

  5. Click Save.

  6. On the Request Information form, click Save to complete the event.
