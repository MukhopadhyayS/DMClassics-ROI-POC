Print, fax, save as file, email or write to disc

This function enables you to print, fax, email, write to disc or save a letter as a PDF file.

Note: Not all preview dialog boxes contain all buttons described below. Instructions are general in nature.

At the Preview dialog box, do any of the following.

Click

To

Then

Print

Print the letter to any network printer connected to your workstation

At the Print dialog box, select a printer, enter other print options as desired,  and click OK to send the job to the print queue.

Fax

Fax the letter

At the Fax dialog box, make any changes to the requestor's information as needed and click OK to send the job to the fax queue.

Save as File

Save the letter as a PDF file

At the Save as File dialog box, make any changes needed and click OK to send the job to the output queue.

Email

Email the letter to the email address

At the Email dialog box, make any changes to the email address in the 'To' field, and click OK to send the job to the output queue.

Disc

Write the letter to disc

At the Output Disc dialog box, make any changes ad click OK to send the job to the output queue.

Continue

Launches the dialog box for the output method (Print, Fax, Save as File, or Email) specified on the Release & Invoice form.

Note: The Continue button displays only if you are releasing documents.

Make any changes as needed and click OK to send the job to the appropriate queue.

Cancel

Close the preview window and return to the calling form

(not applicable)
