View/Edit Requestor's Profile dialog box

The View/Edit Requestor's Profile dialog box enables you to view and/or edit demographic information for the requestor. Any changes you make are global and can optionally be applied to the current request.

To open the form

  1. Search for and select a request on the Requests tab.

  2. Click View / Edit Request.

  3. On the Request Information form, click View / Edit Requestor's Profile.  

Item descriptions

This dialog box uses the same items as the Requestors > Requestor Information form. Exceptions are the Delete Requestor and Create Request buttons, which do not display on this dialog box.  

See Requestor Information form for item descriptions.

Different button behavior

The behavior of the Save and Revert buttons differs between the form and this dialog box. From the dialog box, the buttons behave as follows.

  • Save button
    If you updated contact and shipping information on the request, and THEN accessed the View/Edit function and updated the requestor’s contact and address information, the system prompts you to confirm that you want to apply the same changes to the corresponding information on the request. You have the option of applying updates to the request or leaving the request information unchanged.

  • Revert button
    The button is always labeled Revert. It discards changes and reverts information to the last saved state.

Additional items

Close button

Closes the dialog box and returns you to the Request Information form on the Requests tab.

If unsaved changes exist, launches a message that prompts you whether to save the changes.
