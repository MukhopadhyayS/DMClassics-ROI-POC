Preview of Cover Letter dialog box

The Preview of Cover Letter dialog box displays a preview of a letter you are generating, and outputs the form using the selected method.

To open the form

  1. On the Release & Invoice form, click Release & Invoice.  

  2. In the Release dialog box, click Add Cover Letter to release, select a template, and click Continue.
