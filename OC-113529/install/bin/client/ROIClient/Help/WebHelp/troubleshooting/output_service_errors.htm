Output Service errors

This topic provides troubleshooting tips about problems you may encounter with the Output Service.

"Unable to retrieve data from server" error

Symptom: When viewing ROI reports, users may receive an "Unable to retrieve data from server" error message and the Output Server creates a "temp" file in the C:\ drive.

Cause: Unknown. Occurs sporadically and cannot be reproduced on demand.

Resolution:  

  1. Stop the Output JBoss service.

  2. Delete the "temp" file from the C:\ drive.

  3. Restart the Output JBoss service.

  4. Log in to the ROI Module as any user.

  5. Select any report and click Run. The report should display properly with no error.
