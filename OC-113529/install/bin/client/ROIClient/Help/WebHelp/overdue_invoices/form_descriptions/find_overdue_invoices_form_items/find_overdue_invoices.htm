Find Overdue Invoices

Contains the following items:

  • Number of invoices
    The number of invoices matching the search criteria.

  • Billing Location
    The billing location for the selected invoice.

  • Requestor Type
    The requestor type.

  • Requestor Name
    The requestor's name.

  • Days Overdue
    The number of days the invoice payment is overdue.  Use the slider or enter values in the text boxes.

  • Search
    Searches the system for results matching the selection criteria.

  • Reset
    Resets all the current selections.
