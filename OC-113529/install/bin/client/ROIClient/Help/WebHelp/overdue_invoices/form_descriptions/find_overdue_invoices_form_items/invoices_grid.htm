Invoices grid

Contains the following items:

  • Billing Location
    The billing location for the selected invoice.

  • Requestor Type
    The requestor type.

  • Requestor Name
    The requestor's name.

  • Invoice Number
    The invoice number.

  • Request ID
    The system request ID.

  • Overdue Amount
    The amount overdue.

  • # Days Overdue
    The number of days the invoice payment is overdue.

  • Phone Number
    The phone number of the requestor.

  • View Highlighted
    The system displays the selected invoice in a new window.

  • Output
    Displays the Output Statements and/or Invoices form.
