Zooming pages

Standard zoom functions are effective with any document. A special function called Rubberband Zoom is effective only under certain conditions.

Tip: You can also enable a popup menu that includes the most commonly used page operations. See “View menu” earlier in this document for Popup Menu details.

Using Rubberband Zoom

Use the Rubberband Zoom feature to enlarge a selected portion of a page. This feature toggles on or off and is available only when you are viewing a document as an image.

  • The Drag Scroll menu item must be enabled for Rubberband Zoom to operate.
  • When toggled on, Rubberband Zoom automatically deselects the Popup Menu item (if enabled).

To enlarge a selected area:

  1. View a document as an image. (Scanned documents are always displayed as images. You can display text documents as images by de-selecting the Display as Text option on the View menu.)
  2. Enable Rubberband Zoom on the View menu.
  3. Enable Drag Scroll on the View menu
  4. Point the cursor at a beginning point for the area of the page you want to enlarge. The cursor changes to a magnifying glass.
  1. Click and hold the right mouse button and drag it over the area to enlarge. A rectangular outline appears over the area.
  1. Release the right mouse button. The system displays an enlarged view of the selected area.
