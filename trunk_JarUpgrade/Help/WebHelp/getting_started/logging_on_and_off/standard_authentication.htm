Standard OneContent authentication

This task tells you how to log on using standard OneContent authentication.

  1. Launch the ROI Module.
  1. At the ROI Logon dialog box, enter your OneContent user ID and password. Then click Log On.
  2. If your logon is successful, the system displays the default ROI view and you can begin using the module. If your logon fails, see Logon errors for assistance.

Add Another Mapping dialog box

The Add Another Mapping dialog box allows you to associate more than one user ID with a Windows account. This dialog box displays after you have successfully mapped at least one user ID with your Windows account and choose to add another one.

This feature is effective only with ROI Release 15.0 and later.

To open the form

  1. At the Logon dialog box, enter your Windows user name, password, and domain.
  2. Click Logon.
  3. At the Directory Services Self-Mapping dialog box, enter your user name and password and click Add Mapping.
  4. The system confirms that the mapping was successful and prompts you to map more accounts, if you have more than one. Click Map Another Account.
