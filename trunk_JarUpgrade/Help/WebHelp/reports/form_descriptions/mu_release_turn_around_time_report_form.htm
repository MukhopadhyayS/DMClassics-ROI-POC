MU Release Turn Around Time Report form

The MU Release Turnaround Time Report form allows you to generate a report that shows request turnaround times for Meaningful Use documents for OneContent patients (non-OneContent patients are excluded). The report includes only the first release of an MU document. Subsequent releases of the same document are not included. You can select criteria for the report and specify the date range. Security rights are required.

This report shows request turnaround times as measured in days.

When viewed by Totals Per Facility, the report shows the total number of first releases that were completed within the government-mandated turnaround period. It also shows compliance as a percentage of total MU documents that were released.

Default sort order

The report is initially sorted as follows.

  • Turnaround Hours. Highest total turnaround time in descending order
  • Business Days. Highest total business days in descending order within turnaround time
  • Date Received. Latest MU Request receipt date in descending order within Total business days
  • Date Released. Latest date released in descending order within date received
  • Facility. Ascending alphabetic order within date released
  • User Name. Ascending alphabetic order within Facility
  • Patient Name (Last, First). Ascending alphabetic order within Username

Note: All the requests with a blank Turnaround time value and Date Released are listed at the beginning of the report sorted by the latest date received. They are followed by the highest turnaround time requests as defined in the above sort order.

To open the form

  1. Select the Reports tab.
  2. In the Report Name box, select MU Release Turnaround Time Report.  

When the form first displays, the main content pane is blank. To view a report, select the report options in the left pane and click Run Report.

Item descriptions

Left pane

Facility

Specifies the facility or facilities on which you want to report.

You can select or deselect one or more facilities, click Select All to select all facilities, or click Clear All to clear all previous selections.

Document Type

Lists all Meaningful Use document types and allows you to select the type for which you want a report. Options are:

  • Clinical Summary (selected by default)
  • Discharge Summary
  • Discharge Instructions

Date Range

Specifies the time span for the report. You can select from the following options:

  • Today's Date
  • Custom Date.  If selected, enter the applicable dates in the State Date and End Date fields.
  • Current Week
  • Current Month
  • Previous Month
  • 3 Months Prior
  • 12 Months Period
  • Year to Date
  • Previous Year

Reset

Clears the pane of any selection criteria and displays only the drop-down field with the list of reports.

Run Report

Generates the report based on the specified report criteria and date range.

Main content pane

View By

Provides filtering options to change the information displayed in the report. You do not need to re-run the report to see the new view. The system automatically changes the view when you make a selection. Options include:

  • Details
  • Totals Per Facility

Print Preview

Displays the Print Preview dialog box so you can preview the report before printing it.

Print

Displays your standard Print dialog box so you can select a printer and printing options.

Export

Displays the Export Report dialog box so you can select a file name, file format, and location for saving the report. The available file formats are PDF, XLS, RTF, HTML, and CSV.
