View attachments

This task allows you to view attachments. You must view attachments separately from OneContent documents and you can view only one attachment at a time.

Note: You can also view attachments from the Requests > Patient Information form.

  1. Find a patient.
  2. Go to the Patient Records form and expand the ATTACHMENTS node on the document tree.
  1. Click View Selected.

If no conflicts exist, the system displays the attachment content. Close the viewer window when you are ready.

The system informs you:

• If you have selected an invalid combination of attachments to view. For example, you cannot view attachments and OneContent or non-OneContent documents in the same operation. You must select them separately.
• If the file type you want to view has not been associated with an application (such as Adobe Reader) that can display the attachment. The system allows you to make the association.
