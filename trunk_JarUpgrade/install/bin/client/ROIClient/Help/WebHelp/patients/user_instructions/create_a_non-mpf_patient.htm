Create a non-MPF patient

You can create records for only non-MPF patients. A non-MPF patient does not exist in the McKesson Patient Folder database. This occurs, for example, when you cannot match a patient chart to an MPF patient. Consequently, you must create a patient record to associate with the chart for the purposes of managing the ROI request for that person.

  1. Select Patients > Find Patient.

  1. In the main content pane, click New Patient.

  1. Indicate whether this patient has VIP status.

  2. Enter the patient's name (required), gender, and date of birth.

  3. Optional: Enter the patient's SSN, EPN, Facility, and/or MRN information, and the patient's address and contact information.

  4. Click Save.
