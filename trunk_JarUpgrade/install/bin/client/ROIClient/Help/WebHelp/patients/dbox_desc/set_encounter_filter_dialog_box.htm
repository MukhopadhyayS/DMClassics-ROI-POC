Set Encounter Filter dialog box

The Set Encounter Filter dialog box allows you to select/deselect one or more encounters to display on the calling form.

If you perform a search for a patient and do not specify an encounter, the system displays all encounters specific to the selected patient and selects all encounters by default. If you do specify an encounter, the system defaults to the specified encounter and deselects all other encounters.

In addition:

  • If EPN is turned on for your system, all encounters are displayed.

  • If EPN is turned off for your system, only the selected encounters for the selected patient record (MRN and Facility) are displayed.

To open the form

Click the Set/Update button on the Patient Records form or the Patient Request History form.
