Document tree

Allows you to navigate to and select records associated with the current patient. Always lists encounters, but filters documents based on your record view permissions and on the document types associated with each record view.

For MPF patients, displays Global, MPF, non-MPF, and attachment documents. For non-MPF patients, displays only non-MPF and attachment documents.

  • The "GLOBAL DOCUMENTS" folder lists all documents for the selected patient that are classified in MPF as global. Display is in chart order, most recent first.  Information includes document type name and admit date and time. Display is not affected by Filter, Record View, or Sort By selections.

  • If non-MPF documents exist for this patient, the tree displays a "NON MPF DOCUMENTS" folder. The node lists documents, most recent first, with document type name, facility, department, date of service, and pages. Display is not affected by Filter or Sort By selections.

  • If attachments exist for this patient, the tree displays an "ATTACHMENTS" folder. The node lists documents that have been attached to the patient.

  • Remaining folders represent MPF documents and are sorted according to the Sort By option. MPF documents are grouped by Encounter, Document Names, Version, and Page numbers. At the encounter level, the system displays encounter number, patient status, facility and admit date.

  • Encounters with one or more deficiencies () and, if you have adequate permissions, identifies VIP patients () and locked encounters ().
