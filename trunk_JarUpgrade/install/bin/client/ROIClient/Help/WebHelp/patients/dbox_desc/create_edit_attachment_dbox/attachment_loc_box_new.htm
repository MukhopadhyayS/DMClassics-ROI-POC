Attachment Location box

Allows you to select the current location of the attachment file you want to add to the patient's record. The dialog box then displays different fields, depending on the location you select.

Options are:

  • Local File

Use this option when the file initially resides in a location on your local hard drive, a network drive, or on transportable media like a thumb drive. It is copied to a secure location in the Patient Folder system when you associate it with a patient record.

For item descriptions, see Local File option.

  • Query Clinical System

This option is available only only when the Create New Attachments dialog box is launched from the Requests > Patient Information form. Use this option when you want to check a remote system for eligible files. The ROI Module automatically queries the source clinical system associated with the patient's facility and lists all encounters with documents that are eligible to be attached to the patient's Patient Folder record. The association between a facility and a source clinical system is configured using the MPF Workstation > Administration > Facility function.

For item descriptions, see Query Clinical System option.
