Query Clinical System option

The Create New Attachment dialog box displays the fields listed below only when you select Query Clinical System at the Attachment Location field. This option is available only when the dialog box is launched from the Requests > Patient Information form.

The dialog box enables you to download documents from an external clinical system.

Note: Currently, this function supports acquiring only Clinical Summary files.

If Query Clinical System is selected, the Create New Attachment dialog box contains these items:

  • Clinical System label
    Displays the name of the external clinical system that is associated with the patient's facility. Read only.

  • Encounter grid
    Displays a list of encounters from the clinical system sorted by Admit Date in ascending order. You can select one or more encounters.

  • Add Selected button
    Downloads eligible files from the selected encounter(s) to the McKesson Patient Folder system and associates the file(s) with the current patient record. After the action is complete, the system closes the dialog box and returns you to the parent form.

Note: If an error occurs, the system displays a Status box that contains error information. The system also reports if no files were retrieved.

  • Close button
    Closes the dialog box and returns you to the parent form.
