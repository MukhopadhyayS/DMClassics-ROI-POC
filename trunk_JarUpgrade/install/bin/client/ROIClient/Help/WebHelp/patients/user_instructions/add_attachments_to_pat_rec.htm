Add attachments to a patient record

This task enables you to add files from a local folder to a patient record. The files are added as attachments to the record. You can add attachments for both MPF and non-MPF patients. Currently, only Clinical Summary files can be attached, and you cannot attach an empty file.

Note: The file you want to attach must be a local file. If you need to download a file from an external clinical system, see Download attachments.  

  1. Find a patient.

  2. Go to the Patient Records form and click Add Attachment.

  1. At Attachment Location, select Local File.  

  1. Browse to any file location available to you, whether on your local drive, a network drive, or a removable thumb drive, and select the file you want.

  2. Enter information as needed. (For field descriptions, see Local File option.) Then click Save.

The system associates the file(s) with the patient record and lists them in the patient's document tree.
