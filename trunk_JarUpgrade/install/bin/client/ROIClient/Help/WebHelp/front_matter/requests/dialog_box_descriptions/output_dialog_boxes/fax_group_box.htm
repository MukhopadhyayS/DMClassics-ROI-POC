Fax group box

Allows you to specify a fax number and queue and displays fax attributes. Also allows you set fax properties.

  • Number
    Automatically displays the requestor's fax number, when available.
    Allows you to override the pre-populated number.

  • Name
    Lists all fax queues that have been configured, in alphanumeric order.
    Defaults to the first fax queue in the list for first-time fax or when the last-used fax queue has been deleted; otherwise defaults to the last fax queue you selected.

  • Status, Type, Where, and Comment
    Read-only.
    Displays information for the queue you selected, including the queue status, the queue type, the destination where the output will go, and any comment associated with the queue.
