Queue box

Lists all queues associated with the queue type(s) you selected, in alphanumeric order sectioned by type. All queues are included for administrator users. For a non-administrator user, the list is restricted to only the queues the user is allowed to access.

Displays only if you have the ROI Print/Fax permission.

Defaults to All Queues. Selecting any other option restricts the job list to only those jobs that reside in the queue(s) you selected.
