Generate the Outstanding Invoice Balances Report

To generate the Outstanding Invoice Balances Report, do the following:

  1. Select the Reports tab.

  2. In the Report Name box, select Outstanding Invoice Balances.

  3. In the Report Criteria section, select the criteria for the report.

  4. In the Date Range section, select the date range for the report.

  5. Select Run Report.

The system displays the report in the Outstanding Invoice Balances Report in the main content pane.

  1. Optionally, change the level of detail in the report by selecting a different filtering option in the View By box.
