Create New Non-MPF Document dialog box

The Create New Non-MPF Document dialog box allows you to create a non-MPF document for the currently selected patient.

Note: The Create New Non-MPF Document dialog box shares fields with the Edit Non-MPF Document dialog box. Field descriptions include information for both dialog boxes.

To open the form

Click New Document on the Patient Records form.
