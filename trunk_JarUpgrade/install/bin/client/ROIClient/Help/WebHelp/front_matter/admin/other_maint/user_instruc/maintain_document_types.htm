Maintain disclosure documents

To designate document types as disclosure documents for purposes of the ROI Module, do the following:

  1. Access the Document Types form.

  2. Select the Code Set you want to maintain.

  3. In the Document Types grid, select or de-select one or more document types in the Disclosure column.

  4. Click Save.

Note: Code sets are maintained in McKesson Patient Folder. For information, see the MPF Workstation Administrator's Guide.
