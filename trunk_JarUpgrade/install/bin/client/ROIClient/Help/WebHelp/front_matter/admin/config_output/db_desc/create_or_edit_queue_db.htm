Create New Queue (or Edit Selected Queue) dialog box

The Create New Queue and Edit Selected Queue dialog boxes are the same except for the title. They allow you to add print, file, and fax queues or change their settings.

  • Use the Create New Queue dialog box to add new output queues.

  • Use the Edit Selected Queue dialog box to change the settings for existing output queues.

To open the dialog box

  1. On the menu bar, select Administration > Configure Output Settings.

  2. Select the Queues tab.

  3. Click Create New or Edit.

Depending on the option you select, the system displays the Create New Queue or Edit Selected Queue dialog box.
