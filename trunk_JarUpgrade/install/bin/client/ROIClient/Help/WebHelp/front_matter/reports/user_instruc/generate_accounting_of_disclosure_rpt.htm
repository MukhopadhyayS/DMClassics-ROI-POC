Generate the Accounting of Disclosure Report

To generate the Accounting of Disclosure report, do the following:

  1. Select the Reports tab.

  2. In the Report Name box, select Accounting of Disclosure.

  3. In the Report Criteria section, click Select a Patient.

The system displays the Find Patient dialog box.

  1. Enter or select your search criteria.

Except for the first and last names, the search boxes require at least three characters. The patient's first and last names can be three characters combined.

  1. Click Find Patient.

The system displays the results of the search in the Patients grid.

  1. Locate and select the row for the patient for whom you want to run this report.

  1. Click OK.

The system closes the dialog box and displays the information in the Patient Information section of the Accounting of Disclosure form.

  1. Select TPO (related to treatment, payment, or healthcare operations), Non-TPO, or Both for the Request Type.

  1. Select the Date Range for the report.

  2. Click Run Report.

The system displays a message that a report fee may be charged unless this is the first request in a year.

  1. Click OK.

The system generates the report and displays it in the main content pane of the Accounting of Disclosure form.

  1. Optionally, change the level of detail in the report by selecting a different filtering option in the View By box.
