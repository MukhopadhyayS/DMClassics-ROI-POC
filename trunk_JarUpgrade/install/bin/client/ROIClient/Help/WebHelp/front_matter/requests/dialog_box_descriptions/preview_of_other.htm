Preview of Other

The Preview of Other dialog box displays a preview of a letter that you are generating of type "Other," and allows you to select an output method.  

To open the form

  1. Search for, select, and open a request.

  2. On the Letters form, select Other as the letter type and select a template.

  3. Click Create Letter.
