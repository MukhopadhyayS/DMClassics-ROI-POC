Queues grid

Lists all queues of the type(s) selected in Type. Display behavior depends on your permissions:

  • For an administrator user: Displays all queues for all users and allows you to select one or more queues on which to perform an action. An administrator user can hold, release, and refresh all queues.

  • For a non-administrator user: Displays only the queues you have permission to access. Allows you to select one or more queues to refresh, or to view or act on existing jobs you submitted to a queue.

Initially sorted by Type, then by Status in alphanumeric ascending order. Can be sorted by any column.

Displays the following queue information and buttons:

  • Type
    The type of the queue you want to list.

  • Status
    The status of the queue.

  • Queue
    The name of the queue.

  • Description
    The description of the queue.

  • Jobs
    The number of jobs in the queue.

  • Hold button
    For administrator users only: Holds (that is, pauses) all queue processing.  
    For non-administrator users: Holds only the jobs you submitted.

  • Release button
    For administrator users only: Lifts (that is, unpauses) the hold on queue processing and allows processing to continue. 
    For non-administrator users: Lifts the hold from only the jobs you submitted.

  • Refresh button
    Refreshes the queue grid with updated information, based on currently selected criteria.
    Automatically refreshes every 60 seconds (or as configured) and after every queue event.
