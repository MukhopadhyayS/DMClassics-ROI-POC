Manage output queues

This function enables you to view a list of all configured queues, and to view all active (that is, all uncompleted) output job requests on a specific queue.  

  1. From the menu bar, select Security > Monitor Jobs.

  2. Select the Queue View tab.

  1. View the list as needed or follow instructions below for the task you want to perform.

 

To filter or refresh the queue list

To filter the list of queues, at Type, check one or more options (Print, Fax, or PDF). By default, all options are checked.

To refresh the queue list with current information, click Refresh.

 

To hold or release a queue

When you change queue status to Hold, the system allows any currently running job in the queue(s) to complete, changes the status of the output queue(s) from "Ready" to "Hold," and stops processing any other jobs in the queue(s).

Administrator users can hold or release all jobs in any queue. For a non-administrator user, the hold or release action applies only to jobs in the selected queue(s) that were submitted by that user.

  1. In the Queues grid, select one or more entries.

  2. Click the button for the action you want (Hold or Release).

 

To resubmit, pause, resume, or cancel one or more jobs

  1. In the Jobs grid, select one or more entries.

  2. Click the button for the action you want (Resubmit, Pause, Resume, or Cancel).

 

To view output settings

  1. Select a queue.

  2. Click the View Output Settings button. The system displays the current settings in effect.

 

To refresh the job list

Click the Refresh button. The system updates the Jobs grid based on the currently selected criteria.
