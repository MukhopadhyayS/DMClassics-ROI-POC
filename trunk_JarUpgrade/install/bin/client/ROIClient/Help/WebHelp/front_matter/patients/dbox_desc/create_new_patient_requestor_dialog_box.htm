Create New Patient Requestor dialog box

The Create New Patient Requestor dialog box enables you to quickly add a new patient requestor.

To open the form

The dialog displays when you have selected to create a new patient requestor with the patient selected.

It is triggered from the New Requestor button on the Create a New Request with the Selected Patient Requesting (Matching Requestors Found) dialog.
