Change your password

This function allows you to change your password on-demand or when the system requires you to change it.  

Note: Your MPF Workstation user ID and password and your ROI user ID and password are the same. Changing your password in the ROI Module overwrites the existing password for your MPF Workstation user ID. The reverse is also true. In addition, the new password must be different from the current one and must meet password security rules used by MPF Workstation.

Why you would change your password

You can change your password voluntarily at any time.

You are required to change your password when you have logged on with a temporary password provided by your system administrator. Your system administrator generates a temporary password for you when:

  • You are a new user

  • Your account is locked (such as when the number of failed logon attempts on your account exceeds the maximum number permitted)

  • Your password has expired

 

  1. If you are already logged on and want to change your password, select the Security > Change Password menu option. Then continue with the next step.

If you are in the process of logging on and want to change your password, enter your current password and click Change Password. Then continue with the next step.

If the system is requiring you to change your password, click OK to acknowledge the notification message, and continue with the next step.

  1. On the Change Password dialog box, enter your current password. Then enter your new password twice for validation purposes. Passwords are case-sensitive.

  2. Click OK.

    • If the new password fails password security policies, the system informs you. Click OK to try again.

    • If the new password meets the password security policies, the system saves your entry and displays a message. Click OK to proceed.
