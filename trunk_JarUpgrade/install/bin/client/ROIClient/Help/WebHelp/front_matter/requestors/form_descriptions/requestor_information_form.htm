Requestor Information form

The Requestor Information form enables authorized users to:

  • Create requestors. In this case, all fields are blank or use default values.

  • View and maintain information for an existing requestor. In this case, the form displays current information for the requestor.

  • Create requests.

To open the form

  1. Select Requestors > Find Requestor.

  2. Do either of the following:

    • Click New Requestor to create a requestor.

OR

    • Search for and select a requestor. Then click View/Edit Requestor to view or edit the requestor's current information.
