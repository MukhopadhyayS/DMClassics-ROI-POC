Turn Around Times form

The Turn Around Times form allows you to generate the Turn Around Times Report. You can select criteria for the report and specify the date range. Security rights are required.

This report shows request turn around times as measured in days.

To open the form

  1. Select the Reports tab.

  2. In the Report Name box, select Turn Around Times.  

When the form first displays, the main content pane is blank. To view a report, you must select report options in the left pane and click Run Report.
