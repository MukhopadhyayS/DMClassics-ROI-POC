Configure output queues

Output queues are defined to facilitate and monitor the printing, faxing, and saving of release documents and letters.

To add, modify, or delete output queues, do the following:

  1. On the menu bar, select Administration > Configure Output Settings.

The system displays the Configure Output Settings dialog box.

  1. Follow the instructions below for the task you want to perform.

 

To add a new output queue

  1. On the Queues tab, click Create New.

The system displays the Create New Queue dialog box.

  1. In the Create New Queue dialog box:

    • Select the Queue Type you want to add: Print, Save As File (PDF), or Fax.

    • Enter a Queue Name and, optionally, a Queue Description.

    • In the Set the queue properties for the queue and then click OK group box, select or enter values for the attributes that are listed. The available attributes will vary, depending on the Queue Type you select.

  1. Click OK.

  2. Restart the ROI server to make the queue available to users.

 

To modify an output queue

  1. On the Queues tab, select the queue you want to modify and click Edit.

The system displays the Edit Selected Queue dialog box.

  1. In the Edit Selected Queue dialog box, do one or both of the following:

    • Change the Queue Description.

    • Change the values in the Set the queue properties for the queue and then click OK group box.

  1. Click OK.

  2. Restart the ROI server to make the change available to users.

 

To delete an output queue

  1. On the Queues tab, select the queue you want to delete.

  2. Click Delete.

The system displays the Confirm Queue Deletion message.

  1. Click Delete.

Note: If a job is processing when you select Delete, it will be deleted after it has completed and the purge routine is done.

  1. Restart the ROI server.
