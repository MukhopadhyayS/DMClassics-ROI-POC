Save as File dialog box

The Save as File dialog box enables you to save to a single file any cover letters, invoices, pre-bills, statements and patient documents that are included in a release request.

You cannot choose the save location for the file. The system automatically creates the file in a shared directory path configured by the ROI system administrator; the path also uses your MPF Workstation user name for the directory where your output files will reside.

Path format is: \\<server-name>\<common-dir$>\<ROI-PDF-dir>\<your-username-dir>\.

The actual filename includes <request ID><release sequence><part number><version number>. (See About PDF documents for more details.)

To open the form

The system launches the dialog box when you do any of the following:

  • Click Continue from the Release Documents dialog box without a letter selected

  • Click PDF from the Letters preview dialog box

  • Click PDF from the Invoice preview dialog box

  • Click PDF from the Pre-Bill preview dialog box

  • Select to Resubmit a PDF job from the Monitor Jobs dialog box
