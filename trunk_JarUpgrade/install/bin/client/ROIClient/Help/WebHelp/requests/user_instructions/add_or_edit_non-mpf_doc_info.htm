Add or edit non-MPF document information

This function enables you to create or edit information for a non-MPF document while you are viewing a request. The document itself resides outside of the McKesson Patient Folder and ROI system, but billing charges must be captured if the document is released to a requestor. The information you enter is used to generate the billing charges.

Note: You can also create or edit non-MPF documents from the Patients > Patient Records form. See Maintain non-MPF document information for instructions.

  1. Find a request.

  2. Go to the Request > Patient Information form.  

  3. Conditional: If this request is for a single patient, skip to the next step. If this request includes multiple patients, select the patient name you want in the "x Patient(s) on this Request" box.

  1. Follow instructions below for the task you want to perform.

To add information for a non-MPF document

  1. Click New Document.

  2. On the Create New Non-MPF Document dialog box:

    1. Enter document information.

    2. Click Save. The system closes the dialog box and returns you to the Request > Patient Information form.

  3. Click Save on the Patient Information form.

To edit information for a non-MPF document

  1. At the document view box, select the non-MPF document for which you want to edit information.

  2. At the Edit Non-MPF Document dialog box:  

    1. Change document information as needed.

    2. Click Save. The system closes the dialog box and returns you to the Request > Patient Information form.

  3. Click Save on the Patient Information form.
