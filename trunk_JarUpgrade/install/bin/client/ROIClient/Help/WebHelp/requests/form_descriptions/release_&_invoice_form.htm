Release & Invoice form

The Release & Invoice form enables an authorized user to view, enter, or change billing charges for a currently selected request. If you do not have required permissions, all fields are read-only.

The form also enables an authorized user to generate pre-bill letters and invoices, and to release the requested information.  

You can access the Release & Invoice form only after at least one document has been selected for release on the Patient Information form.

To open the form

Create a request from the Patients, Requestors, or Requests tab.
