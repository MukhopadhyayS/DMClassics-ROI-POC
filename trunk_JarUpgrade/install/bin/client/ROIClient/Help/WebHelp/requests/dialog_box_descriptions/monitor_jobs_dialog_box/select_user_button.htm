Select User button

Enabled only for administrator users.

Launches the "Select a user" dialog box, which enables you to control the list of users displayed in the User box.

At the dialog box, you can:

  • Manually select or clear user names.

  • Click Select All to include all available user IDs in the Monitor Jobs result set. This option causes the User field on the Monitor Jobs dialog box to display All Users as the only option.

  • Click Unselect All to clear all user names. At least one user name must be checked to enable the Select button.

  • Click Select to apply your choices (or Cancel to discard your choices) and return to the Monitor Jobs dialog box.
