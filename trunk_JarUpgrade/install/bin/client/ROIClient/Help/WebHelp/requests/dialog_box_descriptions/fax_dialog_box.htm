Fax dialog box

The Fax dialog box enables you to fax cover letters, invoices, pre-bills, statements and patient documents that are included in a release request.

To open the form

The system launches the dialog box when you do any of the following:

  • Click Continue from the Release Documents dialog box without a letter selected

  • Click Fax from the Letters preview dialog box

  • Click Fax from the Invoice preview dialog box

  • Click Fax from the Pre-Bill preview dialog box

  • Select to Resubmit a fax job from the Monitor Jobs dialog box
