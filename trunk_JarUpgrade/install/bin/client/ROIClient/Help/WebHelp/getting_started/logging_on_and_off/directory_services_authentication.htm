Directory Services authentication

This task tells you how to log on when the Directory Services Authentication feature is enabled. For more details about authentication, see About logging on or off.

This feature is effective only with ROI Release 15.0 or later.

  1. Launch the ROI Module.

OR

If the application is already open, continue with the next step.

  1. At the ROI Logon dialog box, enter your Windows user ID and password and specify the Windows domain associated with your account.

Note: The Domain box lists all trusted Windows domains known to the McKesson Patient Folder system. If this is your first login, the Domain box is blank. Otherwise, it displays the last domain that was specified.

  1. Click Log On.

  2. Conditional: If your Windows account is not currently mapped to your MPF user ID, the system displays the Directory Services Self-Mapping dialog box. Enter your MPF user name and password and click Add Mapping to associate the two IDs.

Note: The system may display additional dialog boxes during login, depending on current conditions (for example, if you specify an account that is already mapped). Follow on-screen instructions.

  1. If your logon is successful, the system displays the default ROI view and you can begin using the module. If your logon fails, see Logon errors for assistance.
