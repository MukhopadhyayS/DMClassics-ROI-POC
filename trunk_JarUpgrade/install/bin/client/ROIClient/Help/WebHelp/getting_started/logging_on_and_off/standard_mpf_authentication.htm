Standard McKesson Patient Folder authentication

This task tells you how to log on using standard McKesson Patient Folder authentication. For more details about authentication, see About logging on or off.

  1. Launch the ROI Module.

OR

If the application is already open, continue with the next step.

  1. At the ROI Logon dialog box, enter your McKesson Patient Folder user ID and password. Then click Log On.

  2. If your logon is successful, the system displays the default ROI view and you can begin using the module. If your logon fails, see Logon errors for assistance.
