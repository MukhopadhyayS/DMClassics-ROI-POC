Add Another Mapping dialog box

The Add Another Mapping dialog box allows you to associate more than one MPF user ID with a Windows account. This dialog box displays after you have successfully mapped at least one MPF user ID with your Windows account and choose to add another one.

This feature is effective only with ROI Release 15.0 and later.

To open the form

  1. At the Logon dialog box, enter your Windows user name, password, and domain.

  2. Click Logon.

  3. At the Directory Services Self-Mapping dialog box, enter your MPF user name and password and click Add Mapping.

  4. The system confirms that the mapping was successful and prompts you to map more MPF accounts, if you have more than one. Click Map Another Account.
