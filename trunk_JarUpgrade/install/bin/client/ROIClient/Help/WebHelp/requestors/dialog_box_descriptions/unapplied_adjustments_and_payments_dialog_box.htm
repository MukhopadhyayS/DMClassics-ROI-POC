Unapplied Adjustments/Payments dialog box

The Unapplied Adjustments/Payments dialog box allows you to apply any unapplied adjustments or payments on the requestors account when generating an invoice.

To open the form

  1. Create a request.

  2. On the Release & Invoice form, click Release & Invoice.  If there are unapplied adjustments/payments, the system displays the Unapplied Adjustments/Payments dialog box.
