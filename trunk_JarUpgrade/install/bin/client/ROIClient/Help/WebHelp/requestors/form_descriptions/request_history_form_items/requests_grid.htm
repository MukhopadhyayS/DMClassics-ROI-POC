Requests grid

Displays requests for the requestor selected on the Find Requestor form and allows you to a select request.

Initially sorted by Last Updated in most recent to oldest order, and then by Status. Can be sorted on all columns.

Note: If a request contains records from facilities that you are not allowed to access, the detail pane list the request in red text and masks data in the Patients column. If you select the request, the system disables the View/Edit Request button.

Contains the following information:

  • Last Updated
    The date when the request was last updated in mm/dd/yyyy format.

  • Updated By
    The user who last updated the request in LastName, FirstName MiddleInitial format.

  • Status
    The status of the request: Auth-Received, Canceled, Completed, Denied, Logged, Pended, or Pre-Billed.

  • Patients
    Lists all patient(s) on the request in alphanumeric ascending order separated by a colon (:).

  • Request Id
    The system-assigned ID of the request.

  • Receipt Date
    Displays the date the request was received in mm/dd/yyyy format.

  • Balance
    Displays the total charges (in dollars) for the request.
