Requestor Type box

Allows you to select a requestor type to limit the search result set. Requestor Types are maintained in the ROI application and identify the category of requester, such as attorney, employer, or insurance. This information is used for statistical purposes.

Lists all requestor types currently defined for your system.

When used for searching:

  • Defaults to All Requestor Types

When used for defining a requestor:

  • Defaults to Select Requestor Type

  • Cannot be changed after the requestor is saved
