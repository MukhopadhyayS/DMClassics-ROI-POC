Output Letters and/or Statements dialog box

The Output Letters and Statements form enables you to generate a letter or statement for a refund.

Note: The Refund button is enabled only when a credit balance exists for the requestor's account.

To open the form

  1. Find a requestor.

  2. In the left pane, select Account Management.

  3. In the Account Management form, click Refund.  

  4. Complete the dialog box information and click Save + Output Letter.
