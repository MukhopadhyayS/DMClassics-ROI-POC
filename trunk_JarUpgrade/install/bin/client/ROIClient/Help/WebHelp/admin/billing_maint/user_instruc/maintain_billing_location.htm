Maintain billing location

To add, modify, or delete a billing location, do the following:

  1. Access the Billing Location form.

  2. Follow the instructions below for the task you want to perform.

To add a new billing location

  1. In the main content pane, click Create New.

  2. In the detail pane, enter a name and, optionally, a tax percentage, a description for the new fee type, and check if this is the default location.

  3. Click Save.

To modify an existing billing location

  1. In the main content pane, select the billing template you want to modify.

  2. In the detail pane, change the information as needed.

  3. Click Save.

To delete a billing location

Note: You cannot delete billing templates that are associated with requestor types.

  1. In the main content pane, select the billing location you want to delete.

  2. Click Delete.

  3. In the confirmation message, click Delete.
