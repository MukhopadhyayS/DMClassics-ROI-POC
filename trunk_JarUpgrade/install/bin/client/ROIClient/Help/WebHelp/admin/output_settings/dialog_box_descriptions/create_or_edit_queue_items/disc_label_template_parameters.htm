Disc label template parameters

Specific parameters are created using the Rimage CD Designer to specify the disc label information.  

Both variable and static parameters (for example, the hospital logo) can be specified.

To ensure this information appears correctly on the label, you must use the parameters below.  Set each parameter as a Merge field.  See the Rimage Label Designer documentation for details on how to create merge fields.

Note: If the request includes more than one patient or encounter, the patient or encounter values will be displayed separated by commas.  For example, if a request contains patients John Doe and Jane Smith, the patient name label will display John Doe, Jane Smith.  

Label Item

Format (if any)

Label KEY

Per Request:

 

 

Requestor Name

 

#label.requestor.name

Request ID

 

#label.request.id

Request Created Date (when released)

MM/DD/YYYY HH:MM

#label.request.created.date

Request Fulfilled Date (when released)

MM/DD/YYYY HH:MM

#label.request.fulfilled.date

Number of Released Documents

 

#label.released.doc.count

Notes (as selected/entered on the Output to Disc form)

Multiple notes will be separated by semi-colons (;)

#label.released.notes

Per patient with released documents:

 

 

Patient Name

 

#label.patient.name

Patient Date of Birth

MM/DD/YYYY

#label.patient.dob

Patient MRN. (If MRN Masking is set, the MRN will be printed masked.)

 

#label.patient.mrn

Per encounter with release documents:

 

 

Encounter Number

 

#label.enc.num

Admit Date

MM/DD/YYYY

#label.enc.admit.date

Discharge Date

MM/DD/YYYY

#label.enc.discharge.date

Patient Type (for example, Inpatient, Outpatient, ER)

 

#label.enc.patient.type

Number of Encounters

 

#label.enc.count

Number of Patients

 

#label.patient.count
