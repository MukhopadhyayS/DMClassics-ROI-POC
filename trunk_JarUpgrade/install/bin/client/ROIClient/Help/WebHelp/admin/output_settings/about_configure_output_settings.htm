About Configuring Output Settings

The Configure Output Settings feature provides ROI administrators the ability to configure:

  • Output queues used for printing, saving documents to a file, faxing, and emailing.

  • Save and retry options for output jobs.

  • Headers, footers, and watermarks that users can add to release documents.

  • The source for MPF documents and ROI letters.

This function is accessed by selecting Configure Output Settings from the Administration option on the menu bar.

Prerequisites

To configure output settings, the following must be true:

  • You must be an ROI administrator and also have administrator rights to the Output Services system.

  • The ROI Module must be integrated with the Output Services system.

  • Both the ROI Module and the Output Services system must be running.

PDF document passwords

While configuring a "Save as file" queue, you can optionally define a "queue" password that prevents unauthorized users from viewing PDF documents that reside in the output queue. The ROI Module applies an additional level of PDF document security by automatically generating a "request" password when the PDF document is generated. Both passwords are displayed in clear text on these windows:Account History and Release History.

The request password is generated only once and applies to subsequent releases for the request (for example, if you add documents to the request, resend documents, or re-release documents). Release documents are always password protected, regardless of whether a queue password exists. ROI users must convey the request password to the requestor in a secure manner following procedures established by your site.
