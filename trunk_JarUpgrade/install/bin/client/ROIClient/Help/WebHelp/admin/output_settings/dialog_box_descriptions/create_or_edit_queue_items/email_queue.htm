Email Queue

Item descriptions

Queue Type
Specifies the type of queue.

Queue Name
Provides a name for the output queue. This name will be displayed to the user when the user selects to output documents or letters.

Queue Description
Provides a description for the output queue.

SMTP Host
Specifies the name of the host or IP Address.

SMTP Port
Specifies the number of the port for your email server. The default is 25.  

Email Size Limit
Specifies the maximum size in megabytes (MB) of the email. The default is 25MB. If the email size exceeds this default, the job will fail.  

From Email Address
Specifies the sending email address.

From Name
Specifics who the email is from.

Email Subject
Indicates the subject of the email.  The field populates automatically if default information has been configured by the system administrator.  If your email server supports encrypted email delivery, please contact your mail server administrator for configuration details.  If the mail server supports filter-based encryption, type SECURE: and the applicable text - for example, "SECURE: Medical Records Documents." The user cannot change this text.

Email Body
Specifies any information related to the email. The field populates automatically if default information has been configured by the system administrator.  

Override subject and body?
Allows the user to overwrite the preconfigured subject and content body content at the time of output.

Password Required for PDF Files
Specifies any protection for the file.  If you select No, the system allows the user to password protect the file when it is output.  If you select Yes, the system protects the document. Documents include: MPF Documents, Supplemental Documents, Invoices, Pre-Bills, Letters, Attachments and so on.

Default Password
Indicates the password required for the PDF Files.

Mime Type
Specifies the email security setting.

OK
Saves the queue with the options you entered.

Cancel
Cancels adding a new queue or any unsaved changes you made for an existing queue.
