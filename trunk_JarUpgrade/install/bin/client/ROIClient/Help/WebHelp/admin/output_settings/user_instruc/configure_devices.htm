Configure devices

To configure devices for printing of labels on CD or DVDs, do the following:

  1. On the menu bar, select Administration > Configure Output Settings.

The system displays the Configure Output Settings dialog box.

  1. Select the Devices tab.

  2. Follow the instructions below for the task you want to perform.

To configure devices

  1. On the Devices tab, click Create New.

  2. Complete the Create New Device dialog box.

  3. Click OK.

To edit devices

  1. On the Devices tab, select the device to edit.

  2. Click Edit.

  3. Make the applicable changes in the Create New Device dialog box.

  4. Click OK.

To delete devices

  1. On the Devices tab, select the device to delete.

  2. Click Delete.

  3. In the confirmation dialog box, click Delete.  Click Don't Delete to cancel the deletion process.

Note: You cannot delete a device that is being used by a queue.

  1. Click OK.
