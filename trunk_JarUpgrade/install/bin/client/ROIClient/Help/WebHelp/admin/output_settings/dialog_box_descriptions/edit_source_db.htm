Edit Source dialog box

The Edit Source dialog box allows you to enable or disable the source for release documents or letters, or edit the source information.

To open the dialog box

  1. On the menu bar, select Administration > Configure Output Settings.

  2. Select the Sources tab.

  3. Select a source and click Edit.
