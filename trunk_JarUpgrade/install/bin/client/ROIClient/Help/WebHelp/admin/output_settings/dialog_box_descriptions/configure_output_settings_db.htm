Configure Output Settings dialog box

The Configure Output Settings dialog box allows you to:

  • Add, modify, or delete output queues for release documents and letters.

  • Specify the length of time to save output jobs for users to access.

  • Specify the length of time to retry failed output jobs.

  • Specify whether to provide the option to include headers, footers, and/or watermarks on release documents and the information to include.

  • Specify the source for documents and letters to be sent with a release.

  • Specify the devices for output of documents to disc.

To open the dialog box

On the menu bar, select Administration > Configure Output Settings.

This form can also be accessed directly from the internet using the URL configured for your system.
