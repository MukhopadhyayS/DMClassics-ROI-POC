About Viewing Documents

Overview

This section tells you how to use the viewer to view one or more pages from one or more documents.

The most basic viewing function allows you to view one page from one document. More advanced viewing functions allow you to view two separate documents from two separate batches or patients.

The viewer displays scanned documents as images, or text-based documents as ASCII text or as images.
